Identifying Core Facts in Extensible Classes

Extensible core classes can contain both core and user-defined facts. Only some attributes of core facts can be updated; for example, the Fact Name attribute cannot be updated for core facts.

To identify core facts in extensible classes:

  • Select the Lock option and select the fact in the Fact Tool. If the Fact Name attribute is greyed-out, the fact is a core fact.
  • If the fact key (displayed in the Key field) is less than 2000, the fact is a core fact.
