Non-extensible Configuration Classes

Configuration classes are usually non-extensible core classes.

Unlike other non-extensible classes, configuration classes have additional data or objects defined using a dedicated configuration tool. For example, additional objects can be added to the Business_Rule class using the Business Rule Tool.

The Ref column contains Yes if the class can be referenced by a reference fact.
