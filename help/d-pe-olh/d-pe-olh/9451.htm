TransactionGet@[]()

Syntax

TransactionGet@[]()

 

Description

Returns a list of objects in the current transaction.

Parameters

Parameter

Description

None.

 

 

Returns

Array of OIDs, or an empty array if there is no current transaction.

Remarks

The first object returned is the root of the transaction, followed by all dependent objects (in the order in which they are displayed in the transaction context).

If the current object executing the trigger that calls the function is not the root object in the private cache, an empty array is returned.

Example

Perform processing for each object within the current transaction:

{
    var objects@[] := TransactionGet@[]();
    for ( i& := 0; I& < length(objects@[]); ++i&) {
        # Process each object in the transaction
    }
}
