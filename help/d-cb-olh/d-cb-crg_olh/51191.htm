PartitionTablespace

The PartitionTablespace derived attribute table defines the data and index tablespaces used to store date partitions.

The table is used when creating partitions by the create_partitions and partition_tasks server utilities, and the Add All Partitions, Add Partitions, and Partition Tasks schedule types.

It has an evaluation type of Table Lookup By Multiple (1-10) Keys.

It is supplied as an empty table for the core Convergent Billing tables.

Note:

A Table Name cannot have two rows with the same partition number and overlapping date ranges.

Column Definitions

Column Name

Properties

Description

Table Name (Index 1)

String (30)

Not null

Name of the date-partitioned base table for which different tablespaces are used for different date ranges.

From Partition Number (Index 2)

Number (2)

Start of the range of customer partition numbers (inclusive) to be associated with the tablespace.

If not specified, all customer partitions in the specified date range are associated with the same tablespace.

To Partition Number (Index 3)

Number (2)

End of the range of customer partition numbers (inclusive) to be associated with the tablespace.

It can only be specified if From Partition Number is specified.

If specified, it must be greater than or equal to From Partition Number.

From Date Range (Index 4)

Date-time

Not null

Start of the date range (inclusive) of partitions for which this tablespace is to be used.

To Date Range (index 5)

Date-time

Not null

End of the date range (inclusive) of partitions for which this tablespace is to be used.

The default value is one month after From Date Range.

Data Tablespace (Result 1)

String (30)

Not null

Data tablespace containing all partitions in the specified date range.

Index Tablespace (Result 2)

String (30)

Not null

Tablespace used for all local indexes associated with partitions in the specified date range.

If not specified, Data Tablespace is used.

Compressed Data Tablespace (Result 3)

String (30)

Data tablespace containing all compressed partitions in the specified date range.

If not specified, Data Tablespace is used to store both compressed and uncompressed partitions for the date range.

Compressed Index Tablespace (Result 4)

String (30)

Tablespace used for all local indexes associated with the compressed partitions in the specified date range.

If not specified, Compressed Data Tablespace is used, if specified; otherwise, Index Tablespace is used.
