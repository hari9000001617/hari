Description

Returns the minimum of the two values or the minimum item of the array or hash.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

value1?

In: First value to be compared.

value2?

In: Second value to be compared.

array?[]

In: Array to be processed.

hash?{}

In: First hash to be processed.

string1$

In: First string to be compared.

String2$

In: Second string to be compared.

 

Returns

Same data type as the passed parameters. If one parameter is an integer and the other a real, a real value is returned. If the values are arrays, the function returns the index of the array element with the minimum value of all defined elements in the array.

If either the array value is empty or contains no defined elements, the function returns -1. If the values are hashes, the function returns the key of the hash element with the minimum value of all defined elements in the hash. If the hash is empty or contains no defined elements, an empty string is returned.

Remarks

Both values must be of the same type, or they must be either integer or real values.

If two strings are being compared, the minimum of the two strings is returned according to the collation order of the character set in use.
