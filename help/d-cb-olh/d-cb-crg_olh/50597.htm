Description

Deletes a row of data in the global derived attribute table identified by the specified keys.

Context

The following groups of variables are available throughout all stages of processing a derived attribute delete request, except where otherwise specified:

  • System client
  • Operator
  • Miscellaneous
    • evFormCode$ contains the value GDAASVR.
    • evLastRowInd& contains the value 0 during all stages of processing except for the evaluation of the global derived attribute pre-commit expressions for the last row processed.
    • evRowStatus$ contains the value Delete for the global derived attribute processing stage.
  • Derived attribute.

Parameters

Parameter

Description

Keys?{}

Keys used to identify the global derived attribute table row to be deleted.

fEVS_ValidateDefaultKeys&() lists valid key combinations.

LastModified~

Initial value must match the value in the record matching the row identified by Keys?{}.

If LastModified~ is undefined, the last modified value of the derived attribute table for the specified StartDate~ is used.

On successful return, the value is the date-time of this deletion.

StartDate~

Date on which the deletion becomes effective.

If not specified, the current date is used.

EndDate~

Date on which the deletion ceases to be effective.

If not specified, the end of time is used.

IdentifierKeys?{}

Values of the derived attribute row identifier fields derived from Keys?{}.

 

Processing

If Keys?{} is supplied, this function calls the global derived attribute array key lookup to retrieve the:

  • Derived attribute ID
  • Sequence number (for standard global derived attributes) or extended row ID (for extended global derived attributes)
  • Effective date.

Continues processing as if fEV_GlobalDADeleteRows&() was called for one row.

Returns

On success, a non-zero value; otherwise, an appropriate error is raised.
