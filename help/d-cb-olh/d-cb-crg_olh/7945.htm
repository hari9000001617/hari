CURRENCY_V

The CURRENCY_V view provides a non-date ranged view of the CURRENCY_HISTORY table.

The view is used in many Convergent Billing forms.

Column Definitions

Column Name

Properties

Description

CURRENCY_ID

NUMBER (8)

Not null

Currency ID.

LAST_MODIFIED

DATE

Not null

Date-time when this row was last modified.

CURRENCY_SYMBOL

VARCHAR (5)

Not null

Symbol or collection of characters for identifying money in this currency (for example, $US or $AUD).

CURRENCY_NAME

VARCHAR (30)

Not null

Name of the currency, used as the description in drop-down lists.

DISPLAY_FORMAT

VARCHAR (80)

Not null

Display format for all output in this currency (not supported for reports).

Any display format supported by Delphi can be entered.

EDIT_FORMAT

VARCHAR (80)

Not null

Edit format for all input in this currency.

Any edit format supported by Delphi can be entered.

DESCRIPTION

VARCHAR (255)

Not null

General description of the currency.

CONVERTIBLE_IND_CODE

NUMBER (1)

Not null

Set to 1 (TRUE) if this currency can be converted automatically to other currencies by the system as required.

The CONVERSION_RATE table defines the conversion rates to use.

ATLANTA_OPERATOR_ID

NUMBER (8)

Not null

ID of user who last modified this record.

ATLANTA_GROUP_ID

NUMBER (8)

Not null

Security group associated with this record.

It controls access to the record.
