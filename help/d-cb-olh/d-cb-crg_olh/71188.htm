Description

Calls the delete function for an external definition in a Process Engine context.

Interfaces 3, 4, 7 and 8 perform a fetch using a linked external definition before performing the delete.

Attributes

Application Environment

PE

Context

Any

Method

Remote

 

Parameter

Description

ExternalDefnId&

In: ID of the external definition.

The ID determines the delete function to be invoked.

ExternalDefnName$

In: Name of the external definition.

The name determines the delete function to be invoked.

FieldValues?{}

In: Hash element structure (potentially nested) and keys matching external definition elements to be used with delete functions.

ControlInfo?{}

Out: Information to align with adapter interfaces.

Note:

Reserved for future use.

Request?{}

In: Hash element structure (potentially nested), keys, and values matching external definition elements to be used with delete functions.

Response?[]

Out: Array of hashes populated by a linked fetch or search definition.

For a fetch function, only a single array element is populated.

Audit&

In: TRUE if external definition auditing is enabled.

 

Returns

1 if successful; otherwise, an error is raised.

Remarks

The remote service for this function is biFnEvaluateRO.

An error is raised if the:

  • External definition does not exist.
  • Fields passed to the function do not correlate to an external definition element.
  • External definition is not in a validated state.
  • External definition is not marked as external.
