Description


Purges and re-loads a derived attribute table from shared memory caches on all enabled instances.


Attributes






Application Environment



Any







Context



Any







Method



Remote








 






Parameter



Description







TableName$



In:  Name of the derived attribute table to purge.







EffectiveDate~



In:  Effective date to purge the derived attribute table.







Reload&



In:  Reload the derived attribute table into the shared memory cache.








 


Returns


1 (TRUE) if successful; otherwise, an error is raised.


Remarks


The function is implemented as a transactional EPM function that schedules biDerivedTableCachePurgeInstance&() on all enabled instances.


Interface 1 removes all copies of the table that have been cached for different date ranges. The table is reloaded for the date ranges that were purged.


Interface 2 removes all copies of the table that have been cached for the date range containing EffectiveDate~. A null or undefined EffectiveDate~ removes all date ranges. If Reload& is set, the table is reloaded for the date range that was purged.
