Customer Node CBF Interface

The Convergent Billing customer node entity has several related components that are treated as separate by the Convergent Billing API.

The purpose of the functions described in this section is to supply a comprehensive set of encapsulated APIs as a functional extension of the Convergent Billing API set. The functions keep the customer node as the central entity but still provide the granularity to update entities related to the customer node.

Local and remote versions are provided for functions. Remote versions have the same function name but include an '_R' suffix. Because the functional behaviour is the same, the functions are documented together.

Remote functions call the local function but are defined so that the remote function executes in a transactional server. Executing in a transactional server ensures that any subsequently called functions are included as part of the remote function transaction. The result is that if any function called by the remote function fails and transaction rollback is invoked, the effects of all called functions are rolled back. This ensures that the transaction either completes in its entirety, or not at all.

Note:

The local CBF APIs (for example, fCMN_TRECustomerCreate&()) are designed to be called from their remote wrappers (for example, fCMN_TRECustomerCreate_R&()). Local CBFs must be executed in a transactional server and cannot be called from the Expression Test form or any other non-transactional component (such as entity validation).

Customer Composite Business Functions

The following table lists the functions available for the CBF interface.

 

Customer CBF Interface Functions

Function

Description

fCMN_TRECustNodeAccountAdd_R&()

Adds a new account (of a new account type) to the customer node as at a specified date.

fCMN_TRECustNodeAccountClose_R~()

Closes an account for a customer node from a specified date onwards.

fCMN_TRECustNodeAccountRetrieve_R?{}()

Retrieves account details for a specific account for a customer node.

fCMN_TRECustNodeContactAdd_R&()

Adds a person to the list of contacts for the customer node.

fCMN_TRECustNodeContactDetach_R~()

Detaches (removes) a person from a customer's contact list.

fCMN_TRECustNodeContactDetR_R~()

Detaches (removes) a person from a customer's contact list and deletes the person.

fCMN_TRECustNodeContactRetrieve_R?{}()

Retrieves the details for a specified contact for a customer node.

fCMN_TRECustNodeContactUpdate_R~()

Updates a person on the list of contacts for a customer node.

fCMN_TRECustNodeContractAdd_R&()

Adds a contract for a specified customer node.

fCMN_TRECustNodeContractCancel_R&()

Cancels a contract for a specified customer node.

fCMN_TRECustNodeContractRetrieve_R?{}()

Retrieves contract details for a specified contract for a customer node.

fCMN_TRECustNodeContractUpdate_R~()

Updates a contract for a specified customer node.

fCMN_TRECustNodeInvFormatAdd_R&()

Adds an invoice format to a specified customer node.

fCMN_TRECustNodeInvFormatDelete_R&()

Deletes an invoice format for a specified customer node.

fCMN_TRECustNodeInvFormatRetrieve_R?[]()

Retrieves all invoice format details for the specified customer node.

fCMN_TRECustNodeInvFormatUpdate_R~()

Updates an invoice format for a specified customer node.

fCMN_TRECustNodeListRowAdd_R&()

Adds a row to the specified customer list.

fCMN_TRECustNodeListRowDelete_R&()

Deletes a row from a specified customer list.

fCMN_TRECustNodeListRowDeleteMany_R&()

Deletes a row from a specified customer list.

fCMN_TRECustNodeListRowUpdate_R~()

Updates a row from a specified customer list.

fCMN_TRECustNodePersonRD_R~()

Replaces the current person for a customer with another person and deletes the original person.

fCMN_TRECustNodePersonReplace_R~

Replaces the current person for a customer with another person.

fCMN_TRECustNodePersonRetrieve_R?{}

Retrieves a customer's primary contact details (a person record).

fCMN_TRECustNodePersonUpdate_R~

Updates the person details for a customer node.

fCMN_TRECustomerCreate_R&

Creates a customer node and related customer entities.

fCMN_TRECustomerNodeCancel_R&

Cancels a customer as at a specified date.

fCMN_TRECustomerNodeHierarchyRetrieve_R?{}

Retrieves the hierarchy of customer nodes for a given customer.

fCMN_TRECustomerNodeRetrieve_R?{}

Retrieves the customer node details and related customer entities.

fCMN_TRECustomerNodeSearch_R?[]

Searches for customer nodes matching specified search parameters.

fCMN_TRECustomerNodeSearchLike_R?[]

Searches for customer nodes matching specified search parameters.

fCMN_TRECustomerNodeUpdate_R~

Updates customer node details.
