treEventSetEpmHandlerx

C

TRE_RESULT treEventSetEpmHandlerx(void (*Handler)(char *EventName, EPM_VALUE *FieldValues))

 

Object Pascal

function treEventSetEpmHandlerx(Handler: TtreEventEpmHandlerx): TRE_RESULT

 

Perl

Not applicable.

 

Purpose

Defines the handler function in the client for receiving event notifications in ATAMSG and FML32 format in the client's current context. If the Handler value is null, the handler becomes undefined. Only one handler can be called.

For the Object Pascal interface, the TtreEventEpmHandlerx type is defined as follows:

TtreEventEpmHandlerx = procedure(EventName: Pchar; FieldValues:
    PepmVariant);
    cdecl;

Tuxedo system events, and events posted using the treEventPostx() and treEventPostWaitx() functions can be interpreted by this handler.

Parameters

Parameter

Description

Handler

In: Name of the locally-defined function used as the event handler.

 

Returns

TRE_SUCC if successful; TRE_FAIL otherwise. The TRE sets treErrno to indicate the type of error.

Remarks

The method of event notification specified when calling treConnectFlags() determines if treEventSetEpmHandlerx() is called. The options are:

  • TRE_POLL (default)

    treEventSetEpmHandlerx() is called as a result of polling to check for waiting messages. treEventSetEpmHandlerx() may be called zero or more times at each poll interval, depending on the number of pending events to the client.

  • TRE_SIGNAL (UNIX only)

    treEventSetEpmHandlerx() is called immediately when the message arrives and a SIGUSER2 signal is received. This method is not valid when the TRE is used in multi-context mode.

  • TRE_THREAD (UNIX only)

    treEventSetEpmHandlerx() is called immediately when the message arrives.

Unknown hash

Class-specific attribute values, determined by the value of the TA_CLASS attribute.

The keys of the hash correspond to the names of the attributes and the value for each key is either of the following:

  • The value of the related attribute (if the related attribute has only one value in the event)
  • An array of all the values of the related attribute (if the related attribute has two or more values in the event).

If the event class does not support specific attributes, the hash is empty.
