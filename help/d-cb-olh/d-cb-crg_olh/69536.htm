WorkflowEngine

This interface type is used for generating configuration settings for the Singleview Workflow Engine. When an interface definition based on this interface type is saved, the TREWFE and associated REMOTE_SERVICE configuration items are automatically created.

Note:

To create interface definitions based on this interface type, users must have security group access level for 'View decrypted data that has been encrypted using this security group'.

 

Interface Type Parameters

Tab

Parameter

Type

Description

Process Details

 

Enabled

List

Specifies if the Singleview Workflow Engine is automatically started by the BMP.

Default is True.

SV Instance

[List]

Singleview instance on which the Singleview Workflow Engine runs in a multi-instance environment.

Default is MASTER.

Note:

Ignored if Cluster Managed is True.

TREWFE Sequence Number

Integer

Sequence number that identifies the TREWFE configuration item.

By default, it is set to the highest value of the next available sequence number of the following configuration items:

  • TREWFE
  • REMOTE_SERVICE.

    Note:

    The sequence number of the configuration items always match.

SV Username

Text

Singleview user name for logging in to the Singleview Workflow Engine.

SV Password

Text

Password for the specified SV Username.

It is masked.

TRE Client Name

[Text]

Name displayed by Tuxedo to identify the Singleview connection to the Singleview Workflow Engine.

Default is TREWFE.

THP Instance

List

Name of the THP configuration item to which the Singleview Workflow Engine connects.

Default is THP1.

THP Group

List

THP group to associate with the Workflow configuration item in a clustered environment.

Process Environment

[Text]

Environment variables to be set before the Singleview Workflow Engine executes.

Command Line Args

[Text]

Command line arguments that the BMP passes to the Singleview Workflow Engine.

Java Options

[Text]

Java Virtual Machine (JVM) options added to the Singleview Workflow Engine Java execution.

Wait Period

[Integer]

Time, in seconds, that the BMP waits after starting the Singleview Workflow Engine before it starts the next process.

Cluster Managed

List

Indicates if this instance is managed by a high availability configuration, and determines the number of REMOTE_SERVICE configuration items generated.

Default is False. If set to:

  • False, only one REMOTE_SERVICE configuration item is created
  • True, a REMOTE_SERVICE configuration item is created for each instance in the InstanceStatus derived attribute table of type Billing or Combined.

Default SFM Partition

[Integer]

Partition number to use when inserting scheduled functions for work items.

Can only be set if Cluster Managed is set to True.

The specified value must:

  • Correspond to a row in the CustomerPartition derived attribute table representing a billing partition. That is, where the billing instance is set to the value specified in the SV Instance field and all other columns are blank.
  • Not be the default partition number for SV Instance, which is configured in the InstanceStatus derived attribute table.

    Note:

    Used in high availability configurations where the instance name associated with the partition number will change after a failover. When the SFM on the new active instance restarts, it processes scheduled functions inserted with this partition number.

Duplicate Detection Period

[Integer]

Time, in seconds, during which duplicate workflows with matching process names and parameters will be rejected with an error.

It defaults to 900 seconds (15 minutes). If set to 0, duplicate workflows are allowed.

TRE Listener

TREPROXY Server Group

[Text]

Tuxedo group of the treproxy server.

Default is IMP.

TREPROXY Port

Text

Port used by the treproxy server to connect to the Singleview Workflow Engine.

The value can be one of:

  • Port number
  • Service name, as defined in /etc/services.

Number Concurrent Requests

Integer

Number of treproxy servers to create.
