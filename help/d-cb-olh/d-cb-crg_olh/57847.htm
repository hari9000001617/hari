PRODUCT_OFFER_TRE_V

The PRODUCT_OFFER_TRE_V view provides details about product offers.

Column Definitions

Column Name

Properties

Description

PRODUCT_OFFER_ID

NUMBER (8)

Not null

ID of the product offer.

LAST_MODIFIED

DATE

Not null

Date-time that the row was last modified.

OPERATOR_ID

NUMBER (8)

Not null

User who last modified the product offer.

GROUP_ID

NUMBER (8)

Not null

User group that has access to the product offer.

PRODUCT_OFFER_NAME

VARCHAR (60)

Not null

Name of the product offer.

PRODUCT_OFFER_DISPLAY_NAME

VARCHAR (120)

Not null

Display name for the product offer.

DESCRIPTION

VARCHAR (4000)

Not null

Description of the product offer.

AVAILABLE_START_DATE

DATE

Not null

Date-time at when the product offer becomes available.

AVAILABLE_END_DATE

DATE

Not null

Date-time after which the product offer is no longer available.

It must be equal to or later than AVAILABLE_START_DATE.

PRODUCT_ID

NUMBER (8)

ID of a base product that this product offer is assigned to.

PRODUCT_OFFER_SKU

VARCHAR (40)

Stock-keeping unit code for the product offer.

It is a short, unique code that can be used to search for a product offer.

No two product offers with the same TENANT_ID can have the same PRODUCT_OFFER_SKU.

CHILD_PRODUCT_OFFER_MIN

NUMBER (4)

Minimum number of child product offers that must be selected as part of the product offer when it is sold.

If specified, it must be greater than or equal to zero.

CHILD_PRODUCT_OFFER_MAX

NUMBER (4)

Maximum number of child product offers that can be selected as part of the product offer when it is sold.

If specified, must be greater than or equal to that of CHILD_PRODUCT_OFFER_MIN.

COMPANION_PRODUCT_MIN

NUMBER (4)

Minimum number of optional companion products that must be selected as part of the product offer when it is sold.

If specified, must be greater than or equal to zero.

COMPANION_PRODUCT_MAX

NUMBER (4)

Maximum number of optional companion products that can be selected as part of the product offer when it is sold.

If specified, must be greater than or equal to that of COMPANION_PRODUCT_MIN.

CATEGORY_CODE

NUMBER (4)

Category to which the product offer belongs.

The PRODUCT_OFFER_CATEGORY reference type defines the possible values.

SUB_CATEGORY_CODE

NUMBER (4)

Sub-category to which the product offer belongs.

The PRODUCT_OFFER_SUB_CATEGORY reference type defines the possible values.

It can only contain a value if CATEGORY_CODE is not NULL.

CUSTOMER_NODE_TYPE_ID

NUMBER (8)

Restricts the customer types for which the product offer is available for sale.

SERVICE_SPECIFIC_IND_CODE

NUMBER (1)

Specifies that the product offer is only available for sale with an individual service.

If set to 1 (TRUE) this product offer:

  • Is a service-specific product offer and can only contain other service-specific product offers or service-specific companion products. All apply to the same service.
  • Must be compatible with at least one service type.

If not NULL, it must have a value of 1.

HIDDEN_IND_CODE

NUMBER (1)

Specifies the product offer as hidden.

If not NULL, it must be set to 1 (TRUE).

If this column is set to 1 (TRUE), the product offer is not displayed in the Product Instance Detail form.

This can only happen where:

  • The product offer is not a top-level offer.
  • All companion products and child offers on the offer are mandatory.
  • The offer is mandatory on any parent product offers.
  • All product offers beneath the current product offer in its hierarchy are mandatory, contain only mandatory components, and are set as hidden.

AUTO_EXPAND_HIERARCHY_IND_
CODE

NUMBER (1)

Determines whether any descendant product offers are automatically in expanded mode when displayed in a form.

VERSION_STR

VARCHAR (255)

Optional version identification string that can be used to tag the product offer.

It is provided for use by third party configuration management tools, and is not directly used by Convergent Billing.

CATEGORIES

VARCHAR (255)

Comma-separated list of categories associated with the product offer.

PRODUCT_OFFER_HELP_URL

VARCHAR2 (4000)

Link for use with a Customer Service interface, to a page providing further information about the product offer.

TENANT_ID

NUMBER (8)

ID of the optional tenant associated with this product offer.

When TENANT_ID is inserted, it cannot be updated.
