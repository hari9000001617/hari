call_stack()

Syntax

Interface 1

 

Interface 2

 
call_stack()

 

 
call_stack(
    var stack?[])

Description

Retrieves the details of the current call stack.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

stack?[]

Out: Two-dimensional array corresponding to the call stack.

Each element in the returned array consists of a three-dimensional array.

 

Returns

Interface 1 returns a formatted text output of the current stack call.

Interface 2 returns the size of the call stack.

Note:

The value is zero if the current expression or function has no caller.

Remarks

For interface 1, if an expression output in the call stack is longer than 80 bytes, the interface outputs the first 80 bytes of the expression, and an ellipsis (...) is appended to indicate that is has been truncated.

Sample output:

[1] Function: Factorial&(), line 18

[2] Function: Factorial&(), line 18

[3] Function: Factorial&(), line 18

[4] Function: Factorial&(), line 18

[5] Expression: Factorial&(5), line 1

The following table lists properties of the returned three-dimensional arrays.

 

Properties of Three-dimensional Arrays

Element

Data Type

Description

0

String

Name of the function, or the full expression.

1

Line number in the function or expression where the call was made.

2

Integer

Indicates if the caller is a function (1)(TRUE) or expression (0)(FALSE)
