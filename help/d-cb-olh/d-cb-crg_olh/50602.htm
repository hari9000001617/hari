Description

Deletes a row of data in a customer derived attribute array, identified by the supplied keys.

Context

The following groups of variables are available throughout all stages of processing customer derived attribute delete row request, except where otherwise specified:

  • System Client
  • Operator
  • Miscellaneous
    • evFormCode$
    • evLastRowInd&, available at all stages of processing except evaluation of customer and derived attribute pre-commit expressions
    • evRowStatus$ contains the value Delete.
  • Customer
    • Available only during customer processing stages and evaluation of customer derived attribute pre-commit expressions.
  • Derived attribute
    • Available only during customer derived attribute processing stages.

Parameters

Parameter

Description

Keys?{}

Keys that identify the customer derived attribute row to delete.

LastModified~

Initial value of this parameter must match the last modified date of the customer record identified by Keys?{}.

If Keys?{} is specified, LastModified~ can be undefined. If LastModified~ is undefined, the last modified value in CUSTOMER_NODE_HISTORY_V for the specified StartDate~ is used.

On successful return, LastModified~ represents the date-time of this deletion.

StartDate~

Date on which the deletion becomes effective.

If not specified, this parameter defaults to the current date.

EndDate~

Date on which the deletion ceases to be effective.

If not specified, this parameter defaults to the end of time.

IdentifierKeys?{}

Values of the customer node derived attribute row identifier fields derived from Keys?{}.

 

Processing

The entity lookup function specified for customer derived attributes is called to determine the customer node ID, derived attribute ID, sequence number, and effective date of the customer derived attribute row. If a unique row is not found an error is raised.

Processing continues as if fEV_CustomerDADeleteRows&() is called for a single row, using the determined IDs and dates as parameters.

Returns

A non-zero value if deletion was successful; otherwise, an appropriate error.
