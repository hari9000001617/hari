Description

Updates the date ranges of a derived attribute table.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

Id&

In: ID of DERIVED_ATTRIBUTE_HISTORY record.

OrigDates~[]

In: Current date ranges.

Contains a set of start dates and a final end date that corresponds to date ranges covered by the derived attribute history records.

DesiredDates~[]

In: Required date ranges.

Contains a contiguous set of date ranges, which consists of a set of start dates, and a final end date.

 

Returns

Interface 1: ID of the new entity; otherwise, an error is raised.

Interface 2: 1, if successful; otherwise, an error is raised.

Remarks

Interface 1

Interface 1 does the following:

  • Prepares the derived attribute for subsequent calls to biDerivedAttributeUpdate&().
  • Does not necessarily set the date ranges of the derived attribute to the supplied set of date ranges. It guarantees that the derived attribute records only have the start and end dates described by DesiredDates~[]. Existing date ranges that do not have a match in DesiredDates~[] are compressed or deleted, while other ranges are expanded to fill resulting gaps.

Interface 2

Interface 2 moves the:

  • Date boundaries of the history records to exactly match the required dates, as specified.
  • Start dates of the table attached to the derived attribute, if present, to match the date ranges of the parent derived attribute. If a table is subdivided into smaller date ranges, the following changes are made if a table record has:
    • Start and end dates that lie between the old and new dates of a moved parent date boundary, the records are deleted.
    • A start date within the date spanned by a parent date boundary move and an end date outside this date span, the record changes to have a start date equal to the parent's new end date.
    • An end date within the date spanned by a parent date boundary move and a start date outside this date span, the record changes to have an end date equal to the parent's start date.
