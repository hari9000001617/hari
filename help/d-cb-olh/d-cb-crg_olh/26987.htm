fCMN_TREProductInstanceRetrieve_R?{}()

Syntax

fCMN_TREProductInstanceRetrieve_R?{}(
    l_ProductInstanceId&,
    l_EffectiveDate~)

 

Description

Retrieves product instance details (PRODUCT_INSTANCE_TRE_V view) and related product instance entities, as displayed in [[Error! Caption does not exist: Figure 26986-1]].

Attributes

Application Environment

Any

Context

Any

 

Parameter

Description

l_ProductInstanceId&

In: Product instance ID of the product instance details to retrieve.

l_EffectiveDate~

In: Effective date of the search. If not specified, the current date is used.

 

Returns

Hash if successful; otherwise, an exception is raised.

Remarks

Cancelled services are not retrieved.
