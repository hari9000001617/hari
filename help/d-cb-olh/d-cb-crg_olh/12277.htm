ChargeItemNumber&()

Syntax

ChargeItemNumber&(
    InvoiceId&,
    ServiceId&,
    Type$,
    Name$)

 

Description

Returns the invoice item number of the charge specified.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

InvoiceId&

In: ID of the invoice.

ServiceId&

In: ID of the service or 0 for no service ID.

Type$

In: Type of charge. Valid values are:

  • T (Tariff)
  • S (Subtotal).

Name$

In: Name of the tariff or subtotal.

 

Returns

Invoice item number of the charge; an integer.
