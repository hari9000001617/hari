biNEFExportGetHeaderData&()

Syntax

biNEFExportGetHeaderData&(
    const NEFFile?[],
    var NEFData?{},
    var HeaderData?{})

 

Description

Retrieves the data required to create a header record in the export file for normalised event files.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

NEFFile?[]

In: Array of hashes containing details of the normalised event files for which charges are to be exported.

The table below lists hash keys.

NEFData?{}

Out: Hash of hashes.

The table below lists hash keys.

HeaderData?{}

Out: Hash of data keyed by NORMALISED_EVENT_FILE.

It is used to generate the header for the export file.

 

Returns

1 if successful; otherwise, an error is raised.

Remarks

Details of normalised event records may appear multiple times if the event results in multiple charge records. The exported record may have event details with no corresponding charge details.

The following table lists NEFFile?[] hash keys.

 

NEFFile?{} Hash Keys

Key

Description

File_ID

Normalised event file ID.

MIN_DATE

Optional. Earliest charge start date.

MAX_DATE

Latest charge end date. It is only available if MIN_DATE is set.

 

The following table lists NEFData?{} hash keys used to create a header record parameter.

 

NEFData?{} Hash Keys

Key

Type

Description

NORMALISED_EVENT_FILE_ID

Integer

Normalised event file ID.

RecordCount

Integer

Number of records in the normalised event file.

EventCount

Integer

Number of events in the normalised event file.

ChargeCount

Integer

Number of charges in the normalised event file.
