Variables Based on INV_PENDING_CONSOLIDATION_V

All direct variables based on the INV_PENDING_CONSOLIDATION_V view have an application environment of Invoicing and a context of Customer Node.

Variable Definitions

Variable

Type

Description

InvAccountCurrencyId&

Integer

Currency of the account, and in which the invoice is issued.

InvAccountId&

Integer

ID of the account to which this invoice was applied.

InvAccountTypeId&

Integer

ID of the account type.

InvAccountTypeName$

String

Name of the account type.

InvAccountTypeDisplayName$

String

Name of this type of account displayed. Defaults to InvAccountTypeName$.

Caution:

The value of this column can be translated into a locale-specific language and therefore differ between implementations.

InvBillRunId&

Integer

ID of the bill run that generated this invoice.

InvCustomerNodeId&

Integer

Customer node that received this invoice, derived from the ACCOUNT_ID.

InvCustomerNodeTypeId&

Integer

ID of the customer node type.

InvCustomerNodeTypeName$

String

Name of the customer node type.

InvCustomerNodeTypeDisplayName$

String

Name of the customer node type displayed.

Defaults to InvCustomerNodeTypeName$.

Caution:

The value of this column can be translated into a locale-specific language and therefore may differ between installations.

InvInvoiceTypeId&

Integer

ID of this invoice type.

InvInvoiceTypeName$

String

Name of the invoice type.

InvMaxBillRunEffectiveDate~

Date

Maximum bill run effective date associated with the statements pending consolidation.

InvMaxBillRunIssueDate~

Date

Maximum issue date associated with the statements pending consolidation.

InvMinBillRunEffectiveDate~

Date

Minimum bill run effective date associated with the statements pending consolidation.

InvMinBillRunIssueDate~

Date

Minimum issue date associated with the statements pending consolidation.

InvQA&

Integer

Set to 1 (TRUE) if this is an invoice or statement generated for a Quality Assurance or Statement-only bill run.

If not 1, it must be Null.

Records from bill runs with this variable set are not applied to accounts.

InvRootCustomerNodeId&

Integer

Root customer node associated with the account.

InvStatementAmount#

Real

Total amount of statements currently pending consolidation for this account.

The amount is in the currency of the account.

InvStatementCount&

Integer

Number of statements currently pending consolidation for this account.

InvTransferredStatementAmount#

Real

Total amount of statements pending consolidation for other accounts that have been transferred to this account.

The amount is in the currency of the account.

InvTransferredStatementCount&

Integer

Number of statements pending consolidation for other accounts that have been transferred to this account.
