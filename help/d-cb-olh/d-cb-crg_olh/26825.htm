fCMN_TRECustomerNodeRetrieve_R?{}()

Syntax

fCMN_TRECustomerNodeRetrieve_R?{}(
    l_CustomerNodeId&,
    l_EffectiveDate~)

 

Description

Retrieves customer node details (CUSTOMER_NODE_TRE_V view) and related customer entities.

Attributes

Application Environment

Any

Context

Any

 

Parameter

Description

l_CustomerNodeId&

In: Customer node ID of the customer node details to retrieve.

l_EffectiveDate~

In: Effective date of the search.

If not specified, the current date is used.

 

Returns

Hash if successful; otherwise, an exception is raised.

Remarks

The following table lists keys of the returned hash.

 

Keys of Returned Hash

Key

Data Type

Description

{<CUSTOMER_NODE_TRE_V>}

Various, depending on the attribute types in the entity validation for the customer node type.

Details for the customer node using the fields from CUSTOMER_NODE_TRE_V with the field name as the <CUSTOMER_NODE_TRE_V> hash ID.
