Variables Based on NORMALISED_EVENT_FILE_V

All direct variables based on the NORMALISED_EVENT_FILE_V view have an application environment of Data Entry and a context of Any.

Variable Definitions

Variable

Type

Description

NefAtlantaOperatorId&

Integer

User who initiated the processing of the file.

NefCorruptEventCount&

Integer

Number of events from this file that could not be decoded during normalisation.

An event is 'decoded' once the values of its data attributes have been assigned successfully to direct variables associated with the NORMALISED_EVENT_V database view.

NefDecodedEventCount&

Integer

Number of events from this file decoded successfully during normalisation.

The value of this variable is the sum of the variables:

  • NefDiscardedEventCount&
  • NefRatedEventCount&
  • NefUnratedEventCount&
  • NefNormErrorEventCount&
  • NefRatingErrorEventCount&
  • NefTracerEventCount&.

NefDeletedErrorEvent
Count&

Integer

Number of error events that have been deleted and not reprocessed.

NefDeletedEventCount&

Integer

Number of successfully rated events that have been deleted (and their charges), and not reprocessed.

NefDiscardedEventCount&

Integer

Number of events discarded during normalisation or after being rated.

NefEventSource$

String (40)

ID of the source from which this group of normalised events originated.

NefExternalInputEvent
Count&

Integer

Number of events that should have been processed in association with this file.

This can be defined in the file header or specified by the external event feed. For some logical grouping of normalised events (for example, hot rating), it may not be specified.

When rating is complete, this variable should equal NefInputEventCount&. If it does not equal NefInputEventCount& on file completion, the Database Validation Process reports a warning that the file may be corrupted.

NefFileName$

String (255)

If the file was processed on the rating machine, it is the name of the file (excluding the directory component).

If a logical file (corresponding to a logical grouping of records passed to the system by a third party), the name is defined by the remote system.

NefInputEventCount&

Integer

Number of events that have been processed in association with this file.

Its value is the sum of NefDecodedEventCount& and NefCorruptEventCount&.

NefNormalisedEventFileId&

Integer

ID of this normalised event file.

NefNormalisedEventFtId&

Integer

Normalised event file type associated with this file.

This specifies the mechanism by which the events in the file were decoded.

NefNormErrorEventCount&

Integer

Number of events from this file that could not be normalised by an ENM process due to an error condition.

NefProcessEndDate~

Date

Date-time when all events from this file were processed.

NefProcessStartDate~

Date

Date-time when events from this file commenced to be processed.

NefRatedEventCount&

Integer

Number of events from this file successfully rated.

NefRaterVersion$

String (20)

Version of the rating engine that rated this file (for example, 4.00.07).

NefRatingErrorEventCount&

Integer

Number of events from this file that could not be rated by one or more ERT processes due to an error condition.

The value of the variable is the total count of all error events.

NefReprocessedErrorEvent
Count&

Integer

Number of error events that have been reprocessed and deleted from the NORMALISED_EVENT_ERROR table (and therefore no longer directly associated with this file).

NefReprocessedEventCount&

Integer

Number of successfully rated events that have been reprocessed and deleted from the NORMALISED_EVENT table (and therefore no longer directly associated with this file).

NefTaskQueueId&

Integer

ID of the task that rated this file (if applicable).

NefTracerEventCount&

Integer

Number of events from this file flagged as tracer events by an ENM process.

NefUnratedEventCount&

Integer

Number of events from this file flagged by an ENM process as not to be rated.
