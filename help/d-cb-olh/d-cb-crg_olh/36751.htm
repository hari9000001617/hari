eval_cache_stats()

Syntax

eval_cache_stats()

 

Description

Returns statistics for cached expressions used by eval() and parse() calls.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

None.

 

 

Returns

Unknown hash of statistics.

Remarks

The following table lists keys that are present in the returned hash.

 

Keys Present in Returned Hash

Key

Type

Description

Accesses

Real

Number of accesses to cached expressions.

BytesUsed

Real

Approximate number of bytes unused by all cached expressions (in a multi-threaded process, this value can be inaccurate).

Hits

Real

Number of times a matching expression was identified in the cache.

Inserts

Real

Number of parsed expressions inserted into the cache.

It is usually equal to Misses.

In a multi-threaded process, it can be less than Misses if two threads attempt to insert the same cached expression.

Items

Real

Number of cached parsed expressions.

MaxItems

Real

Maximum number of cached expressions supported.

Note:

MaxItems can be changed using the eval_cache_maxsize() function.

Misses

Real

Number of accesses to the cache that did not find a suitable cached expression (equal to Accesses - Hits).

Purges

Real

Number of parsed expressions purged (resulting from a call to eval_cache_maxsize()).

Replacements

Real

Number of times a parsed expression was automatically purged because the cache was full and the LRU algorithm deleted an expression.
