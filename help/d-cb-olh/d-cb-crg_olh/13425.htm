Description

Interface 1 appends a message string to the Convergent Billing log.out file.

Interfaces 2 and 3 convert the message text of the message specified by ID or message name, using the specified parameters.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

MessageId&

In: ID of the message to convert.

MessageName$

In: Name of the message to convert.

MessageString$

In: Message text to append to the log.out file.

It must be in the format of a message string returned by the MessageString$() function (<error number><string>).

For example, MessageLog&('A dummy message') silently fails, but MessageLog&('<E00001> A dummy message') succeeds.

Params?[]

In: Substitution parameters.

Message text inside angled brackets (for example, <text>) is substituted, in turn, by each parameter value.

Alternatively, message text brackets can be numbered, using <number:text>, according to their order of substitution by the parameters.

 

Returns

1 (TRUE).
