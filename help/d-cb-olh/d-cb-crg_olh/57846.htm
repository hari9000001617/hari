PRODUCT_OFFER_PRODUCT_TRE_V

The PRODUCT_OFFER_PRODUCT_TRE_V view defines the companion products contained in each product offer.

Column Definitions

Column Name

Properties

Description

PRODUCT_OFFER_ID

NUMBER (8)

Not null

ID of the product offer.

PRODUCT_ID

NUMBER (8)

Not null

ID of a companion product compatible with the product offer.

SEQNR

NUMBER (4)

Not null

Controls the display order of the product in the current product offer.

DESCRIPTION

VARCHAR (255)

Description of the compatibility between the companion product and product offer.

MANDATORY_IND_CODE

NUMBER (1)

If set to 1 (True), this companion product must be instantiated for a base product instance that is assigned either this product offer or a product offer that includes this product offer as a child product offer. Otherwise, the inclusion of this companion product is optional.

If not NULL, it must have a value of 1.

GENERAL_1 - GENERAL_10

VARCHAR (255)

General fields 1-10.

They contain values that override default values for GENERAL_1 to GENERAL_10 for an instance of this companion product that is created as part of this product offer.

DEFAULT_IND_CODE

NUMBER (1)

If set to 1 (TRUE), the companion product is automatically selected by default when ordering the product offer. The companion product can be deselected during the ordering process.

If it is not NULL, it must be 1 (TRUE).

If MANDATORY_IND_CODE is 1, it must be NULL.
