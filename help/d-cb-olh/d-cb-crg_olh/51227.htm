biTuxedoServerStatistics?{}()

Syntax

biTuxedoServerStatistics?{}()

 

Description

Returns statistics for Tuxedo server processes.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

None.

 

 

Returns

Hash of Tuxedo server statistics listed in the following table.

 

Statistics Returned by biTuxedoServerStatistics?{}()

Key

Type

Description

Instance

String

Name of the Convergent Billing instance from which these statistics were generated.

It is only present when the function is called in a multi-instance environment.

TotalServers

Integer

Total number of server processes.

ActiveServers

Integer

Total number of server processes that are not idle.

Server

Array

General statistics for all running servers in the domain, comma ordered by Machine, Group, and ServerId.

The table below lists hash values.

Active

Array

Statistics for servers that are processing requests, comma ordered by Machine, Group, and ServerId.

The table below lists hash values.

 

Server Hash Values

Key

Type

Description

Machine

String

Logical machine on which the server is running.

Group

String

Logical name of the server group.

If there is no name for the group (for example, for processes such as the BBL and BRIDGE), the group ID of the server is returned as a string.

GroupId

Integer

ID of the server group.

ServerId

Integer

ID of the server in the server group.

ProcessId

Integer

ID of the UNIX system process.

MessageQueueId

Integer

ID of the UNIX message queue being used by the server.

Requests

Integer

Number of requests processed by the server since it started.

ServerName

String

Name of the server executable file, including its full path.

CLOPT

String

Command-line options passed to the server since it is was started.

 

Active Hash Values

Key

Type

Description

Machine

String

Logical machine on which the server is running.

Group

String

Amount of free space in the cache in bytes.

On creation, this is 24 bytes greater than PoolSize.

GroupId

Integer

ID of the server group.

ServerId

Integer

ID of the server in the server group.

Requests

Integer

Number of requests processed by the server since it started.

CurReq

Integer

Number of outgoing requests from the server waiting for responses.

NumReq

Integer

Total number of outgoing requests made by the server.

LastGrp

Integer

Tuxedo group ID of the server that received the last outgoing request.

TimeLeft

Integer

Time left (in seconds) before the outgoing request times out.

CurRequest

String

Service being processed by this server.

MpLoad

Integer

Current load on this server (MP mode only).

Queued

Integer

Number of requests waiting to be processed by the server (SHM mode only).
