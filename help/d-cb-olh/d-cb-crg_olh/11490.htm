biBillRunInvoiceRevoke&()

Syntax

biBillRunInvoiceRevoke&(
    BillRunId&,
    EffectiveDate~,
    EffectiveDayOfMonth&,
    BillRunOperationId&,
    QAInd&,
    RootCustomerNodeList&[],
    var SuccessCustomerNodeList&[],
    var ErrorCustomerNodeList&[],
    var SuppressedCustomerNodeList&[],
    var OperationStatistics?{})

 

Description

Performs the revoke invoices or statements billing operation for a set of customers in a bill run.

This is a billing configuration-compliant remote wrapper function around biInvoiceRevoke&().

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

BillRunId&

In: ID of the bill run to be processed.

EffectiveDate~

In: Effective date of the bill run to be processed.

EffectiveDayOfMonth&

In: Logical day of the month to associate with EffectiveDate~.

It is usually set to the day of the month of EffectiveDate~. However, if EffectiveDate~ is the last day of the month, EffectiveDayOfMonth& may be a later day.

BillRunOperationId&

In: ID of the bill run operation to be processed.

QAInd&

In: Indicator to identify whether the bill run to be processed is a real bill run or a QA bill run.

A non-zero value (TRUE) indicates a QA bill run.

RootCustomerNodeList&[]

In: List of root customer nodes to be processed.

The list must contain at least one customer node.

SuccessCustomerNodeList&[]

Out: List of all root customer nodes successfully processed.

The list is a subset of RootCustomerNodeList&[] and is in the same order as RootCustomerNodeList&[].

ErrorCustomerNodeList&[]

Out: List of root customers not successfully processed.

SuppressedCustomerNodeList&[]

Out: List of root customers that have been suppressed.

OperationStatistics?{}

Out: Operation statistics in an unknown hash.

The table below lists the hash keys.

 

Returns

Number of customers successfully processed; otherwise, an error is raised.

Remarks

The function aborts if the number of errors detected during the operation exceeds the threshold provided by BillRunMaxRevokeErrorsPerBatch&().

The following table lists the OperationStatistics?{} parameter hash keys.

 

OperationStatistics?{} Parameter Hash Keys

Key

Value

AMOUNT

Total amount of all invoices revoked in the bill run operation.

GENERAL_STATS1

Number of invoices deleted in the bill run operation.

GENERAL_STATS2

Number of statements deleted in the bill run operation.

GENERAL_STATS3

Number of invoice images deleted in the bill run operation.

GENERAL_STATS9

Number of charges updated in the bill run operation.

GENERAL_STATS10

Number of charges deleted in the bill run operation.
