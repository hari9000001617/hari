ClientDynamicFormSimple&()

Syntax

ClientDynamicFormSimple&(
    l_ParamNames$[],
    l_ParamValues?[])

 

Description

Displays a plain dynamic form that displays simple, read-only information in a long text field.

Attributes

Application Environment

Data Entry

Context

Any

Method

Local

 

Parameter

Description

l_ParamNames$[]

In: Array of the parameter names to display.

l_ParamValues?[]

In: Array of values corresponding to the parameter names.

 

Returns

ID of the button pressed if the dynamic form is set to display modally (FORM.MODAL=TRUE); otherwise, 0 (FALSE).

Remarks

The following items can also be displayed:

  • Form caption
  • Prompt label
  • Buttons (including caption).

No validation takes place on this form. The form can be configured to display not only plain-text information, but also an HTML page.

Buttons

Parameter Name

Properties

Description

BUTTON.ALIGNRIGHT

Boolean

Defaults to FALSE

Right alignment of the button.

If TRUE, the button is aligned to the right-hand edge of the form.

BUTTON.CANCEL

Boolean

Defaults to FALSE

Cancel state of the button.

If set to TRUE, pressing the button does not execute any validation expressions on the form.

The Close button on the menu bar effectively has BUTTON.CANCEL=TRUE, and returns an ID of 0.

BUTTON.CAPTION

String

Mandatory

Caption for a new button.

Setting BUTTON.CAPTION creates a new button and any further setting of BUTTON parameters act on that newly created button until BUTTON.CAPTION is set again.

BUTTON.DEFAULT

Boolean

Defaults to FALSE

Default state of the button.

If TRUE, the button is triggered when the Enter key is pressed and the cursor is in a field on the form.

BUTTON.ID

Integer

Mandatory

ID of the button.

If the form is called modally, the ID of the activated button is returned as l_ModalResult&[].

The Close button on the menu bar has an ID of 0.

BUTTON.NAVBUTTON

Boolean

Defaults to FALSE

State of the navigation button on the form.

If TRUE, the navigation button arrow is displayed on the form.

 

Form Structure

Parameter Name

Properties

Description

FORM.ALIGN

String

Specifies the position of the form.

Options are LEFT, RIGHT, TOP, BOTTOM.

Tip:

To anchor a child form to three edges of a parent form, use FORM.ALIGN and FORM.ANCHORS.

FORM.ANCHORS.BOTTOM

Ignored

Resizes a child form relative to the size of the parent form.

Sets the form anchor to the bottom of the parent form.

FORM.ANCHORS.LEFT

Ignored

Resizes a child form relative to the size of the parent form.

Sets the form anchors to the left of the parent form.

FORM.ANCHORS.RIGHT

Ignored

Resizes a child form relative to the size of the parent form.

Sets the form anchors to the right of the parent form.

Tip:

To horizontally resize the form when the parent form is resized, use FORM.ANCHORS.LEFT and .RIGHT.

FORM.ANCHORS.TOP

Ignored

Resizes a child form relative to the size of the parent form.

Sets the form anchors to the top of the parent form.

Tip:

To vertically resize the form when the parent form is resized, use FORM.ANCHORS.TOP and .BOTTOM.

FORM.BRINGTOFRONT

Boolean

Specifies that the form can be displayed on top of other active forms.

FORM.CAPTION

String

Mandatory

Caption displayed on the title bar of the form.

FORM.HEIGHT

Integer

Height of the form.

FORM.HELPFORNAME

String

Specifies that the form can be used to control the Form Field Help Dump form, and be available for F1 help and form help.

FORM.LEFT

Integer

Left location of the form.

FORM.MINHEIGHT

Integer

Minimum height of the form.

FORM.MINWIDTH

Integer

Minimum width of the form.

FORM.MODAL

Boolean

Defaults to TRUE

Specifies whether the form is displayed modally.

FORM.POSNFORMCODE

String

Creates a unique static value key used to identify the form and store its position in the system registry.

Note:

The string supports a maximum character length of 12 characters.

FORM.PROMPT

String

Text displayed in the prompt label.

FORM.REFRESH.INTERVAL

Integer

Refresh interval for the dynamic form.

The interval is only implemented for the web browser.

Note:

An interval of 0 does not activate the refresh timer.

FORM.REUSEFORMCODE

String

Code that identifies the form.

The form is reused if all of the following conditions are met:

  • FORM.REUSEFORMCODE is set.
  • The reuse flag is set (using ClientSetReuse&).
  • Another call is received to display a form with the same reuse form code.

If a form is reused, only parameters displayed after this parameter are used in the form.

FORM.TOP

Integer

Top location of the form.

FORM.WIDTH

Integer

Width of the form.

 

Text Area

Parameter Name

Properties

Description

MEMO.APPENDTEXT

String

Text to append to the memo field.

MEMO.BROWSER_EXPRESSION

String

EPM expression to evaluate for the HTML or SVG code display.

The expression expects a blob to be returned. The expression is re-evaluated when the form is refreshed.

MEMO.BROWSER_STREAM

String

HTML or SVG source code to display.

MEMO.SCROLLBOTTOM

(Value ignored)

Specifies how form scrolling is controlled.

If MEMO.SCROLLBOTTOM is listed as a parameter name, the form automatically scrolls to the bottom of the memo.

MEMO.SCROLLTOP

(Value ignored)

Specifies how form scrolling is controlled.

If MEMO.SCROLLTOP is listed as a parameter name, the form automatically scrolls to the top of the memo.

MEMO.SHOWHORZSCROLL

Boolean

Defaults to FALSE

Specified if the horizontal scroll bar is displayed.

If set to TRUE, the horizontal scroll bar is displayed on the memo.

MEMO.SHOWVERTSCROLL

Boolean

Defaults to FALSE

If the vertical scroll bar is displayed.

If set to TRUE, the vertical scroll bar is displayed.

MEMO.TEXT

String

Text to display in the memo field.

MEMO.TEXTFORMAT

String

Mandatory

Format for the text in the memo.

The following values are acceptable:

  • MONOSPACED

    Text uses regular spacing of letters and white spaces.

  • PROPORTIONAL

    Text uses proportional spacing of letters and white spaces.

  • HTML

    Memo consists of a HTML page taken from the address set in MEMO.URL and displayed in an Internet Explorer browser.

    Note:

    Internet Explorer must be installed on the workstation for HTML to work.

MEMO.URL

String

Displayed HTML page's URL address.

Note:

This parameter is referenced as a value in MEMO.TEXTFORMAT.

MEMO.WORDWRAP

Boolean

Defaults to TRUE

Specifies how text is displayed.

If set to TRUE, all text is word wrapped.
