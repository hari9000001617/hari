biPaymentReceiptPrint&()

Syntax

biPaymentReceiptPrint&(
    PaymentFields$[],
    PaymentValues?[],
    InvoiceFields$[],
    InvoiceValues?[],
    ItemFields$[],
    ItemValues?[])

 

Description

Produces a payment receipt using specified payment details, and prints it to the user's default printer.

Attributes

Application Environment

Data Entry

Context

Any

Method

Local

 

Parameter

Description

PaymentFields$[]

In: Array of payment field names.

PaymentValues?[]

In: Array of payment field values corresponding to the fields in PaymentFields$[].

InvoiceFields$[]

In: Array of invoice allocation field names.

InvoiceValues?[]

In: Two-dimensional array of invoice allocation field values for the payment, in [row][column] order.

ItemFields$[]

In: Array of upfront charge item field names.

ItemValues?[]

In: Two-dimensional array of upfront charge item field values for the payment, in [row][column] order.

 

Returns

1 (TRUE) if the payment receipt prints successfully; 0 (FALSE) if it does not; otherwise, an error is raised.

Remarks

The function is assigned the BASE_INSTALL security group and can be updated.

Field names passed in PaymentFields$[], InvoiceFields$[], and ItemFields$[] are listed in the following tables.
