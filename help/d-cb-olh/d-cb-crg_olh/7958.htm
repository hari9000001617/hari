INVOICE_CUSTOMER_NODE_V

The INVOICE_CUSTOMER_NODE_V view is a join between the INVOICE and CUSTOMER_NODE_INV_FORMAT tables. It is used by the INVOICE_FORMAT_V view.

Column Definitions

Column Name

Properties

Description

INVOICE_ID

NUMBER (15)

Not null

ID of an invoice cycle.

EFFECTIVE_DATE

DATE

Not null

Effective date of the bill run that generated this invoice.

ISSUE_DATE

DATE

Not null

Date on which this invoice was issued.

SEQNR

NUMBER (3)

Not null

Sequence number of a specific invoice format for this customer node.

Note:

For a given invoice, it may not be contiguous and may not start at 1.

INVOICE_FORMAT_ID

NUMBER (8)

Not null

ID of an invoice format.

PERSON_ID

NUMBER (15)

Not null

Person or contact who is to receive this invoice.

ADDRESS_CHOICE_CODE

NUMBER (2)

Not null

Address to use for this invoice or statement. Valid values are:

  • 1 (Customer postal address): This is the default for the first row.
  • 2 (Customer site address)
  • 3 (Contact's postal address): Default for second and subsequent statements.
  • 4 (Contact's home address)
  • 5 (Contact's work address).
