biTemplateListUpdate&()

Syntax

biTemplateListUpdate&(
    Opcodes&[],
    FieldNames$[],
    FieldValues?[])

 

Description

Maintains the TEMPLATE_HISTORY table.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

Opcodes&[]

Operations to perform on the row.

FieldNames$[]

In: Column names from the TEMPLATE_HISTORY table for which corresponding values are specified.

FieldValues?[]

In: Field values (output in [Row][Field] order) corresponding to the column name order specified in FieldNames$[].

 

Returns

Total number of templates inserted, updated, and deleted; otherwise, an error is raised.

Remarks

biTemplateDelete&(), biTemplateInsert&(), and biTemplateUpdate&() are called to perform:

  • Changes
  • Operation specified in Opcodes&[].

For the FieldNames$[] parameter, two extra values are supported within the column name list:

  • ELIGIBILITY_EXPR_LIST, which returns a list of all eligibility expressions used in the template.
  • POST_ELIGIBILITY_EXPR_LIST, which returns a list of all post eligibility expressions used in the template.

Note:

If these values are included in the FieldNames$[] parameter, the corresponding ID value must be included in the field list. The accepted values are ELIGIBILITY_EXPR_LIST_ID and POST_ELIGIBILITY_EXPR_LIST_ID.
