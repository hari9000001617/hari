fCMN_TREProductInstanceUpdate_R~()

Syntax

fCMN_TREProductInstanceUpdate_R~(
    l_ProductInstanceId&,
    l_EffectiveDate~,
    l_LastModDate~,
    l_ProductDetails?{})

 

Description

Updates the details of a specified product instance.

Attributes

Application Environment

Any

Context

Any

 

Parameter

Description

l_ProductInstanceId&

In: Product instance ID of the product instance to update.

l_EffectiveDate~

In: Effective date of the record to be updated.

l_LastModDate~

In: Last modified date of the product instance record.

It must match the last modified date for the product instance record being updated; otherwise, an exception is raised.

The last modified date can be retrieved using fCMN_TREProductInstanceSearch_R?[]().

l_ProductDetails?{}

In: Hash table specifying the updates for the product instance using fields from the PRODUCT_INSTANCE_TRE_V view with the field names as the hash keys.

 

Returns

Last modified date, if successful; otherwise, an error is raised.

Remarks

If the product offer details or the status of a companion product instance is updated, the associated base product instance can be flagged as requiring re-completion. The base product instance can be re-completed using the biProductInstanceComplete&() function.
