Variables Based on INV_COMPANION_PRODUCT_V

All direct variables based on the INV_COMPANION_PRODUCT_V view have an application environment of Invoicing and a context of Service.

Variable Definitions

Variable

Type

Description

InvCompanionBaseProductInstanceId&

Integer

ID of the base product instance associated with the companion product instance.

InvCompanionCustomerNodeId&

Integer

ID of the customer hierarchy element or node to which the companion product instance belongs.

InvCompanionEndDate~

Date

End of the effective date-time range for the row.

Note:

The effective end date-time may be less than the effective end date-time of the companion product instance.

InvCompanionGeneral1$ -
InvCompanionGeneral10$

String

General fields 1 - 10 associated with the companion product instance.

InvCompanionPreviousProductOfferId&

Integer

ID of the previous product offer associated with this product instance.

InvCompanionPreviousProductOffer
NodeId&

Integer

ID of the product offer hierarchy node from which the product instance was previously allocated.

InvCompanionProductDescription$

String

Description of the product.

InvCompanionProductDisplayName$

String

Display name of the product.

InvCompanionProductId&

Integer

ID of the product associated with the companion product instance.

InvCompanionProductInstanceId&

Integer

ID of the companion product instance.

InvCompanionProductInstanceStatus$

String

Status of the companion product instance.

InvCompanionProductInstanceStatus&

Integer

Status of the companion product instance.

InvCompanionProductLastModified~

Date

Date-time when the row was last modified.

InvCompanionProductName$

String

Name of the product.

Date~

Date-time at which this product instance was allocated from the product offer specified in InvCompanionProductOfferId&.

InvCompanionProductOfferId&

Integer

ID of the product offer from which this companion product instance was generated.

InvCompanionProductOfferNodeId&

Integer

ID of the product offer hierarchy node from which the product instance was allocated.

InvCompanionStartDate~

Date

Start of the effective date-time range for the row.
