BMP.EventNotify&()

Syntax

BMP.EventNotify&(
    EventName$,
    const Parameters?[])

 

Description

Default event handler function (specified by the BMP.EVENT_NOTIFY_FUNCTION configuration item attribute) for the BMP.

Note:

The function can be updated. If updated, the security group must be changed to prevent the function being overwritten in an upgrade.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

EventName$

In: Name of the event.

Parameters?[]

In: Values for the event.

For information about values, refer to the table below.

 

Returns

Value of TuxedoSystemEventHandler&() (for events starting with the .SysServer string), or 1 if successful; otherwise, an error is raised.

Remarks

The function handles unsolicited events received by the BMP. If the event name starts with the string .SysServer, the function calls the default Tuxedo system event handler function (TuxedoSystemEventHandler&()).

The following table lists values returned by Parameters?[]. For more information about the values associated with the content type, refer to the Tuxedo documentation.

The last array element, an unknown hash, contains various attributes (hash key) and the values (hash value) for each attribute applicable to an object of the specific TA_CLASS.

Class-specific attribute values, determined by the value of the TA_CLASS attribute.

The keys of the hash correspond to the names of the attributes, and the value for each key is either of the following:

  • Value of the related attribute (if the attribute has only one value in the event)
  • Array of all the values of the related attribute (if the attribute has two or more values in the event).

If the event class does not support specific attributes, the hash is empty.
