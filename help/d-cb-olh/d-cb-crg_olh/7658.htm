treEventCheck()

C

TRE_RESULT treEventCheck(INTSV *EventCount)

 

Object Pascal

function treEventCheck(var EventCount: Int64): TRE_RESULT

 

Perl

Not applicable.

 

Purpose

Checks for any pending client events or notifications.

Parameters

Parameter

Description

EventCount

Out: Returns the number of pending events or notifications.

 

Returns

TRE_SUCC if successful, the number of pending events or notifications received is returned in EventCount; TRE_FAIL otherwise. The TRE sets treErrno to indicate the type of error.

Remarks

If the client is multi-context, only events or notifications for the client's context are received, unless a process-wide event handler is defined. For each pending event or notification, the last handler defined in treEventSetHandlerx() or treEventSetEpmHandlerx() is called.
