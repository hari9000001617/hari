EPM Built-in Functions

The following table lists the EPM built-in functions, with a brief description, in alphabetical order.

 

EPM Built-in Functions

Function

Description

abort()

Aborts evaluation of the current expression and displays the specified error message.

abs()

Returns the absolute value of a number (that is, the number without its sign).

acos()

Returns the arc cosine of the specified value; for example, the angle (in radians) that has a cosine as the passed value.

add_months()

Returns a date calculated as a specified number of months from the given date-time.

array_maxsize()

Sets the maximum size of an array.

asin()

Returns the arc sine of the specified value.

assert()

Tests the passed value (which is normally a conditional expression).

atan()

Returns the arc tangent of a number.

atan2()

Returns the arc tangent of y#/x# as an angle in radians in the range - pi to pi.

avg()

Returns the average value of all the defined elements in the array or hash.

bcd_add()

Returns a string representation of the sum of two large integer values.

bit_and()

Returns a bit-wise AND of the two parameters.

bit_not()

Returns a bit-wise NOT (or complement) of the parameter.

bit_or()

Returns a bit-wise OR of the two parameters.

bit_reset()

Returns the passed value with the specified bit reset.

bit_set()

Returns the passed value with the specified bit set.

bit_shl()

Returns the passed value shifted left by the specified number of bits.

bit_shr()

Returns the passed value right shifted by the specified number of bits.

bit_test()

Tests if the specified bit is set in the passed value.

bit_xor()

Returns a bit-wise EXCLUSIVE OR of the two parameters.

bsearch()

Performs a binary search on the sorted array for the specified value.

bytes_total()

Returns the total amount of heap memory requested by the current thread since it was started.

bytes_used()

Returns the total number of heap memory allocated for this process using the Convergent Billing memory allocation routines.

call_stack()

Retrieves details of the current call stack.

caller()

Retrieves details of the calling function or expression for the current expression.

ceil()

Returns a real number rounded up to the nearest integer value (for a real value) or a date rounded up to the nearest whole day (for a date value).

char()

Returns the character represented by the specified character set value.

compact()

Compacts the specified hash by scanning its elements and reallocating memory for those elements associated with memory blocks with free, unusable memory.

conceal()

Hides the specified value from all debug and trace output.

cos()

Returns the natural cosine of the passed angle specified in radians.

count()

Returns the number of defined elements in the array or hash.

current_date()

Returns the current date-time.

current_time()

Returns, as a real value, the number of fractional seconds since midnight.

dayofmonth()

Returns the day of the month for the date supplied.

dayofweek()

Returns the day of the week for the date supplied.

dayofyear()

Returns the day of the year for the supplied date.

daysbetween()

Returns the number of days between the two dates.

daysinmonth()

Returns the number of days in the month for the date supplied.

debug()

Outputs a message to the debug trace output if expressions are being evaluated in debug mode.

debug_level()

Returns the debug level value.

debug_off()

Disables function call stack debugging and the debug() function.

debug_on()

Enables function call stack debugging and the debug() function.

decimal_point()

Returns the character being used for the decimal point symbol in the current locale.

delete()

Deletes the specified key value from the hash.

distinct()

Removes duplicate elements from the specified array.

effective_date()

Returns the effective date-time that is used to evaluate the current expression.

error_message()

Returns the current message.

error_number()

Returns the current error number.

eval()

Returns the result of parsing and evaluating the parameter as an EPM expression.

exists()

Tests whether the specified key exists in the given hash parameter.

exp()

Returns the exponential (ex) of the parameter.

expect()

Tests the passed value, which is normally a conditional expression.

expect_cmp()

Similar to expect(), but provides additional information when an expectation failure occurs.

expect_throw()

Confirms that an expression throws an exception as expected.

first()

Returns the key value of the first element in the specified hash.

floor()

Returns a real number rounded down to the nearest integer value (for a real value) or a date rounded down to the nearest whole day (for a date value).

gstats()

Returns statistics for functions for which call graph statistics have been gathered.

gstats_off()

Disables gathering of statistical information for functions.

gstats_on()

Enables gathering of statistical information for functions including their call graph statistics.

hash()

Creates a hash from the input.

hash_maxsize()

Sets the maximum size of a hash.

hires_time()

Used to measure elapsed time within a process.

hourofday()

Returns the hour of day for the date supplied.

is_array()

Tests whether the specified element is a defined or undefined array.

is_blob()

Tests whether the specified value is of type blob.

is_concealed()

Tests whether the specified value is hidden.

is_date()

Tests whether the specified value is of type date.

is_debug_on()

Tests whether function call stack debugging is enabled.

is_defined()

Tests whether the specified variable is defined.

is_double()

Tests whether the specified value is of type real.

is_hash()

Tests whether the specified element is a defined or undefined hash.

is_integer()

Tests whether the specified value is of type integer.

is_long()

Tests whether the specified value is of type integer.

is_real()

Tests whether the specified value is of type real.

is_string()

Tests whether the specified value is of type string.

keys()

Returns an array of sorted key values from the specified hash.

length()

Returns the number of characters in the specified string.

lengthb()

Returns the number of bytes in the specified string.

line()

Returns the current line number of the function or expression.

log()

Returns the natural logarithm of the parameter.

log10()

Returns the logarithm to the base 10 of the parameter.

loop_limit()

Sets the maximum number of times an expression can loop before the parser fails with an 'infinite loop detected' message.

lpad()

Returns the string padded on the left with the specified character.

ltrim()

Returns the string trimmed on the left of all specified characters.

max()

Returns the maximum of two values or the maximum value in an array or hash.

message_box()

Displays a message box with the specified title, containing the specified string and buttons.

message_number()

Returns the message number for specified message name.

min()

Returns the minimum of the two values or the minimum item of the array or hash.

minutes()

Returns the number of minutes in the time portion of the date supplied.

monthofyear()

Returns the month of the year for the date supplied.

monthsbetween()

Returns the number of whole months between the two dates supplied.

next()

Returns the key value of the next key in the specified hash from the last value in the hash that was accessed.

nvl()

Returns the parameter if defined; otherwise, the specified default.

ord()

Returns the character set value of the first character of a string or the numerical value (0-255) of the first byte of a blob.

parse()

Parses the specified string and returns the internal representation of the expression.

parse_date()

Returns the current date-time like current_date(), but is allowed to be evaluated at parse time.

pop()

Removes the last element from an array.

position()

Returns the position of a string within another string, or of a byte pattern within a blob.

positionb()

Returns the byte offset of the first occurrence of a substring within a string.

print()

Prints the parameter.

push()

Adds the value to the end of the array.

reveal()

Removes the HIDDEN flag from data, allowing it to be output by debug and trace outputs.

reverse()

Returns the specified parameter with all characters or bytes in reverse order.

round()

Returns a value rounded to the nearest decimal place as specified by the scale parameter.

rpad()

Returns a string padded on the right with the specified character.

rtrim()

Returns the string trimmed on the right of all specified characters.

seconds()

Returns the number of seconds in the time component of the date supplied.

sin()

Returns the natural sine of an angle in radians.

sizeof()

Returns the number of elements in the specified array or hash.

sleep()

Suspends execution for a specified number of seconds.

sort()

Sorts the elements in the array.

sqrt()

Returns the square root of the number.

stack_limit()

Sets the maximum function call stack depth loop before the parser fails with an 'infinite recursion detected' error message.

stats()

Returns statistics for a particular function or all functions, as specified by the parameter.

stats_date()

Returns the date-time for which the current statistics have been collected.

stats_off()

Disables the gathering of statistical information.

stats_on()

Enables the gathering of statistical information.

stats_reset()

Resets all statistics for all functions.

strict_off()

Disables parsing in strict mode.

strict_on()

Enables parsing in strict mode.

substr()

Returns a specified part of a string or blob, using character semantics for strings and byte semantics for blobs.

substrb()

Returns a specified part of a string using byte semantics.

sum()

Returns the sum of the defined elements in the array or hash.

tan()

Returns the natural tangent of an angle in radians.

to_blob()

Converts a string to a blob, or a single-byte integer into a single-byte blob.

to_date()

Converts the specified value to a date value.

to_double()

Converts the specified value to a real value.

to_integer()

Converts the specified value to an integer.

to_long()

Converts the specified value to an integer.

to_lower()

Converts a string to lower case.

to_real()

Converts the specified value, either an integer or a string, to a real value.

to_string()

Returns a string representation of the specified value.

to_string_blob()

Returns a string representation of the specified blob.

to_time()

Returns the time component of the date supplied.

to_upper()

Converts a string to upper case.

transfer()

Transfers the specified value to the calculation stack and undefines the value in the process.

undefine()

Deletes the definition of the specified variable.

values()

Returns an array of the values from the specified hash.

weekofyear()

Returns the week of the year for the specified date.
