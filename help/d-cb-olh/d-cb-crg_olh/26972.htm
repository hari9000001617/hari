fCMN_TREProdInstFacilityGroupRetrieve_R?[]()

Syntax

fCMN_TREProdInstFacilityGroupRetrieve_R?[](
    l_ProductInstanceId&,
    l_EffectiveDate~)

 

Description

Retrieves the facility groups associated with a specified product instance.

Attributes

Application Environment

Any

Context

Any

 

Parameter

Description

l_ProductInstanceId&

In: Product instance ID of the product instance to which the facility groups belong.

l_EffectiveDate~

In: Effective date of the search.

If not specified, the current date is used.

 

Returns

If successful, an array of hash tables with the details of each facility group based on FAC_GROUP_INSTANCE_TRE_V view. If the product instance does not have any facility groups associated with it, an empty array is returned.

If unsuccessful, an exception is raised.

Remarks

If the product instance does not have any facility group associated with it, an empty array is returned.
