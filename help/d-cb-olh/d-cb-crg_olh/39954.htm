Description

Interface 1 generates adjustment images and stores them in the ADJUSTMENT table.

Interface 2 is a wrapper around interface 1.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

AdjustmentIdList&[]

In: List of adjustment IDs for which images are to be generated.

SuccessAdjustmentIdList&[]

Out: List of adjustment IDs for which images were generated successfully.

ErrorAdjustmentIdList&[]

Out: List of adjustment IDs for which images were not generated successfully.

ErrorMessageIdList&[]

Out: List of error message IDs, one for each adjustment that failed image generation.

ErrorMessageList$[]

Out: List of error messages, one for each adjustment that failed image generation.

AdjustmentID&

In: ID of adjustment for which an image is to be generated.

 

Returns

Interface 1:

1 if successful; 0 if unsuccessful. If the function encounters an error condition from which it cannot recover, an error is raised.

Interface 2:

1 if an image was generated for the adjustment; otherwise, an error is raised.

Remarks

Interface 1

The function is unsuccessful if the number of adjustments in error equals or exceeds the value specified in the IGP.ERROR_THRESHOLD configuration item attribute.

In the case of failure, any images successfully generated are retained.

Each adjustment image is generated according to the order in AdjustmentIDList&[]; duplicate adjustment IDs are ignored.

For image generation to be successful, the invoice format for each adjustment must:

  • Have a template type of Adjustment Image.
  • Have a context of Customer Node.
  • Reference a database view that includes an ADJUSTMENT_ID column (for example, INV_CUSTOMER_ADJUSTMENT_V).

On successful generation of an adjustment image, the adjustment ID is appended to SuccessAdjustmentIdList&[].

If an image cannot be generated for an adjustment, the:

  • Adjustment ID is appended to ErrorAdjustmentIdList&[].
  • Error message ID is appended to ErrorMessageIdList&[].
  • Explanatory message text is appended to the ErrorMessageList$[].

Interface 2

The function passes AdjustmentId& to interface 1 and raises an error if the return indicates an error has occurred.

This function is used if only an image is required for a single adjustment.
