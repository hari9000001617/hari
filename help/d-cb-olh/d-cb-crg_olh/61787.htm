biERBStats?{}()

Syntax

biERBStats?{}(
    BkrSeqNr&)

 

Description

Returns processing statistics for the current ERB process.

Attributes

Application Environment

Any

Context

Any

Method

Remote

Parameters

Parameter

Description

BkrSeqNr&

In: Sequence number of the BKR process to return statistics for.

Returns

Table 14 lists statistics returned by ERBStats?().

 

Table 14 Statistics Returned by ERBStats?()

Key

Type

Description

Created

Date

Date-time when the ERB was created.

Size

Integer

Size (in bytes) of the shared memory segment used by the ERB.

IsDatabaseOn

Integer

Valid values are:

  • 1 (True): Database is available
  • 0 (False): Non-database mode.

DatabaseOnDate

Date

Date-time when the database was last available.

DatabaseOffDate

Date

Date-time when the database was last unavailable.

It is omitted if the database has always been available.

FilesFree

Integer

Number of free file records in the cache.

The maximum number of concurrent files that can be stored in the cache is the sum of FilesFree and the length of Files array.

FilesMax

Integer

Maximum number of files that have been present in the cache at any one time.

FilesTotal

Integer

Total number of opened files.

Transactions

Integer

Number of open transactions that are open in the cache.

TransactionsFree

Integer

Number of free transaction records in the cache.

The maximum number of concurrent transactions that can be stored in the cache is the sum of TransactionsFree and Transactions.

TransactionsMax

Integer

Maximum number of open transactions in the cache at any one time.

TransactionsTotal

Integer

Total number of opened transactions.

EventsPendingRating

Integer

Number of events pending rating.

EventsPendingRatingMax

Integer

Maximum number of events pending rating at any one time.

EventsPendingOutput

Integer

Number of events including error events pending output.

EventsPendingOutputMax

Integer

Maximum number of including error events pending output at any one time.

EventsOutputTotal

Integer

Total number of including error events that have been output.

ChargesPendingOutput

Integer

Number of charges pending output.

ChargesPendingOutputMax

Integer

Maximum number of charges pending output at any one time.

ChargesOutputTotal

Integer

Total number of output charges.

TotalRatedEvents

Integer

Total number of events rated successfully, and output to the database.

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalUnratedEvents

Integer

Total number of events flagged as not to be ,and output to the database.

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalNormErrorEvents

Integer

Total number of events resulting in normalisation errors, and output to the database.

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalRatingErrorEvents

Integer

Total number of events resulting in rating errors, and output to the database. 

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalCorruptEvents

Integer

Total number of events that could not be successfully decoded. 

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalDiscardedEvents

Integer

Total number of events discarded either before or after rating. 

This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

TotalTracerEvents

Integer

Total number of events identified as tracer records. This includes statistics from open files (that is, statistics from the Files array listed in Table 15).

Files

Array

Array of details of each normalised event file being processed.

Each element in this array is a hash containing the details listed in Table 15.

EventPool

Hash

Details of the pool used to store events, error events, and charges.

It contains details listed in Table 16.

CommandPool

Hash

Details of the records and pool used to store pending commands and responses.

It contains the details listed in Table 17.

EnmProcesses

Array

Array of details of each ENM process attached to the BKR.

Each element in this array is a hash containing the details listed in Table 18.

ErtProcesses

Array

Array of details of each ENM process attached to the BKR.

Each element in this array is a hash containing the details listed in Table 19.

EroProcesses

Array

Array of details of each ERO process attached to the BKR.

Each element in this array is a hash containing the details listed in Table 20.

RasProcesses

Array

Array of details of each Event Rate and Store process attached to the BKR.

Each element in this array is a hash containing the details listed in Table 21.

Key

Type

Description

Filename

String

Name of file.

ProcessStartDate

Date

Date-time when processing of the file commenced.

ProcessEndDate

Date

RatedEvents

Integer

Number of events from the file rated successfully, and output to database.

UnratedEvents

Integer

Number of events from the file flagged as not to be rated, and output to database.

NormErrorEvents

Integer

Number of events from the file that resulted in normalisation errors, and output to database.

RatingErrorEvents

Integer

Number of events from the file that resulted in rating errors, and output to database.

CorruptEvents

Integer

Number of events from the file that could not be successfully decoded.

DiscardedEvents

Integer

Number of events from the file that were discarded either before or after rating.

TracerEvents

Integer

Number of events from the file identified as tracer records.

EnmNr

Integer

Number of the ENM process that is normalising events from the file.

EroNr

Integer

Number of the ERO process that is outputting data for the file.
