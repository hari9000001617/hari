INVOICE_OTHER_DETAIL_TRE_V

The INVOICE_OTHER_DETAIL_TRE_V view displays details of invoice charges for the customer node and service event level.

Column Definitions

Column Name

Properties

Description

INVOICE_ID

NUMBER (15)

Not null

ID of a customer node invoice.

CHARGE_ID

NUMBER (15)

Not null

Charge record associated with the normalised event.

CUSTOMER_NODE_ID

NUMBER (15)

Customer node associated with the charge.

SERVICE_ID

NUMBER (15)

Service associated with the normalised event.

ACCOUNT_ID

NUMBER (15)

Not null

Account to which the charge was allocated.

LAST_MODIFIED

DATE

Not null

Date-time that the row was last modified.

CHARGE

NUMBER (22,6)

Not null

Value of the charge in the specified currency.

PROPORTION

NUMBER (22,6)

Proportion of the total charge that this charge corresponds to.

It only contains a value if the charge is associated with a tariff.

CURRENCY_ID

NUMBER (8)

ID of the currency in which the charge was made.

TARIFF_ID

NUMBER (8)

ID of the tariff, discount, or tax that generated the tariff.

TARIFF_NAME

VARCHAR (33)

Name of the tariff, discount, or tax that generated the tariff.

TARIFF_TYPE_CODE

NUMBER (8)

Numeric code that defines the type of tariff.

It is based on the TARIFF_TYPE reference type.

SUBTOTAL_ID

NUMBER (8)

Subtotal associated with the charge.

SUBTOTAL_NAME

VARCHAR (33)

Name of the subtotal associated with the charge.

TARIFF_OR_SUBTOTAL_NAME

VARCHAR (33)

Not null

Name of the tariff or subtotal for each event level charge.

DONT_DISPLAY_IND_CODE

NUMBER (1)

Indicator that the charge should not be displayed.

CONTEXT_CODE

NUMBER (2)

Not null

Processing context associated with the tariff or subtotal. Valid values are:

  • 2 (Service)
  • 3 (Customer Node)
  • 4 (Customer).

It is based on the CONTEXT_SUBTOTAL reference type.

INVOICE_SECTION_NR

NUMBER (8)

Invoice section number associated with the charge.

INVOICE_ITEM_NR

NUMBER (8)

Invoice item number associated with the charge.

INVOICE_TXT

VARCHAR (255)

Optional invoice text that can be displayed in association with the charge.

KEY_VALUE

VARCHAR (255)

Hash key value or array index value associated with the charge.

CHARGE_GENERAL_1 - CHARGE_GENERAL_20

VARCHAR (255)

General fields 1 - 20 for storing calculations and lookups performed during rating.

The information is accessible by the Bill Generation Process (BGP) and the Invoice Generation Process (IGP).
