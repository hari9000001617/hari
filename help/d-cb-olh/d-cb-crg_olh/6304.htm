Derived Attribute Table Variables

The following tables list the entity validation variables for standard and extended derived attrribute tables. Different entity validation variables for derived attribute tables are available depending on the context in which they are used.

For standard derived attributes fields, entity validation variables are associated with the DERIVED_ATTRIBUTE_ARRAY view and are available for use in the Convergent Billing client and Entity Validation Server (EVS) context.

For additional extended derived attribute fields, entity validation variables are not associated with DERIVED_ATTRIBUTE_ARRAY, and are only available for use in the EVS context.
