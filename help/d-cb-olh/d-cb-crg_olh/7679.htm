treMallocWString

C

wchar_t *treMallocWString(const wchar_t *WString)

 

Object Pascal

function treMallocWString(PWString: PWideChar): PWideChar

 

Perl

Not applicable.

 

Purpose

Allocates memory (using treMalloc()) to store the specified wide character string. The string is copied to the allocated memory and a pointer to the location is returned.

Note:

Wide character strings are only supported for the UTF-16 character set.

Parameters

Parameter

Description

Wstring, PWString

In: Wide character string to be copied.

 

Returns

Pointer to the copied string that was successfully allocated; otherwise, null if the memory could not be allocated, or if the size was less than, or equal to, zero.

Remarks

For the C interface, the TRE_MALLOCWSTRING(WString) macro is provided to:

  • Record line and file information associated with the allocated memory
  • Assist in debugging memory allocation and de-allocation problems, or memory leaks.
