Form Automation Parameters

The OLE and EPM Automation Interfaces provide functions for external processes and programs to control Convergent Billing forms. Each form has a specific set of parameters used by the automation interface functions to control various details of the forms, and an entity name used when calling the form by automation.

Many of the form automation parameters refer directly to the fields contained in the database views. For more information about database views, refer to the Configurer's Reference Guide for Singleview Convergent Billing.
