EventRealTimeGenerate&()

Syntax

EventRealTimeGenerate&(
    const InputEntitlementId&,
    const InputTariffId&,
    const InputProductInstanceId&,
    const InputPeriodStartDate~,
    const InputPeriodEndDate~,
    const InputDuration#,
    const FieldNames$[],
    var FieldValues?[])

 

Description

Attempts to generate an event for a real-time tariff.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

InputEntitlementId&

In: ID of the entitlement associated with which the real-time tariff.

InputTariffId&

In: ID of a tariff for which an event is generated. If the tariff is not a real-time tariff, an error is raised.

InputProductInstanceId&

In: ID of the product instance with which the tariff is associated.

InputPeriodStartDate~

In: Start date-time after which events are generated.

InputPeriodEnddate~

In: End date-time before which to generate events.

InputDuration#

In: Duration in seconds of the period from InputPeriodStartDate~ to InputPeriodEndDate~ specified as a fraction of the recurring charge period of the tariff.

FieldNames$[]

In: Array of field name populated from the NORMALISED_EVENT_V view.

FieldValues?[]

Out: Array of field values ordered as in FieldNames$[].

 

Returns

1 if successful; otherwise, 0, and an error is raised.

Remarks

This function:

  • Retrieves product instance information for the specified product instance, and entitlement information for the related entitlement
  • Populates direct variables
  • Evaluates the tariff's eligibility expressions.

If the tariff is eligible, the assignment expressions are evaluated, FieldValues?[] are populated with all fields requested in FieldNames$[], and the function returns 1.
