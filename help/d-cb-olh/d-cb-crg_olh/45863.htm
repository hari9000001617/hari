biBillRunInvoiceConsolidate&()

Syntax

biBillRunInvoiceConsolidate&(
    BillRunId&,
    EffectiveDate~,
    EffectiveDayOfMonth&,
    BillRunOperationId&,
    QAInd&,
    RootCustomerNodeList&[],
    var SuccessCustomerNodeList&[],
    var ErrorCustomerNodeList&[],
    var SuppressedCustomerNodeList&[],
    var OperationStatistics?{})

 

Description

Calls biInvoiceConsolidate&() and populates the BILL_RUN_OPERATION table with statistics from the OperationStatistics?() hash.

This is a billing configuration-compliant remote wrapper function for biInvoiceConsolidate&().

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

BillRunId&

In: ID of the bill run to be processed.

EffectiveDate~

In: Effective date of the bill run to be processed.

EffectiveDayOfMonth&

In: Logical day of the month to associate with EffectiveDate~.

It is usually set to the day of the month of EffectiveDate~. However, if EffectiveDate~ is the last day of the month, EffectiveDayOfMonth& may be a later day.

BillRunOperationId&

In: ID of the bill run operation to be processed.

QAInd&

In: Indicator to identify if the bill run to be processed is a real bill run or a QA bill run.

A non-zero value (TRUE) indicates a QA bill run. A zero value (FALSE) indicates a real bill run.

RootCustomerNodeList&[]

In: List of root customer nodes to be processed.

The list must contain at least one customer node.

SuccessCustomerNodeList&[]

Out: List of all root customer nodes successfully processed.

It is a subset of RootCustomerNodeList&[] and is in the same order as RootCustomerNodeList&[].

ErrorCustomerNodeList&[]

Out: List of root customers not successfully processed.

OperationStatistics?{}

Out: Unknown EPM hash containing the statistics gathered during the bill run operation.

The table below lists the parameter hash keys.

 

Returns

1 if successful; otherwise, an error is raised.

Remarks

The following table lists the OperationStatistics?{} parameter hash keys.

 

OperationStatistics?{} Parameter Hash Keys

Key

Value

ConsolidateInvoicesGenerated

Number of consolidated invoices generated by the bill run operation.

General_Stats1

Number of consolidated invoices generated by the bill run operation.

General_Stat2

Number of statements pending consolidation consolidated by the bill run operation.

General_Stats3

Number of customer nodes processed by the bill run operation.
