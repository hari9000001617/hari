SpinStats?[]()

Syntax

SpinStats?[]()

 

Description

Returns the current spinlock statistics for the process.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

None.

 

 

Returns

Unknown hash.

Remarks

The performance of processes using spinlocks is affected by the configuration of the ATA_SCHED_POLICY, ATA_SCHED_PRIORITY, ATA_SPIN_LIMIT, ATA_SPIN_USLEEP, and ATA_SPIN_BASE environment variables (see the System Reference Guide for Singleview).

Sleeptime, Spintime, and Locktime statistics are only output in the hash if timing statistics gathering is enabled (that is, SpinStatsOn&() was called).

For multi-threaded processes, statistics are collected per thread with the statistics aggregated to report total statistics per process. The collection of statistics per thread means that statistics collection does not significantly impact on thread concurrency within a process.

The following lists statistics returned in the unknown hash.

 

Statistics Returned in Unknown Hash

Key

Type

Description

SpinLock

String

Name of the spinlock.

NrLocks

Integer

Number of lock requests performed.

NrBusyLocks

Integer

Number of lock requests not granted immediately.

It is always less than or equal to NrLocks.

NrSleeps

Integer

Number of times usleep()or select() was called while waiting to acquire busy locks.

NrSpins

Integer

Total number of times that the process has spun waiting to acquire this lock over all lock requests.

MaxSleeps

Integer

Maximum number of times that usleep() or select() was called in any single-lock request.

MaxSpins

Integer

Maximum number of times that the process has spun waiting to acquire any single lock-request.

SleepTime

Real

Number of seconds that the process has spent waiting for the lock to be released.

It does not consume CPU seconds, and has a granularity of microseconds.

SpinTime

Real

Number of seconds that the process has spun or polled waiting for the lock to be released.

It can be considered 'lost' CPU time.

LockTime

Real

Number of seconds the process held the lock, with a granularity of microseconds.
