Description

Interface 1 fetches the values of fields specified by FieldNames$[] for all reference codes with the specified reference type ID.

Interface 2 fetches details of all reference codes with the specified reference type ID.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

ReferenceTypeId&

In: ID of the reference type to fetch.

RefCode&[]

In: Optional. Single reference code to restrict the reference types returned.

Out: Reference codes of all matching rows.

Abbreviation$[]

In: Optional. Single reference abbreviation to restrict the reference types returned.

Out: Abbreviation of all matching rows.

CodeLabel$[]

In: Optional. A single code label to restrict the reference types returned.

Out: Code labels of all matching rows.

Enabled&[]

Out: Enabled indicators of all matching rows.

Description$[]

Out: Description of all matching rows.

FieldNames$[]

In: Names of fields in REFERENCE_CODE_TRE_V view for values to be fetched.

FieldValues?[]

Out: Field values specified in the same order as their corresponding fields in FieldNames$[].

Returns

Number of rows fetched; otherwise, an error is raised.

Remarks

For interface 1, details of the returned rows are returned in the second and subsequent parameters.

For both interfaces, if the reference type is tenanted, details can only be retrieved by a user associated with the same tenant.
