Description

Retrieves a list of product names (both the base product and its companion products) associated with a service.

Note:

Both interfaces access the service cache directly. Interface 1 uses service details fetched into memory from an earlier cache lookup (making it more efficient), and should be used if possible.

Returns

String array of all product names associated with a service. The base product is the first element in the array, followed by the associated companion products.

Note:

The array does not contain duplicates of names; that is, within the array, the value of each element is unique.

An error is raised if the first function interface is called from outside the Service context.

Remarks

Interface 1 retrieves the list of product names associated with the service being processed.

Interface 2 retrieves the list of product names associated with a specified service.

A companion product is associated with a service if one of the following is true:

  • The companion product is active, regardless of the service's status.
  • The companion product and the service have the same status and are not cancelled.
  • The service and companion product have a status of Cancelled over the same effective date-range, which can span multiple history records for the service or companion product instance.
