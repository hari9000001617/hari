GL_GUIDANCE_TRE_V

The GL_GUIDANCE_TRE_V view contains details of general ledger guidance entities.

General ledger guidance entities are assigned to billable charges during rating and billing, and are used to define GL codes assigned to charges.

Column Definitions

Column Name

Properties

Description

GL_GUIDANCE_ID

NUMBER (10)

Not null

ID of the general ledger guidance entity.

EFFECTIVE_START_DATE

DATE

Not null

Date-time that the record becomes effective.

Its granularity is one day.

Time component always corresponds to midnight (that is, 00:00:00 hours).

EFFECTIVE_END_DATE

DATE

Not null

Date-time that the record ceases to be effective.

Its granularity is one day.

Time component always corresponds to one second before midnight (that is, 23:59:59).

GL_GUIDANCE_NAME

VARCHAR (129)

Not null

Name of this GL guidance entity.

If the name is not specified, it defaults to either of the following strings:

  • <BE_DEBIT_GL_CODE_NAME>:
    <BE_CREDIT_GL_CODE_NAME>
  • <EBUB_DEBIT_GL_CODE_NAME>:
    <EBUB_CREDIT_GL_CODE_NAME>

The strings are identified respectively by:

  • BE_DEBIT_GL_CODE_ID and BE_CREDIT_GL_CODE_ID
  • EBUB_DEBIT_GL_CODE_ID and EBUB_CREDIT_GL_CODE_ID.

DESCRIPTION

VARCHAR (4000)

Description of this GL_GUIDANCE entity.

LAST_MODIFIED

DATE

Not null

Date-time the record was last modified.

OPERATOR_ID

NUMBER (8)

Not null

ID of the user who last modified this record.

GROUP_ID

NUMBER (8)

Not null

Security group associated with, and controlling access to, this record.

VERSION_STR

VARCHAR (255)

Version ID string used to tag this entity.

It is provided for use by third party configuration management tools and is not directly used by Singleview.

CATEGORIES

VARCHAR (255)

Comma-separated list of categories associated with this entity.

Refer to the CATEGORY_V view for the list of available categories.

CURRENCY_ID

NUMBER (8)

Not null

Currency used for all amounts associated with this GL guidance entity.

All Billed but Unearned (BUE) and Billed and Earned (BE) general ledger codes associated with this entity must have this currency.

Any Earned But Unbilled general ledger codes specified for statements pending consolidation must also have this currency.

If any of the BUE or BE general ledger codes are for AR accounts, this currency must match the currency of the account being billed.

EBUB_CURRENCY_ID

NUMBER (10)

Currency of all amounts associated with charges generated at rating time.

It defaults to CURRENCY_ID.

If any of the BE GL codes are for AR accounts, this currency must match the currency of the charge that was rated.

EBUB_DEBIT_GL_CODE_ID

NUMBER (10)

GL code to debit for all rating charges associated with this guidance entity.

An AR GL code cannot be specified for this column.

EBUB_CREDIT_GL_CODE_ID

NUMBER (10)

GL code to credit for all rating charges associated with this guidance entity.

It is mandatory, and can only be specified if EBUB_DEBIT_GL_CODE_ID is specified.

An AR GL code cannot be specified for this column.

BE_DEBIT_GL_CODE_ID

NUMBER (10)

GL code to debit for all general billing charges associated with this entity.

If BE_CREDIT_GL_CODE_ID is set to an AR GL code, it cannot be set to an AR GL code.

BE_CREDIT_GL_CODE_ID

NUMBER (10)

GL code to credit for all general billing charges associated with this entity.

It is mandatory, and can only be specified if BE_DEBIT_GL_CODE_ID is specified.

If BE_CREDIT_GL_CODE_ID is set to an AR GL code, it cannot be set to an AR GL code.

BUE_DEBIT_GL_CODE_ID

NUMBER (10)

GL code to debit all advance rental charges associated with this entity.

It is only applicable if BE_DEBIT_GL_COE_ID is specified and EBUB_DEBIT_GL_CODE_ID is not specified.

If BUE_CREDIT_GL_CODE_ID is set to an AR GL code, it cannot be set to an AR GL code.

BUE_CREDIT_GL_CODE_ID

NUMBER (10)

GL code to credit all advance rental charges associated with this entity.

It can only be, and must be, specified if BUE_DEBIT_GL_CODE_ID is specified.

If BUE_DEBIT_GL_CODE_ID is set to an AR GL code, it cannot be set to an AR GL code.

EBUB_CONS_DEBIT_GL_
CODE_ID

NUMBER (10)

GL code to debit for all nominally billed and earned charges associated with this entity while the statement is pending consolidation.

After the statement is consolidated, all these charges are reversed and posted to BE_DEBIT_GL_CODE_ID.

It is only applicable if BE_DEBIT_GL_CODE_ID is specified, and EBUB_DEBIT_GL_CODE_ID is not specified.

An AR GL code cannot be specified for this column.

EBUB_CONS_CREDIT_GL_
CODE_ID

NUMBER (10)

GL code to credit for all nominally billed and earned charges associated with this entity while the statement is pending consolidation.

After the statement is consolidated, all these charges are reversed and posted to BE_CREDIT_GL_CODE_ID.

It is only applicable if BE_CREDIT_GL_CODE_ID is specified, and EBUB_DEBIT_GL_CODE_ID is not specified.

An AR GL code cannot be specified for this column.
