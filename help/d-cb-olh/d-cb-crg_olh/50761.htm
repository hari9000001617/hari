BILL_RUN_TYPE_TRE_V

The BILL_RUN_TYPE_TRE_V view provides information on the types of bill runs configured in the system.

Column Definitions

Column Name

Properties

Description

BILL_RUN_TYPE_ID

NUMBER (8)

Not null

ID of the bill run type.

LAST_MODIFIED

DATE

Not null

Date-time that the row was last modified.

BILL_RUN_TYPE_NAME

VARCHAR (30)

Not null

Name of the type of the bill run, as known to users.

BILL_RUN_TYPE_DISPLAY_NAME

VARCHAR (60)

Name of the bill run type displayed.

It defaults to BILL_RUN_TYPE_NAME.

Caution:

The value can be translated into a locale-specific language and therefore differ between implementations.

DESCRIPTION

VARCHAR (2000)

Not null

Description of the type of the bill run.

OPERATOR_ID

NUMBER (8)

Not null

User who last modified the bill run.

GROUP_ID

NUMBER (8)

Not null

Group associated with the record.

QA_IND_CODE

NUMBER (1)

If set to 1 (TRUE), the bill run being performed is for quality assurance purposes or to get an indication of the current amount owing for a given customer or set of customers.

DEFAULT_ICON_NAME

VARCHAR (30)

Not null

Default icon used to represent bill runs of a specific type.

DEFAULT_ICON_ID

NUMBER (8)

Not null

Default icon used to represent bill runs of a specific type.

CURRENCY_SYMBOL

VARCHAR (5)

Not null

Currency used for all billed and revoked amounts.

CURRENCY_ID

NUMBER (8)

Not null

Currency used for all billed and revoked amounts.

SUMMARY_FUNCTION_DEFN_NAME

VARCHAR (67)

Summary of the bill run after processing is completed.

SUMMARY_FUNCTION_DEFN_ID

NUMBER (10)

Summary of the bill run after processing is completed.

BILL_RUN_ENTITY_VALIDATION

VARCHAR (30)

Entity validation used to configure the display of general statistics for all bill runs of this type.

BILL_RUN_ENTITY_VALIDATION_
ID

NUMBER (8)

Entity validation used to configure the display of general statistics for all bill runs of this type.

OPERATION_ENTITY_VALIDATION

VARCHAR (30)

Entity validation used to configure the display of general statistics for all bill run operations of this type.

OPERATION_ENTITY_VALIDATION_
ID

NUMBER (8)

Entity validation used to configure the display of general statistics for all bill run operations of this type.

VERSION_STR

VARCHAR (255)

Version identification string used to tag this entity.

It is provided for use by third party configuration management tools and is not directly used by Singleview.

CATEGORIES

VARCHAR (255)

Comma-separated list of categories associated with this entity.

Refer to the CATEGORY_V view for the list of available categories.
