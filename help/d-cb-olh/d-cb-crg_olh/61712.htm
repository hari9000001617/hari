Scheduled Function Search Form

The following table lists automation parameters for the Scheduled Function Search form, data type, and supported functions.

The description column is provided to supplement the description in the SCHEDULED_FUNCTION_TRE_V view, or to provide information about parameters not contained within the database view.

Parameter Definitions

Parameter or Field Name

Data Type

Supported Functions

Description

FROM_START_DATE

Date

Not applicable

Limits the search to scheduled functions with a start date-time on or after the specified date.

TO_START_DATE

Date

Not applicable

Limits the search to schedules with a start date on or before the specified date.

INSTANCE_NAME

String

Not applicable

Name of the instance where the scheduled function was created.

PARTITION_NR

Integer

Not applicable

Number of the partition associated with the scheduled function.

STATUS_CODE

Integer

Not applicable

Current execution status of the scheduled function.

CATEGORY_CODE

Integer

Not applicable

Category associated with the scheduled function.

DISPLAY_SUB_CATEGORY

Integer

Not applicable

If set to:

  • 0, the sub-category is not displayed on the Scheduled Function List form.
  • 1, the sub-category is displayed.

SUB_CATEGORY_CODE

Integer

Not applicable

Sub-category associated with the scheduled function.

If set, DISPLAY_SUB_CATEGORY is set to 1.

DISPLAY_FUNCTION_NAME

Integer

Not applicable

If set to:

  • 0, the function name is not displayed on the Scheduled Function List form.
  • 1, the function name is displayed.

FUNCTION_NAME

String

Not applicable

Name of the function that has been scheduled.

If set, DISPLAY_FUNCTION_NAME is set to 1.

DISPLAY_DESTINATION

Integer

Not applicable

If set to:

  • 0, the destination is not displayed on the Scheduled Function List form.
  • 1, the destination is displayed.

DESTINATION

String

Not applicable

Destination associated with the scheduled function.

If set, DISPLAY_DESTINATION is set to 1.

LAST_ERROR_MESSAGE_ID

Integer

Not applicable

ID of the last error message raised by a failure of the scheduled function.

LAST_ERROR_MESSAGE

String

Not applicable

Text of the last error message raised by a failure of the scheduled function.

SUMMARY_DATE_RANGE_INTERVAL

Integer

Not applicable

Groups the search results into time-ranged intervals on the Summary tab of the Scheduled Function List form, in minutes.

It must be specified as a multiple of 15, and has a maximum of 1440 (1 day).

REFRESH_INTERVAL

Integer

Not applicable

Interval between automatic refreshes of the Summary tab of the Scheduled Function List form, in seconds.

It has an effect only if the Automatic Refresh? check box is selected.

It must be specified as a multiple of 15, and has a maximum of 3600 (1 hour).

DISPLAY_WORKFLOW

Integer

Not applicable

If set to:

  • 0, the workflow details are not displayed on the Scheduled Function List form.
  • 1, the workflow details are displayed.

WORKFLOW_FUNCTION_NAME

String

Not applicable

Name of the function that performs the work item task.

If set, DISPLAY_WORKFLOW is set to 1.

PROCESS_NAME

String

Not applicable

Name of the business process that initiates the work item.

If set, DISPLAY_WORKFLOW is set to 1.

WORK_ITEM_NAME

String

Not applicable

Name of the work item that is executed.

If set, DISPLAY_WORKFLOW is set to 1.

CLEARCLICK

Integer

Not applicable

Equivalent to clicking the Clear button.

The value is irrelevant.

SEARCHCLICK

Integer

Not applicable

Equivalent to clicking the Search button.

The value is irrelevant.

It must be placed last.
