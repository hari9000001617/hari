Return Codes

The automation methods return an integer value indicating the status.

The following table displays the status code values and their descriptions.

 

Automation Interface Status Codes

Value

Meaning

Description

0

OK

Automation call was successful.

1

General Error

Unknown general failure.

2

No Form

Specified entity name does not correspond to a form.

3

No View

The user has inadequate security to view the specified form or record.

4

No Update

The user has inadequate security to update the specified form or record.

5

No Insert

The user has inadequate security to insert the specified form or record.

6

Validation

An invalid field was specified.

20

Not Logged In

No user is connected to the database. No details can be retrieved.

21

Variant Wrong Type

The variant supplied is the wrong data type.

The parameter list and the corresponding values must be a variant containing null or a variant containing an array.

22

Array Not 1D

The parameter or value array is not one-dimensional.

23

Array Size Differ

The parameter array is a different size to the value array.

 

Note:

A call to the LastError method returns the description of the error message.
