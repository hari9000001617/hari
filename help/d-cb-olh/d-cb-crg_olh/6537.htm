Task Search Form

The following table lists automation parameters for the Task Search form, data type, and supported functions.

Note:

The automation parameters for this form do not correspond to any specific TRE view.

Parameter Definitions

Parameter or Field Name

Data Type

Supported Functions

Description

TASK_ID

Integer

Not applicable

Limits the search to the specified Task ID.

Exact matching is performed on this field.

SCHEDULE_NAME

String

Not applicable

Limits the search to the specified schedule.

Case-sensitive, wildcard matching is performed on this field.

FROM_START_DATE

Date

Not applicable

Limits the search to tasks with a start date on or after the specified date.

TO_START_DATE

Date

Not applicable

Limits the search to tasks with a start date on or before the specified date.

FROM_EFFECTIVE_DATE

Date

Not applicable

Limits the search to tasks with an effective date on or after the specified date.

TO_EFFECTIVE_DATE

Date

Not applicable

Limits the search to tasks with an effective date on or before the specified date.

TASK_STATUS_CODE

Integer

Not applicable

Limits the search to the specified status.

SCHEDULE_TYPE_ID

Integer

Not applicable

Limits the search to the specified schedule type.

TASK_TYPE_CODE

Integer

Not applicable

Limits the search to tasks associated with schedules that have the specified task type.

It is based on the TASK_TYPE reference type.

BILL_RUN_ID

Integer

Not applicable

Limits the search to the specified ID.

It is ignored when the Bill Run field is disabled.

The Bill Run field is disabled when TASK_TYPE_CODE is set to a value other than:

  • 5: BILL_RUN_OPERATION
  • 6: BILL_RUN_CUSTOMER_OPERATION
  • 10: BILL_RUN_OPERATION_PREREQUISITE.

INSTANCE_NAME

String

Not applicable

Limits the search to the specified instance name.

ALWAYSDISPLAYLIST

Integer

Not applicable

A value of 0 allows display in either a detail form or a list form.

A value of 1 indicates that the results should always be displayed in a list form.

REUSEDETAILFORM

Integer

Not applicable

A value of 0 indicates the usage of a new form.

A value of 1 indicates that the detail form opened by this search, should be reused.

CLEARCLICK

Integer

Not applicable

Equivalent to pressing the Clear button.

The value is irrelevant.

SEARCHCLICK

Integer

Not applicable

Equivalent to pressing the Search button.

The value is irrelevant.

It must be placed last.
