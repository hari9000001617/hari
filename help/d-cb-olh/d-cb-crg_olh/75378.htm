RSCStats?{}()

Syntax

RSCStats?{}()

 

Description

Provides statistics for the rating subtotal cache (RSC) of the current instance.

Attributes

Function Type

Local Built-in

Service

(Local)

Application Environment

Any

Context

Any

 

Parameter

Description

None.

 

 

Returns

Unknown hash of RSC statistics; an exception is raised if an error is encountered.

Remarks

The following table lists RSC statistics.

RSC statistics also include statistics for the general memory pool, used for storing subtotal aggregates, deltas, and reservations. Statistics for this memory pool have each key prefixed with Pool (for example, PoolTotalAllocatedBytes).

For backwards compatibility, the following values are included twice, with two different keys:

  • PoolTotalSize is the same as PoolTotalBytes
  • PoolAvailSize is the same as PoolFreeBytes
  • PoolUsedSize is the same as PoolAllocatedBytes
  • PoolMaxUsedSize is the same as PoolMaxAllocatedBytes
  • PoolAllocatedBlockCnt is the same as PoolNrAllocatedBlocks
  • PoolFreeBlockCnt is the same as PoolNrFreeBlocks.
