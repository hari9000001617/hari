ScheduledFunctionMonitor

The ScheduledFunctionMonitor interface type is used for generating configuration settings for the Scheduled Function Monitor (SFM) process. An SFM configuration item is automatically created or updated when an interface definition based on this interface type is saved.

Note:

CSG International recommends that:

  • Only one SFM be configured per instance running with an appropriate number of threads to handle the load.
  • If more than one SFM is configured, each SFM must process a distinct set of categories (see the System Configuration Guide for Singleview Convergent Billing).

Wait Period

[Integer]

Period (in seconds) the BMP should pause after starting the SFM.

Queue Maximum

[Integer]

Maximum number of scheduled functions in the queue before a warning is logged.

The warning is logged no more than once every Warn Repeat Period.

If set to null or 0, the number of scheduled functions that can be queued is unlimited.

For more information about this value, refer to Setting Queue_Maximum.

Maximum Delay Time

[Integer]

Maximum delay (in seconds) between when a scheduled function is scheduled to start and when it is started, after which a warning is logged.

The warning is logged no more than once every Warn Repeat Period.

If set to null or 0, the maximum delay is unlimited.

Warn Repeat Period

[Integer]

Minimum frequency (in seconds) for logging warnings caused by exceeding either Queue Maximum or Maximum Delay Time.

If set to null or 0, warnings are not suppressed.

Default Maximum
Thread Count

[Integer]

Default maximum number of concurrent scheduled function evaluation threads.

Time Band

[List]

Time band to which the Maximum Thread Count for the row applies.

It is based on the ResourceTimeBand derived attribute table.

Category

[List]

Category of scheduled functions allowed to use the row's maximum thread count.

It is based on the SCHEDULED_FUNC_CATEGORY reference type.

If set to ALL, Default Maximum Thread Count is overridden by this row's Maximum Thread Count.

Maximum Thread Count

[Integer]

Maximum number of concurrent scheduled function evaluation threads for the specified Time Band and Category.

If Category is:

  • Set to ALL, this value defines the maximum total thread count for the time band, shared by all categories. Otherwise, Default Maximum Thread Count is used.
  • Not set to ALL and this value is greater than the maximum total thread count for the time band, the maximum total thread count is used as the maximum for the specified category.

If set to 0, evaluation for the specified category is disabled.

Failover Delay

[Integer]

Period (in seconds) the SFM delays failing orphaned, 'Running' scheduled functions upon initialisation.
