ScheduleReplicationGroup

The ScheduleReplicationGroup derived attribute table defines the set of Singleview instances associated with each schedule replication group.

Schedule replication groups are used to replicate and maintain schedules across multiple instances in a Singleview cluster.

The table has an evaluation type of Table Lookup By Multiple (1-10) Keys.

Column Definitions

Column Name

Properties

Description

Replication Group (Index 1)

Number (8)

Not null

Schedule replication group to which instances are specified.

It is based on the SCHEDULE_REPLICATION_GROUP reference type.

Instance (Index 2)

String (9)

Not null

Singleview instance associated with the replication group.

The instance can be configured to be a member of multiple replication groups.

The ATA_INSTANCE reference type defines the list of instances in the Singleview cluster.

Enabled? (Result 1)

Integer

If set to:

  • 1 (TRUE), the instance is enabled for use in the replication group.
  • 0 (FALSE), the instance is ignored.
