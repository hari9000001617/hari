gstats()

Syntax

gstats()

 

Description

Returns statistics for functions for which call graph statistics have been gathered.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

None.

 

 

Returns

Array of hashes containing call graph statistics for each function for which call graph statistics have been gathered, or an empty array if no call graph statistics are available.

Remarks

This function is not usually called directly, but called using the Getstats tool (see the Performance Guide for Singleview).

This function generates call graph statistics. To generate only standard statistical information, use stats().

The following table describes the statistical elements of each hash.

Note:

The combination of Function, CallerId, and LineNr is unique for each hash in the array. Id is also unique for each hash in the array.

 

Statistics Hash Elements

Key

Type

Description

CallerId

Integer

ID of the call stack that evaluated the function, or 0 if there is no call stack.

Calls

Integer

Number of times this function has been called from the call stack in CallerId and the line number in LineNr in this hash.

ElapsedMax

Real

Maximum execution duration for the function in this hash.

It is undefined if Calls in this hash is 0.

ElapsedMin

Real

Minimum execution duration for the function in this hash.

It is undefined if Calls in this hash is 0.

ElapsedTime

Real

Duration (in seconds) of execution time for the function in this hash.

Note:

The precision of ElapsedTime is platform-dependent.

ElapsedTime2

Real

Sum of the elapsed time squared of each call to the function in this hash from the call stack specified in CallerId and line number specified in LineNr.

It can be used to calculate the standard deviation of the elapsed time of the function in this hash.

Function

String

Name of the function for the generated statistics, including the interface number (if not 1).

Processes and modules can register their own entries to also be included in statistics generated by gstats(). They will have a function name in the format of <type>: <name> (for example, SQL: Open).

The special function of ** All Expressions ** is used to gather statistics for EPM expressions that are not evaluated directly by a function,or as part of a call stack entry.

Id

Integer

ID of a call stack that contains the combination of Function, CalledId, and LineNr contained in this hash.

LineNr

Integer

Line number in the EPM function that called this function.

NetTime

Real

Total amount of time spent evaluating this function (in seconds) when called from the call stack specified in CallerId and line number specified in LineNr, excluding the time spent calling other functions, processes, or modules from this function.

It provides the amount of time taken to execute the code in this function.
