EntitlementCategory

The EntitlementCategory derived attribute table defines the categories that apply to entitlements.

Column Definitions

Column Name

Properties

Description

Category (Index 1)

Integer

Not null

ID of an entitlement category.

Type (Result 1)

Integer

Not null

Name of an entitlement type to which the category is associated.

Unit of Measure (Result 2)

Integer

Not null

Units in which allocations to entitlements of the category are measured.

It is based on the ENTITLEMENT_UNIT_OF_MEASURE reference type.

Class (Result 3)

Integer

Not null

Classification of entitlements for the category. Valid values are:

  • 1 (Recurring)
  • 2 (One-off)
  • 3 (Dynamic).

It is based on the ENTITLEMENT_CLASS reference type.

Recurring Period (Result 4)

Integer

Period of time between each repetition of an entitlement for the category.

This column is mandatory and only available when the entitlement has a class of Recurring. The value must be greater than zero.

Recurring Period Units (Result 5)

Integer

Units for the duration specified by Recurring Period. Valid values are:

  • 0 (Bill Cycles)
  • 1 (Months)
  • 2 (Days)
  • 3 (Hours).

It is based on the ENTITLEMENT_PERIOD_UNIT reference type.

If entitlements are aligned with bill cycles, the value must be 0 (Bill Cycles).

Validity Period (Result 6)

Integer

Period of time for which allocations to the entitlement remain valid.

The value must be greater than zero.

Validity Period Units (Result 7)

Integer

Units for the duration specified by Validity Period. Valid values are:

  • 0 (Bill Cycles)
  • 1 (Months)
  • 2 (Days)
  • 3 (Hours).

It is based on the ENTITLEMENT_PERIOD_UNIT reference type.

Warning Period (Result 8)

Integer

Duration of the advance period before the entitlement expires.

At the start of the advance period, a warning function (defined in the entitlement type) is evaluated, if specified.

Warning Period Units (Result 9)

Integer

Units for the duration specified by Warning Period. Valid values are:

  • 2 (Days)
  • 3 (Hours)
  • 4 (Minutes).

If Warning Period is specified, a value must be supplied for this column.

It is based on the ENTITLEMENT_WARNING_PERIOD_UNIT reference type.

Expire on Suspension? (Result 10)

Integer

Not null

If set to 1 (TRUE), entitlement allocations expire when an entitlement associate with this category is suspended.

If set to 0 (FALSE), entitlement allocations are unchanged.

Consumption Priority (Result 11)

Integer

Not null

Order in which allocations associated with this category are consumed, relative to entitlements of other categories.

The value must be greater than zero.

A lower value is considered as a higher priority.

Delay Period (Result 12)

Integer

Duration to delay an update to the status of a Units for the duration specified by Warning Period. Valid values are:

  • 2 (Days)
  • 3 (Hours).
  • 4 (Minutes)

If Warning Period is specified, a value must be supplied for this column.

It is based on the ENTITLEMENT_WARNING_PERIOD_UNIT reference type.companion product.

The update occurs when a recurring real time tariff is evaluated, and fails to reserve an amount from an allocation to an entitlement of this category.

The value must be greater than 0.

Delay Period Units (Result 13)

Integer

Units for the duration specified by Delay Period. Valid values are:

  • 2 (Days)
  • 3 (Hours).

If Delay Period is specified, a value must be supplied for this column.

It is based on the ENTITLEMENT_DELAY_PERIOD_UNIT reference type.

Warning Allocation Period (Result 14)

Integer

Duration of the advance period before the entitlement allocation expires.

At the start of the advance period, a warning function (defaulted from the entitlement type) is evaluated, if specified.

Warning Allocation Period Units (Result 15)

Integer

Units for the duration specified by Warning Allocation Period. Valid values are:

  • 2 (Days)
  • 3 (Hours)
  • 4 (Minutes).

If Warning Allocation Period is specified, a value must be supplied for this column.

It is based on the ENTITLEMENT_WARNING_PERIOD_UNIT reference type.
