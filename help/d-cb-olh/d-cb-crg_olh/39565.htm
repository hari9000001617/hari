reveal()

Syntax

reveal(
    <value>)

 

Description

Reveals the contents of value, previously hidden by conceal().

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

value<data_type>

In: Value, with its <data_type>, whose contents need to be revealed.

 

Returns

value with its contents no longer hidden.

Remarks

If value is not hidden, reveal() has no effect.

When using a value returned by biAttributeValueDecrypt$(), or after using the reveal() function on a value, ensure that the value remains hidden in outputs.
