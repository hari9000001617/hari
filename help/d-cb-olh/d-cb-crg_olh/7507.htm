SCHEDULE_PREREQUISITE_TRE_V

The SCHEDULE_PREREQUISITE_TRE_V view provides access to the prerequisite conditions for the schedule.

In the column descriptions of the view the following definitions apply:

  • 'This schedule' refers to the schedule identified by SCHEDULE_ID.
  • 'Prerequisite schedule' refers to the schedule identified by PREREQUISITE_SCHEDULE_ID.

Column Definitions

Column Name

Properties

Description

SCHEDULE_ID

NUMBER (15)

Not null

ID of a schedule for which prerequisite conditions are specified.

SEQNR

NUMBER (4)

Not null

Sequence number that defines the display order of prerequisite conditions.

LAST_MODIFIED

DATE

Not null

Date-time that this record was last modified.

PREREQUISITE_SCHEDULE_
ID

NUMBER (15)

Not null

ID of a schedule whose tasks form a prerequisite condition for running the tasks associated with this schedule.

Note:

When inserting or updating prerequisite schedules, specify either PREREQUISITE_SCHEDULE_ID or PREREQUISITE_SCHEDULE_NAME, but not both.

PREREQUISITE_SCHEDULE_
NAME

VARCHAR (60)

Not null

Name of a schedule whose tasks form a prerequisite condition for running the corresponding tasks associated with this schedule.

Note:

When inserting or updating prerequisite schedules, specify either PREREQUISITE_SCHEDULE_ID or PREREQUISITE_SCHEDULE_NAME, but not both.

PREREQUISITE_TASK_ID

NUMBER (15)

If SCHEDULE_ID corresponds to an immediate schedule, it must be populated; otherwise, it must be NULL.

It is the ID of the task belonging to the prerequisite schedule that forms a prerequisite condition for running the task associated with this schedule. 

PREREQUISITE_CONDITION_
CODE

NUMBER (2)

Not null

Condition on each task associated with the prerequisite schedule that must be satisfied before the task associated with this schedule can run.

Valid values are:

  • 1 (Completed successfully)
  • 2 (Completed with error)
  • 3 (Completed with any status)
  • 4 (Running or completed with any status).

It is based on the PREREQUISITE_CONDITION reference type.

DESCRIPTION

VARCHAR (255)

Optional description of the PREREQUISITE_CONDITION_CODE.

PREREQUISITE_INSTANCE_
NAME

VARCHAR (9)

Read-only

Instance on which all tasks associated with the prerequisite schedule run.

It equals INSTANCE_NAME in SCHEDULE_TRE_V for the prerequisite schedule record.

PREREQUISITE_FIRST_DATE

DATE

Not null

Read-only

Date-time before which all tasks associated with the prerequisite schedule will not run. It is static throughout the life of the prerequisite schedule.

It equals FIRST_DATE in SCHEDULE_TRE_V for the prerequisite schedule record.

PREREQUISITE_EFFECTIVE_
DATE

DATE

Not null

Read-only

Effective date-time for details referenced by the first task run in the prerequisite schedule.

It equals EFFECTIVE_DATE in SCHEDULE_TRE_V for the prerequisite schedule record.

PREREQUISITE_STATUS_
CODE

NUMBER (8)

Not null

Read-only

Current status of the prerequisite schedule. Valid values are:

  • 1 (Active)
  • 2 (Complete).

It equals STATUS_CODE in SCHEDULE_TRE_V for the prerequisite schedule record.

It is based on the SCHEDULE_STATUS reference type.

PREREQUISITE_BASELINE_
DURATION

NUMBER (10)

Not null

Read-only

Expected duration of tasks of this prerequisite schedule in seconds.

It equals BASELINE_DURATION in SCHEDULE_TRE_V for the prerequisite schedule record.
