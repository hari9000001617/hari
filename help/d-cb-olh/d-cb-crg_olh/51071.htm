Variables Based on RGP_PRODUCT_INSTANCE_V

All direct variables based on the RGP_PRODUCT_INSTANCE_V view have an application environment of Data Entry and a context of Any.

Variable Definitions

Variable

Type

Description

ProdActiveDate~

Date

Date-time when the status of this product instance first changed to Active.

ProdBaseProductInstanceId&

Integer

ID of the base product instance for the companion product instance.

ProdCustomerNodeId&

Integer

Customer node that owns the instance of the product.

ProdEndDate~

Date

End date-time for the effective date of the record.

ProdFromProductId&

Integer

Product ID associated with the ProdFromProductInstanceId&

ProdFromProductInstanceId&

Integer

ID of the product instance that is replaced.

ProdGeneral1$ - ProdGeneral10$

String (255)

General fields 1 - 10.

ProdId&

Integer

ID of the product associated with the product instance.

ProdGeneral1$ - ProdGeneral10$

String (255)

General fields 1 - 10.

ProdLastModified~

Date

Date-time when the row was last modified.

ProdPreviousProductOfferId&

Integer

ID of the previous product offer associated with this product instance.

ProdPreviousProductOfferNodeId&

Integer

ID of the product offer hierarchy node from which the product instance was previously allocated.

ProdProductId&

Integer

ID of the product associated with the product instance.

ProdProductName$

String (30)

Name of the product associated with the product instance.

ProdProductDisplayName$

String (120)

Name of the product displayed.

ProdProductOfferAllocationDate~

Date

Date-time at which this product instance was allocated from the product offer specified in ProdProductOfferId&.

ProdProductOfferId&

Integer

ID of the product offer associated with this product instance.

ProdProductOfferNodeId&

Integer

ID of the product offer hierarchy node from which the product instance was allocated.

ProdStartDate~

Date

Start date-time for the effective date of the record.

ProdStatus&

Integer

Status code for the product.

ProdToProductId&

Integer

Product ID associated with the ProdToProductInstanceId&.

ProdToProductInstanceId&

Integer

ID of the replacement product instance.
