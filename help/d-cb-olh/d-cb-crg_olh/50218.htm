Interface 2

 
SFMTrace&(
    DebugLevel&)
 
 
 
SFMTrace&(
    DebugLevel$)

 

Description

Sets the trace level for the Scheduled Function Monitor (SFM).

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

DebugLevel&

In: Diagnostic debug level.

If DebugLevel& is less than or equal to zero or DebugLevel$ is an empty string, diagnostic tracing for the SFM process is deactivated.

Debug levels can be added together.

For example, if both multithreading tracing (THR, 8) and EPM tracing (EPM, 64) are required, either SFMTrace&(72) or SFMTrace&('THR,EPM') can be specified.

DebugLevel$

 

Returns

True.
