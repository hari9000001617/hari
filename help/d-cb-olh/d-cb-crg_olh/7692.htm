treStatsOff

C

void treStatsOff(void)

 

Object Pascal

procedure treStatsOff

 

Perl

treStatsOff()

 

Purpose

Disables the collection of TRE function call statistics for the current process. Statistics are not reset by this function.

Parameters

None.

Returns

None.

Remarks

If only specific functions require statistics, the treStatsOn() and treStatsOff() calls can be used to start and stop statistics collection for individual functions.

To activate and deactivate statistics collection for the function, call treStatsOn() before the function, then call treStatsOff() at a later stage in the expression. Any processing done by the function during the collection period is reported when the statistics are retrieved using GetStats.
