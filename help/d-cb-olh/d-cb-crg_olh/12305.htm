ClientShellExecute&()

Syntax

ClientShellExecute&(
    Filename$,
    CommandLineParams$,
    WorkingDir$)

 

Description

Invokes the workstation's operating system to execute the specified file, with the specified command-line parameters, and the specified working directory.

Attributes

Application Environment

Data Entry

Context

Any

Method

Local

 

Parameter

Description

Filename$

In: Application to open or the file to execute.

Note:

A complete path and file name must be specified and environment variables cannot be used (for example, %TEMP%).

CommandLineParams$

In: Command-line parameters to pass if Filename$ is an application.

WorkingDir$

In: Working directory.

 

Returns

Result of the call to ShellExecute().

Remarks

The Windows API function ShellExecute() is called directly.

If Filename$ specifies an application, that application starts with the given command-line parameters. If Filename$ specifies a document, the application that is configured to work with files of that type (the 'default application') starts and the specified file is loaded.

For example, to launch a web browser to a specific page, specify:

ClientShellExecute&('http://www.CSG Internationalbilling.com', '', '')

To load a PDF document in the appropriate viewer, specify:

ClientShellExecute&('c:\\Program Files\\Adobe\\Acrobat 4.0\\Reader\\ReadMe.pdf', '', '')

If an application other than the default application is required to launch a specific file, Filename should specify the path and file name of the executable; CommandLineParams$ is the name of the file to be launched. For example, to load a file into notepad, specify:

ClientShellExecute&('notepad', 'c:\\myfile.txt', '')

If Filename starts with 'sv://', the specified file is downloaded from the server using secure file transfer. The downloaded file is passed to the ShellExecute() Windows API function and placed in the location specified by WorkingDir$.

ClientShellExecute&('sv://SecureDownload/SecureFileTransfer_LargeFile1.pdf', '', 'c:\\temp\\DownloadedFile_1.pdf')
