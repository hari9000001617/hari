biEntitlementEntityPurgeInstance&()

Syntax

biEntitlementEntityPurgeInstance&(
    const Context&[], 
    const EntityId&[])

 

Description

Purges entities from entitlement caches.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

Context&[]

In: Array of contexts for the entities being purged. Valid values are:

  • 2 (Service)
  • 3 (Customer Node)
  • 4 (Customer).

EntityId&[]

In: Array of IDs of the entities being purged.

Each entity in the array must have a corresponding entry in the Context&[] array.

Note:

For entities with a context of:

  • Service, it corresponds to the service ID.
  • Customer Node or Customer, it corresponds to the customer node ID.

 

Returns

1 if successful; otherwise, 0.

Remarks

This is a synchronous remote function that runs in biCache service.

It calls EntitlementEntityPurge&() for each entity to be purged.
