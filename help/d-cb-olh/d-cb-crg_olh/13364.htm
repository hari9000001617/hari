GLGuidanceFetch?[](
    const GLGuidanceName$,
    const EffectiveDate~,
    const FieldNames$[])

 

Description

Retrieves GL guidance details for the specified GL guidance entity as of the specified date.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

GlGuidanceId&

In: GL guidance entity ID.

GlGuidanceName$

In: GL guidance entity name.

EffectiveDate~

In: Effective date for the details required.

FieldNames$[]

In: List of field names to retrieve.

For a list of valid field names, refer to the table below.

Note:

Not all fields contain a value for all GL guidance entities in the database. If a valid field name is requested in FieldNames$[] but does not have a value defined for the GL guidance entity, an undefined value is returned.

 

Returns

Array of values corresponding to the fields specified in FieldNames$[]. The order of the fields is the same as specified in FieldNames$[]. If an invalid field name is requested, an error is raised.

Remarks

The GL guidance entity can be specified by ID or by name. If the GL guidance entity does not exist in the cache, it is loaded into the cache.

The following table lists the valid field names for the FieldNames$[] parameter. Most field names correspond to fields from the GL_GUIDANCE_HISTORY table.
