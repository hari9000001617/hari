Configurer's Reference Guide for Singleview Convergent Billing 9.1

Task Detail Form

The following table lists automation parameters for the Task Detail form, data type, and supported functions.

The description column is provided to supplement the description in the TASK_TRE_V view, or to provide information about parameters not contained within the database view.

Parameter Definitions

Parameter or Field Name

Data Type

Supported Functions

Description

BASELINE_DURATION

Integer

Update

Expected duration of the task in seconds.

BILLRUNCLICK

Integer

View

Activates the Bill Run button if it is enabled.

The value is irrelevant.

EFFECTIVE_DATE

Date

Update

Effective date-time used for the task.

EMAIL_ADDRESS

String

Update

Email address that receives the results of the task.

It defaults to the value specified in the schedule definition.

IMMEDIATE_START_IND_CODE

Integer

Update

Specifies that the task start immediately when all prerequisite conditions are satisfied.

A task with no prerequisites starts when the previous task in the schedule is completed.

INITIAL_HELD_IND_CODE

Integer

Update

Specifies that the task enter 'Held' status when ready to run.

INSTANCE_NAME

String

Update

Convergent Billing instance the task was run on.

MAX_DURATION

Integer

Update

Maximum duration of the task in seconds.

PREREQUISITE_FAILURE_IND_
CODE

Integer

Update

Specifies that the failure of any prerequisite task updates the status of the task to Failure.

PURGE_PERIOD

Integer

Update

Maximum number of days temporary data is to be retained after the task has completed.

It defaults to the value specified in the schedule definition.

SERVER_COMMAND_ARGS

String

Update

Additional server command-line parameters used when invoking the tasks of this schedule.

SCHEDULED_START_DATE

Date

View

Date-time a task commences, if the task is pending.

If the task is running, or is completed, specifies date-time the task started.

RESUBMITCLICK

Integer

View

Activates the Re-submit button if it is enabled.

The value is irrelevant.

RESUBMIT

Integer

View

Provided for backward compatibility.

Use RESUBMITCLICK instead.

REPORTCLICK

Integer

View

Activates the Reports button if it is enabled.

The value is irrelevant.

REPORT

Integer

View

Provided only for backward compatibility.

Use REPORTCLICK instead.

GENERAL_1 - GENERAL_25

String

Update

General fields 1 - 25 used for setting parameters for scheduled tasks.

The fields default to the values specified in the schedule definition.
