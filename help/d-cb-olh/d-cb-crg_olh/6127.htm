Variables Based on CHARGE_INPUT_V

All direct variables based on the CHARGE_INPUT_V view have an application environment of Billing and a context of Charge.

Variable Definitions

Variable

Type

Description

ChgInputAccountCategory
Code&

Integer

Account category code of the account type of the account associated with the charge.

ChgInputAccountClassCode&

Integer

Account class code of the account type of the account associated with the charge.

ChgInputAccountCurrencyId&

Integer

Currency of the account associated with the charge.

ChgInputAccountId&

Integer

Account to which the charge is allocated.

ChgInputAdjustmentId&

Integer

Adjustment that generated the charge.

ChgInputApplicationEnv&

Integer

Application environment code used when the charge is allocated.

ChgInputBillable&

Integer

Billable input code used when the charge is allocated.

ChgInputBilledCharge#

Real

Billed charges for the charge.

ChgInputBilledProportion#

Real

Proportion billed for the charge.

ChgInputCharge#

Real

Value of the charge.

ChgInputChargeDate~

Date

Date-time used for the calculation of the charge.

ChgInputChargeId&

Integer

ID of a charge.

ChgInputChargeStartDate~

Date

For any charges with a non-null ChgInputNormalisedEventId&, the variable is equal to the charge start date of the normalised event (NORMALISED_EVENT.CHARGE_START_DATE).

ChgInputCurrencyId&

Integer

Currency of the charge.

ChgInputCustomerNodeId&

Integer

Customer node associated with the charge.

ChgInputFromGlCodeId&

Integer

ID of an optional second GL code to be associated with the charge.

Integer

Invoice that resulted in the generation of the charge.

Integer

ChgInputInvoiceItemNr&

Integer

ChgInputInvoiceSectionNr&

Integer

Invoice section number associated with the charge.

ChgInputInvoiceTxt$

String (80)

Optional invoice text that may be displayed in association with the charge.

String (255)

Hash key value or array index value associated with the charge for the tariff or subtotal.

ChgInputLastModified~

Date

Date-time the row was last modified.

Integer

ChgInputPaymentId&

Integer

Payment that generated the charge.

Date

ChgInputSubtotalId&

Integer

Subtotal, or total calculation that generated the charge.

ChgInputSubtotalName$

String (30)

Name allocated to the subtotal.

ChgInputTariffId&

Integer

Tariff, discount, or tax that generated the charge.

ChgInputTariffName$

String (30)

Name allocated to the tariff.

ChgInputTariffType&

Integer

Type of tariff used by the charge.

ChgInputUninvoiceInd&

Integer

Set to 1 (TRUE) if the charge is not assigned to an invoice.

Otherwise, the value of the variable is undefined.
