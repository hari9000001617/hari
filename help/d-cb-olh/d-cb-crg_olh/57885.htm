Description


Generates email configuration information for the Web Services Adapter from an interface definition. 


Note:
Interface 1 is deprecated and maintained for backward compatibility.


Attributes






Application Environment



Any







Context



Any







Method



Remote








 






Parameter



Description







InterfaceDefnId&



In:  ID of the interface definition used to generate Web Services Adapter configuration items.







RowStatus$



In:  Interface definition operation to perform.


Options are:


  • Insert
  • Update
  • Delete.







 


Returns


1 (TRUE) if successful; otherwise, an error is raised.


Remarks


This function generates:


  • Email XML configuration file
  • EMAIL configuration item
  • REMOTE_SERVICE configuration item.
The value specified in the Config File Template field of the Static Email tab of an interface template based on the WSAEmail interface type, is an XML file that is used as a template for the XML configuration file.


The values in the template file are read into an EPM unknown array, and modified, based on the values in the interface definition. The result is written to a file named with the following format:


$ATA_DATA_SERVER_CONFIG/email<SEQNR>/email.xml 


where <SEQNR> is the value of the EMAIL Sequence Number field on the Operational Email tab of the Interface Definition form.


An EMAIL configuration item is created with the following name:


EMAIL<SEQNR>


where <SEQNR> is the value specified in the EMAIL Sequence Number field on the Operational Email tab of the Interface Definition form. 


REMOTE_SERVICE configuration items are created or updated with the name:


  • <INTDFN>_RS_<INSTANCE> for cluster managed interface definitions
    where <INTDFN> is the name of the interface definition, and <INSTANCE> is the name of the instance in the InstanceStatus derived attribute table
    

  • <INTDFN>_RS for interface definitions that are not cluster managed
    where <INTDFN> is the name of the interface definition.
    

Note:
If an outgoing mail monitor is selected in the Supported Mail Monitors field on the Operational Email tab of the Interface Definition form, one or more REMOTE_SERVICE configuration items are created. If the interface definition is:
  • Defined as cluster managed, a REMOTE_SERVICE configuration item is created for every Singleview instance defined in the InstanceStatus derived attribute table
  • Not defined as cluster managed, only one REMOTE_SERVICE configuration item is created.
For information about how to define an interface definition as cluster managed, refer to the Configurer's Reference Guide for Singleview Convergent Billing (Vol. 1).
