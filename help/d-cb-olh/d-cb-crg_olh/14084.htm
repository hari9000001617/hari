treStats?[]()

Syntax

treStats?[]()

 

Description

Returns function call statistics for the current process.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

None.

 

 

Returns

Unknown array where each element of the array consists of an unknown hash that contains statistics for a specified function called (or received calls) one or more times. The elements in the hash are listed in the following table.

 

Return Hash

Key

Type

Description

Function

String

Name of the function for which statistics have been generated.

Direction

String

Direction is only present when TRE statistics are being collected from a Tuxedo server process. Valid values are:

  • IN: Statistics are for incoming requests for the specified function.
  • OUT: Statistics are for outgoing calls for the specified function.

Calls

Integer

Number of times this function was called while statistics were enabled.

ElapsedTime

Real

Total amount of elapsed time (in seconds) spent in this function while statistics have been enabled.

The accuracy of this time is dependent on the platform. On server platforms, this should be accurate to approximately 1/10000th of a second.

On the client (Windows) platform, it may only be accurate to 1/100th of a second.

ElapsedMin

Real

Minimum elapsed time of all calls to this function during which statistics have been enabled.

It is undefined if Calls=0.

ElapsedMax

Real

Maximum elapsed time of all calls to this function during which statistics have been enabled.

It is undefined if Calls=0.

ElapsedAvg

Real

Average elapsed time per call spent in this function. It is equal to ElapsedTime/Calls.

It is undefined if Calls=0.

ElapsedStd

Real

Standard deviation for calls to this function (N-1).

It is undefined if Calls<=1.

NetTime

Real

Time spent in this function, excluding outgoing calls made to other user-defined and registered functions from within this function.

It is a measure of the time required to execute the code in this function within this server process.

It is only present if Direction=IN.

NetAvg

Real

Average net time spent in this function per call.

It is equal to NetTime/Calls.

It is only present if Direction=IN.

DataRX

Real

Total amount of data transmitted (in bytes) for incoming function responses or outgoing function requests.

Tuxedo and network message overheads are not included.

DataTX

Real

Total amount of data received (in bytes) for incoming function requests or outgoing function responses.

Tuxedo and network message overheads are not included.

T0

Real

Threshold, or upper limit, (in seconds) for the first elapsed band.

T0 is defined only if thresholds are specified.

T1

Real

Threshold, or upper limit, (in seconds) for the second elapsed band (if defined).

T2

Real

Threshold, or upper limit, (in seconds) for the third elapsed band (if defined).

T3

Real

Threshold, or upper limit, (in seconds) for the fourth elapsed band (if defined).

T4

Real

Threshold, or upper limit, (in seconds) for the fifth elapsed band (if defined).

T5

Real

Threshold, or upper limit, (in seconds) for the sixth elapsed band (if defined).

T0Count

Real

Number of times the function executed with an elapsed time less than T0.

T0Count is output only if thresholds are specified.

T1Count

Real

Number of times the function executed with an elapsed time within the second band (if defined).

T2Count

Real

Number of times the function executed with an elapsed time within the third band (if defined).

T3Count

Real

Number of times the function executed with an elapsed time within the fourth band (if defined).

T4Count

Real

Number of times the function executed with an elapsed time within the fifth band (if defined).

T5Count

Real

Number of times the function executed with an elapsed time within the sixth band (if defined).

T6Count

Real

Number of times the function executed with an elapsed time within the seventh band (if defined).

T0Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the first band.

T0Elapsed is defined only if thresholds are specified.

T1Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the second band (if defined).

T2Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the third band (if defined).

T3Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the fourth band (if defined).

T4Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the fifth band (if defined).

T5Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the sixth band (if defined).

T6Elapsed

Real

Sum of elapsed times where the function executed with an elapsed time in the seventh band (if defined).

 

Remarks

For more information about thresholds and bands, refer to the Performance Guide for Singleview.
