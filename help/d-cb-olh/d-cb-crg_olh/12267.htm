CacheConfigurationDefaultOverrides?{}()

Syntax

CacheConfigurationDefaultOverrides?{}()

 

Description

Overrides default configuration attributes for cache configuration items generated from a CACHE_CONFIGURATION configuration item.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

None.

 

 

Returns

Hash containing cache configuration sizing details.

Remarks

The function is called when a Generate/Update Cache Configuration task is run to update cache configuration.

The function calls CacheConfigurationDefaults?{}() to generate a hash of default values for the configuration items. The function can be modified to override the default values in the hash returned by CacheConfigurationDefaults?{}().

The following table lists the hash of cache configuration attributes returned by CacheConfigurationDefaults?{}().
