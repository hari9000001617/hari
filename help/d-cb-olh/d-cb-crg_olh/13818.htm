stats()

Syntax

Interface 1

 

Interface 2

 
stats()
 
 
 
stats(
    function_name$)

Description

Returns statistics for functions for which standard statistics have been gathered, or for a single function, if specified.

Attributes

Application Environment

Any

Context

Any

Method

Built-in

 

Parameter

Description

function_name$

In: Function name for which statistics have been gathered.

Process and module names can also be specified in the format <type>: <name> (for example, SQL: Open).

If ** AllExpressions ** is specified, summary statistics for all EPM expressions that are not evaluated directly by a function, or associated directly with a process or module, are retrieved.

 

Returns

This function returns one of the following:

  • Hash containing standard statistics for function_name$, if specified.
  • Array of hashes containing statistics for functions for which statistics have been gathered.
  • Empty hash or empty array, if no standard statistics are available.

Remarks

This function is not usually called directly, but called using the Getstats tool (see the Performance Guide for Singleview).

This function generates only standard statistics. To also generate call graph statistical information, use gstats().

The following table lists statistics returned in each hash.

 

Statistics Hash Elements

Key

Type

Description

Calls

Real

Number of times this function was called while statistics have been enabled.

ElapsedAvg

Real

Average elapsed time per call spent in this function.

It is equal to ElapsedTime/Calls. It is undefined if Calls = 0.

ElapsedMax

Real

Maximum elapsed time (in seconds) of all calls made to this function.

It is undefined if Calls = 0.

ElapsedMin

Real

Minimum elapsed time (in seconds) of all calls made to this function.

It is undefined if Calls = 0.

ElapsedStd

Real

Standard deviation for calls to this function (N - 1).

It is undefined if Calls <= 1.

ElapsedTime

Real

Total amount of elapsed time in seconds spent in this function while statistics have been enabled.

On both server and client platforms, this measure should be accurate to at least 10 micro-seconds.

Function

String

Name of the function for the generated statistics, including the interface number (if not 1).

Processes and modules can also appear in statistics generated by gstats(). These entries will have a function name in the format of <type>: <name> (for example, SQL: Open).

The special function of ** All Expressions ** is used to gather statistics for EPM expressions that are not evaluated directly by a function, or associated directly with a process or module.

NetAvg

Real

Average net time per call spent in this function.

It is equal to NetTime/Calls.

NetTime

Real

Total amount of time spent evaluating this function (in seconds), excluding the time spent calling other functions, processes, or modules from this function.

It provides the amount of time taken to execute the code in this function.
