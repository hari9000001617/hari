File Processing Search Form

The following table lists automation parameters for the File Processing Search form, the data type, and supported functions for the parameter.

The description column is provided to supplement the description in the FILE_REGISTER_TRE_V view, or to provide information about parameters not contained within the database view.

Parameter Definitions

Parameter or Field Name

Data Type

Supported Functions

Description

FILENAME

String

View

Name or start of the name of the file.

It can include the * wildcard character.

DIRECTORY

String

View

Directory or part of the directory where the file resides.

It can include the * wildcard character.

FILE_TYPE_CODE

Integer

View

Reference code for the file type (for example, Payments).

It matches an entry in the FILE_TYPE reference type.

FILE_PROCESSING_STATUS_CODE

Integer

View

Processing status of the file (for example, Purged).

It matches an entry in the FILE_PROCESSING_STATUS reference type.

FROM_DATE

Date

View

Start of date range for files that are being processed or have completed processing on or after this date.

TO_DATE

Date

View

End of date range for files that are being processed or have completed processing on or before this date.

ERROR_CODE

String

View

Error code for files that completed processing with this error code (for example, W0001), or alternatively, the error ID (for example, 1).

ERROR_MESSAGE

String

View

String contained in the error message for files that failed processing.

INTERFACE_DEFN_ID

Integer

View

Interface definition ID used to generate a file extraction or a file load.

INSTANCE

String

View

Instance on which the file processing record resides.

ALWAYS_DISPLAY_LIST

Integer

View

Value of the Always Display List? check box. Valid values are:

  • 0: List of queries is not displayed
  • 1: List of queries is displayed.

REUSE_DETAIL_FORM

Integer

View

Value of the Reuse Display Form? check box. Valid values are:

  • 0: Use new form
  • 1: Use existing form.

SEARCHCLICK

Integer

View

When a value is entered for this parameter, the Search button is clicked on the form.The value is irrelevant.

It must be specified last.

CLEARCLICK

Integer

View

Clears the search fields.
