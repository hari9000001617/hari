BILL_RUN_SUMMARY_TRE_V

The BILL_RUN_SUMMARY_TRE_V view provides a summary of results for each operation performed on bill runs.

The view is provided only for reporting purposes.

Column Definitions

Column Name

Properties

Description

BILL_RUN_ID

NUMBER (15)

Not null

ID of the bill run.

BILL_RUN_OPERATION_CODE

NUMBER (8)

Not null

Type of operation being performed.

It is based on the BILL_RUN_OPERATION reference type.

Apart from the exception below, operations with a code 128 greater than another operation undo the effect of the first operation. For example, BILL_RUN_OPERATION_CODE value 158 undoes the effect of BILL_RUN_OPERATION_CODE value 30.

The exception is '138 - Revoke rental events and adjustments' operation, which undoes the effects of the following operations:

  • 3 (Future bill run: Rental adjustments)
  • 6 (Future bill run: Rentals, activations, cancellations)
  • 10 (Rental adjustment event generation)
  • 20 (Rental event generation (RGP)).

MIN_START_DATE

DATE

Not null

First date-time that the operation commenced on the bill run.

MAX_END_DATE

DATE

Not null

Last date-time that the operation completed on the bill run.

OPERATION_COUNT

NUMBER

Not null

Number of times the operation was performed on the bill run.

TOTAL_DURATION

NUMBER

Not null

Total duration, in seconds, of all occurrences of the operation on the bill run.

INPUT_COUNT

NUMBER

Total number of customers, for all occurrences of the operation, on the bill run.

SUCCESS_COUNT

NUMBER

Total number of customers successfully processed, by all occurrences of the operation, on the bill run.

AVG_SUCCESS_DURATION

NUMBER

Average time to successfully perform the operation on each customer.

ERROR_COUNT

NUMBER

Total number of customers not successfully processed by all occurrences of the bill run.

PRE_SKIP_COUNT

NUMBER

Total number of customers included in INPUT_COUNT but flagged not to be processed.

For the RGP and BGP processes, it is the number of customers flagged to be suppressed from billing.

POST_SKIP_COUNT

NUMBER

Total number of customers not processed after the operation completed.

It is only applicable to the BGP process, where invoices can be suppressed after being calculated.

AMOUNT

NUMBER (19,3)

Total amount billed, applied, unapplied, or revoked for all occurrences of the operation on the bill run, in the currency of the bill run type.

GENERAL_STATS1 - GENERAL_STATS10

NUMBER (22,6)

General fields 1 - 10 that can be configured to store the summation of various statistics relating to all occurrences of the operation on the bill run.

Note:

They must be configured consistently for all operations of a specific bill run type.
