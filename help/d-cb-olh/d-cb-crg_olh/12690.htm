BillRunSummary?{}()

Syntax

BillRunSummary?{}(
    const BillRunDetails?{})

 

Description

Summarises the status of a bill run and gathers statistics for the bill run.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

BillRunDetails?{}

In: Details from the BILL_RUN_TRE_V view of the bill run whose summary status and statistics are to be summarised.

 

Returns

Hash for the bill run summary.

Remarks

The interface allows the function to be specified as the summary function for bill run types. It is the default summary function used by bill run types and can also be used as a template for creating alternative summary functions.

The function is called as part of BillRunUpdate&() processing when the UPDATE_STATISTICS_IND_CODE field is set to 1; this is normally done on completion of an operation on a bill run.

The function gathers statistics by querying the BILL_RUN_OPERATION table for the bill run and aggregating the statistics from each operation performed. If failed operations are present, some statistics are gathered by directly querying other tables.

The summary status is determined by examining the number of customers that have successfully completed each billing operation; if 80% of customers have successfully completed that operation, the summary status is updated to indicate that the operation is completed successfully.

The following table lists BillRunDetails?{} hash statistics.

 

BillRunDetails?{} Hash Statistics

Statistic

Key

Description

Summary Status Code

SUMMARY_STATUS_
CODE

Summary status of the bill run.

Amount

AMOUNT

Net amount invoiced in the currency of the bill run type obtained from the AMOUNT statistics of the Invoice/Statement generation, Invoice Consolidation, and Revoke Invoice Consolidation bill run operations.

If any of the operations fail, Amount is derived by directly querying the INVOICE table for the bill run.

Invoices

GENERAL_STATS1

Net number of invoices generated for the bill run obtained from the statistics of the Invoice/Statement generation and Revoke Invoices/Statements bill run operations.

If any of the operations fail, Invoices is derived by directly querying the INVOICE table for the bill run.

Statements

GENERAL_STATS2

Net number of statements generated for the bill run obtained from the statistics of the Invoice/Statement generation and Revoke Invoices/Statements bill run operations.

If any of the operations fail, Statements is derived by directly querying the INVOICE table for the bill run.

Images

GENERAL_STATS3

Net number of invoice images generated for the bill run obtained from the statistics of the Invoice/Statement Image generation and Revoke Invoice/Statement images bill run operations.

If any of the operations fail, Images may not accurately reflect the number of invoice images generated for the bill run.

Stored Images Size

GENERAL_STATS4

Cumulative size of all invoice images generated by the Invoice/Statement Image Generate bill run operations, including invoice images that have been subsequently revoked.

If any Invoice/Statement Image generation operations fail, the stored images size statistic may not be accurate.

The stored images size statistic does not represent the size of the current invoice images stored in the database for the bill run.

Customer Nodes

GENERAL_STATS5

Cumulative number of customer nodes processed by the Invoice/Statement generation bill run operation, including customer nodes for which invoices or statements have been subsequently revoked.

If any Invoice/Statement generation operations fail, the customer node statistic may not be accurate.

Services

GENERAL_STATS6

Cumulative number of services processed by the Invoice/Statement generation bill run operation, including services for which invoices or statements have been subsequently revoked.

If any Invoice/Statement generation operations fail, the services statistic may not be accurate.

Events

GENERAL_STATS7

Cumulative number of events processed by the Invoice/Statement generation bill run operation, including events associated with invoices or statements that have been subsequently revoked.

If any Invoice/Statement generation operations fail, the events statistic may not be accurate.

Error Events

GENERAL_STATS8

Net number of rental and rental adjustment error events for the bill run obtained from the statistics for the Rental event generation, Rental adjustment event generation, and Revoke rental events and adjustments bill run operations.

If any of the operations fail, the error events statistic may not be accurate.

Rating Charges

GENERAL_STATS9

Net number of rating and rental charges updated by the bill run obtained from the statistics for the Invoice/Statement generation and Revoke Invoices/Statements bill run operations.

If any of the operations fail, the rating charges statistic may not be accurate.

Billing Charges

GENERAL_STATS10

Net number of billing charges inserted for the bill run obtained from the statistics for the Invoice/Statement generation and Revoke Invoices/Statements bill run operations.

If any of the operations fail, the billing charges statistic may not be accurate.
