Description

Inserts query details into the database.

Note:

LAST_MODIFIED and QUERY_STATUS_CODE cannot be specified, and default to the current date-time and 1 (a status of OPENED), respectively.

Attributes

Application Environment

Any

Context

Any

Method

Local

 

Parameter

Description

LastModified~

Out: Last modified date-time of the new query.

FieldNames$[]

In: List of fields being inserted.

String fields should be specified first, followed by integer fields, real fields, and date fields.

FieldValues?[]

In: Field values (corresponding to the fields listed in FieldNames$[]).

StringFieldValues$[]

In: Values of string fields to insert.

IntegerFieldValues&[]

In: Values of integer fields to insert.

RealFieldValues#[]

In: Values of real fields to insert.

DateFieldValues~[]

In: Values of date fields to insert.

 

Returns

ID assigned to the new query if successful. If the function is not successful, an error is raised.

Remarks

The names of the fields being inserted are passed in the array FieldNames$[].

The following table lists fields that can be specified when using the function, whether the fields are mandatory or optional, and default values if values are not specified.
