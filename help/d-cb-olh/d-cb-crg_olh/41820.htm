Searching

The Search tab contains the following elements:

  • Search term field

    The field at the top of the Search tab. Enter search terms in the field.

  • Buttons

    The buttons are:

    • Combined Start or Stop search and Open button.

      The button is a solid circle.

      When a search term is entered, the button image changes to include a magnifying glass. Click the button to start a search. The button image changes to include a rotating octagon to indicate that a search is in progress. If the search completes quickly, the change in the image may not be noticed.

      To stop a search, click the button.

      When a search is complete, or it has been stopped, the button image reverts to a solid button. Topics that match the search term are displayed in the results list field.

      Note:

      The search returns a maximum of 10,000 matches. If the maximum is reached, specify additional search terms to refine the search.

  • Help button.

    The Help button is a solid circle containing a question mark character. Clicking the button displays this information.

  • Results list field

    The results list field is located below the search term field, and contains the results of a search.

    Items in the list are suffixed with an acronym in parentheses to denote the location of the term.

    To open an item, do one of the following:

    • Double-click the item.
    • Select an item, which causes the Start button to change to the Open button (the image changes to include an arrow). Click the Open button.

Tip:

The search includes text in pop-up or expandable blocks and the relevant topic appears in the search results list. When a search result item is opened, the text is not visible until the pop-up or expandable block is selected.

 

Specifying Search Terms

A search term can be any of the following:

  • Word

    Specify a word; to match, a topic must contain the specified word.

  • Phrase

    Specify a phrase by enclosing it in quotes; to match, a topic must contain the exact phrase.

  • Exclusion

    Specify an exclusion by inserting a minus (-) character before a word or phrase; to match, a topic must not contain the word or phrase.

  • Logical expression

    Specify a logical expression using words, phrases, or exclusions joined by logical OR and AND operators represented by | and &, respectively; to match, a topic must cause the logical expression to be true.

    If no operator is specified between terms, an AND operator is assumed (so its use is unnecessary). For example, multiple words in the search term field matches only topics containing all the words.

    A word starting with a plus (+) character is treated as one term of a logical AND expression.

When searching, the following rules apply:

  • Character case is ignored. For example, account matches Account and aCCOUNT.
  • Partial matches are made. For example, account matches Account History form and AccountUpdate&.
  • Special characters are relevant. For example, méxico matches México, but not Mexico.

If a search word contains special characters or logical operators at the beginning of the word, enclose it with quotes.
