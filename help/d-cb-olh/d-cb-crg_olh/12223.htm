biTrcRental&()

Syntax

biTrcRental&(
    DebugLevel$)

 

Description

Sets the specified debug level for all RGP servers running in the current instance.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

DebugLevel$

In: Level of debug tracing required.

The table below lists valid debug levels.

 

Returns

Number of servers notified of the event.

Remarks

Trace information is written to $ATA_DATA_SERVER_LOG/biRental.P.<process_ID> and biRental.C.<process ID>, for the parent and child processes respectively.

Multiple options can be enabled simultaneously by adding the decimal value of the options, or by specifying mnemonics, decimal values, or hexadecimal values in a comma-separated list. Setting a new trace level turns off any existing trace levels, unless there is a + sign in front of the specified trace level, indicating that the trace level is additive.

The following alternative methods of specifying debug levels each turn on Oracle SQL and database tracing:

  • 'ORA,RDB' or '+,ORA,RDB'
  • '1,4' or '+,1,4'
  • '5'.
