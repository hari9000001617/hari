DOCUMENT_IMAGE_TRE_V

The DOCUMENT_IMAGE_TRE_V view contains the contents of generated document images.

The images are generated by the IGP, based on a supplied set of parameters and a template type of Document Image or Microsoft Word 2007 XML Document Image.

Column Definitions

Column Name

Properties

Description

DOCUMENT_IMAGE_ID

NUMBER (15)

Not null

ID of the document image.

DOCUMENT_IMAGE_NAME

VARCHAR (255)

Not null

Name of the document.

If not specified, it defaults to DOCUMENT_IMAGE_ID.

DESCRIPTION

VARCHAR (4000)

Description for the document image.

CREATED_DATE

DATE

Not null

Date-time that the record was created.

If not specified, it defaults to the current date-time.

LAST_MODIFIED

DATE

Not null

Date-time that the record was last modified.

OPERATOR_ID

NUMBER (8)

Not null

User who last modified this record.

GROUP_ID

NUMBER (8)

Not null

Security group associated with this document image.

TEMPLATE_NAME

VARCHAR (60)

Name of the template from which this document was generated.

The template must have a type of Document Image or Microsoft Word 2007 XML Document Image.

If specified when inserting or updating document images, TEMPLATE_ID cannot be specified.

When retrieving TEMPLATE_NAME, it is retrieved as of the current date-time.

TEMPLATE_ID

NUMBER (8)

Not null

ID of the template from which this document was generated.

The template must have a type of Document Image or Microsoft Word 2007 XML Document Image.

If specified when inserting or updating document images, TEMPLATE_NAME cannot be specified.

STORED_IMAGE_TYPE

VARCHAR (60)

Read-only

Type of image stored.

It is derived from the specified TEMPLATE_ID.

If backdated changes are made to the template, it is not re-derived.

STORED_IMAGE_TYPE_CODE

NUMBER (2)

Not null

Read-only

Code of the stored image type.

It is derived from the specified TEMPLATE_ID.

If backdated changes are made to the templates, it is not re-derived.

DOCUMENT_PARAMETERS

UNKNOWN HASH

Pseudo-column

Used for inserts and updates only

Specifies the parameters passed to the DocumentImageGenerate@() function to generate or regenerate the document image.

REFERENCE_ENTITY_TYPE

VARCHAR (64)

Type of external entity that created this document image.

REFERENCE_ENTITY_ID

VARCHAR (64)

ID of external entity that created this document image.

It is mandatory if REFERENCE_ENTITY_TYPE is specified.

For document images generated from Process Engine, the reference is a numeric value corresponding to the Object Key of the Process Engine object that requested the generation of this image.

CUSTOMER_NODE_ID

NUMBER (15)

Reference to the customer node for which this document was generated.

PERSON_ID

NUMBER (15)

Reference to the person or contact for which this document was generated.

CONTRACT_ID

NUMBER (15)

Reference to the contract for which this document was generated.

QUERY_ID

NUMBER (15)

Reference to the customer query associated with this document.

PENDING_PRINT_IND_CODE

NUMBER (1)

Indicates if the printing of the document image is pending, or is to be reprinted.

Set to 1 if the image is pending.

PRINTED_IND_CODE

NUMBER (1)

Set to 1 if the image has been printed or emailed.

DOCUMENT_CONTENTS

BLOB

Not null

Contents of the document in the format specified by STORED_IMAGE_TYPE_CODE.
