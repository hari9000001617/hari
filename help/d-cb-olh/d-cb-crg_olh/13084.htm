Description

Purges reference type information from all trerwdb, trerodb, trergp, trebgp, treigp, treigp_tran, trerate, trerate_tran, treproxy, tresok, and trexml Tuxedo servers, Process Engine servers, and BMP, BKR, ERT, ENM, and ERO processes.

Attributes

Application Environment

Any

Context

Any

Method

Remote

 

Parameter

Description

ReferenceTypeAbbreviation$

In: Abbreviation of the reference type to purge.

ExcludeServers$[]

In: Array of Tuxedo servers and non-Tuxedo servers to be excluded from performing this function call.

Tuxedo functions are identified using the format:

<ServerName>:<GroupId>.<ServerId>

Non-Tuxedo server processes are identified using the format:

<ProcessName>:<ProcessNumber>

 

Returns

1 (TRUE) if successful; otherwise, an error is raised.

Remarks

Additional servers or processes can be specified using Purge.NotifyTuxedoServerList$[] and Purge.NotifyProcessList$[].

The instances that the entity is purged on are controlled by the purge settings in the InstanceStatus derived attribute table.

Interface 1 is not transactional, and the purge is performed at the time of the call.

Interface 3 is used to purge a reference type that is being updated as part of the current transaction.
