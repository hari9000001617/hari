Variables Based on CUSTOMER_NODE_HISTORY_V

All direct variables based on the CUSTOMER_NODE_HISTORY_V view have an application environment of Billing and a context of Customer Node.

These variables are also available during rating. To maximise performance, configure the customer node cache (using the CUSTOMER_CACHE configuration item) to cache the field values corresponding to the set of direct variables being accessed at rating time.
