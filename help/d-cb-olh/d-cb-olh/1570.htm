File Menu

The following table lists the File menu options.

 

Menu Option

Description

Save

Saves the current form. Equivalent to clicking the Save button on the toolbar.

Cancel

Cancels the current operation. Equivalent to clicking the Cancel button in the toolbar.

Close

Closes the active window without quitting the application.

Close All

Closes all windows without quitting the application.

Change Password

Used to change a Convergent Billing password. The option is disabled if external authentication is enabled on a user's security profile.

Refresh Security Settings

Refreshes security settings after a user has made changes to them on behalf of another user.

Cache Status...

Displays query cache statistics.

Reconnect

Opens the Convergent Billing login form to reconnect as the same or another user.

Save form image to file

Saves a copy of the current form to a file.

Print form image

Prints a copy of the current.

Exit

Exits from Convergent Billing.
