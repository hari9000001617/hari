Variables Declared with 'var'

Local variables are declared within an expression block or a function by using the var keyword. For example, var temp$ declares a local string variable, temp$.

The following conditions apply:

  • The variable must be of the correct type for the data it will contain.
  • The variable has scope from its declaration up to the end of the expression block or function.
  • Each variable must be declared on a separate line.
  • Local variables take precedence over parameters, global variables, and other local variables of the same name.
  • Each array or hash type variable must have a different name (excluding data type). For example, the following expression block returns an error stating that the a variable has been declared with the wrong data type:

    {
    if (TRUE) {
    var a&[] := [1];
    } else {
    var a$[] := ['No'];
    }
    }

    This restriction does not apply to scalar variables, or hash or array type variables with the same name and data type.
