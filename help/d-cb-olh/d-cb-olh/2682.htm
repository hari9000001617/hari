Customer Query Type Definition Form

The Customer Query Type Definition form is used to view, define, and update a customer query type definition.

Enquiry made by a customer or contact that is recorded. The resolution of that enquiry generally results in information being passed back to that customer or contact. A query can range from an enquiry about available products by a prospective customer, to a complaint from a customer about charges on an invoice. If the query cannot be resolved immediately, it is progressed to a work group for resolution.

The query facility is also used to record internal enquiries. For example, the query facility can be used to enquire about a customer's credit rating by raising a query and forwarding it to an organisation's finance department.

Context: Convergent Billing, Customer Management

See also Progress, Progression, and Work Group.

The upper part of the form displays, or allows entry of, basic information about the query type.

The lower part of the form consists of the following tabs for entering or displaying more specific information:

Settings

The fields on this tab are used to specify default values for response and resolution times, default work force to handle queries, expression used to create unique query numbers, and icon for representing queries of that type.

Help Text

The Help Text tab is used to enter text to assist in handling the query (for example, questions the customer should be asked).

Entity Validation

The Entity Validation tab is used to select entity validation definitions to customise the Customer Query form, and its associated forms, for customers of a specific type.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing a Customer Query Type

To view a customer query type:

  1. Select the Maintenance Customer Care Customer Query Type option. The Customer Query Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined customer query types matching the specified criteria is displayed.
  4. Select the appropriate customer query type and click the Detail... button, or double-click the appropriate customer query type. The Customer Query Type Definition form is displayed in View Only mode.

Need more information on customer query types?

Defining a Customer Query Type

To define a customer query type:

  1. Select the Maintenance Customer Care Customer Query Type option. The Customer Query Type Search Form is displayed.
  2. Click the New... button. The Customer Query Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on customer query types?

Updating a Customer Query Type

To update a customer query type:

  1. View the appropriate customer query type.
  2. Click the Update button on the toolbar. The Customer Query Type Definition form is displayed in Update mode.
  3. Update the customer query type details as required.
  4. Click the Save button on the toolbar.

Need more information on customer query types?

Deleting a Customer Query Type

To delete a customer query type:

  1. View the appropriate customer query type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the customer query type is to be deleted.
