Invoice Message Definition Form

The Invoice Message Definition form is used to view, define, and maintain an invoice message by entering general information about the invoice message, including name, validity dates, description, and text to appear on the invoice message.

The middle part of the form is used to specify an optional invoice format, schedule, account type, or customer name for the invoice message, restricting the invoices on which the message appears. For example, specifying a schedule allows the invoice message to be added to invoices produced by all tasks for this schedule.

Note:

The Task Id field is enabled only if a schedule is specified, so a specific task can be nominated. The Account Type and Customer Name fields determine the receivers of the invoice message.

The Search Customer… button displays the Customer Search form, for locating a customer. Double-clicking on a search result row assigns the selected customer to the invoice message.

Note:

Searches based on customer name are case-insensitive.

The Qualification… button is used to create an expression list for the invoice message, to further refine the circumstances in which the invoice message is included.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing an Invoice Message

To view an invoice message:

  1. Select the Maintenance Printing Invoice Messages option. The Invoice Message Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of invoice messages matching the specified criteria is displayed.
  4. Select the appropriate invoice message and click the Detail… button, or double-click the invoice message. The Invoice Message Definition form is displayed in View Only mode.

Need more information on invoice messages?

Defining an Invoice Message

To define an invoice message:

  1. Select the Maintenance Printing Invoice Messages option. The Invoice Message Search form is displayed.
  2. Click the New… button. The Invoice Message Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on invoice messages?

Updating an Invoice Message

To update an invoice message:

  1. View the appropriate invoice message.
  2. Click the Update button on the toolbar.
  3. Update the invoice message details as required.
  4. Click the Save button on the toolbar.

Need more information on invoice messages?

Deleting an Invoice Message

To delete an invoice message:

  1. View the appropriate invoice message.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the invoice message is to be deleted.
