Adjustment Detail Search Form

The Adjustment Detail Search form is used to search for one or more adjustment records, based on the specified search criteria (for example, entry date, adjustment type, status, and number, credit and debit account numbers, and customer query number).

Selecting the Search for adjustments option enables the Search… button, to search for batches matching the specified criteria. Selecting the Create a new adjustment option enables the New… button, to create an adjustment.

The display format for the retrieved records can be determined by selecting or clearing the Always Display List? and the Reuse Detail Form? check boxes.

If multiple instances of the detail form are required for display (for example, to access details for several payments) the Reuse detail form? option should not be selected.

Caution:

If selected, the Reuse detail form? option overrides the display of information in a new form; the details are displayed in the existing form.

If a single adjustment record is returned from a search, the details of that record are displayed on the Individual Adjustment Detail form. If more than one adjustment record is returned, the records are listed on the Adjustment Detail List form. The required record can then be selected and its details displayed on the Individual Adjustment Detail form.

If selected, the Always Display List? option overrides the automatic display of the Individual Adjustment Detail form when only a single record is retrieved; in such cases the Adjustment Detail List form is displayed.

Note:

Only the Adjustment Type and Entry Date fields are enabled when the Create a new adjustment option is selected.

Associated Actions

Viewing an Adjustment

To view an adjustment:

  1. Select the Accounts Adjustments Manage Individual Adjustments option. The Adjustment Detail Search form is displayed.
  2. Enter or select the appropriate search criteria.

    Tip:

    If the adjustment date is unknown, remove the date from the Adjustment Date field.

  3. If appropriate, select the Always Display List? option.
  4. Click the Search… button.

If the search returns:

  • A single record, the Individual Adjustment form is displayed (if the Always Display List? option was not selected).
  • Multiple records, or the Always Display List? option was selected, the Adjustment List form is displayed. To access the Individual Adjustment form from the Adjustment List form, double-click the appropriate adjustment in the list of retrieved adjustments, or select the appropriate adjustment and click the Details… button. The Individual Adjustment form is displayed in View Only mode.

Recording an Adjustment

Adjustments can be entered in either of the following ways:

  • Individually
  • In batches of the same adjustment type.

Before entering an adjustment, ensure that accurate information about the customer is available, amount of the adjustment, and the adjustment type. The details need to be recorded or verified to successfully enter an adjustment.

Tip:

When entering an adjustment, it is useful to have the Customer Detail form open for the customer getting the adjustment. The customer's account number can then be copied from the Customer Detail form and pasted into the Account Number field on the relevant form.

Reversing an Adjustment

Tip:

An adjustment can only be reversed once it is posted to a customer's account. Therefore, an adjustment made as part of an adjustment batch can only be reversed if that batch is applied. Adjustments included in an adjustment batch that is not yet applied can simply be removed from the batch.

To reverse an adjustment:

  1. Enter an adjustment of equal (but opposite) value to the adjustment to be reversed.
  2. In the Description field, cross-reference the adjustment number of the reversed adjustment.

Deleting an Adjustment

To delete an adjustment not posted to a customer's account:

  1. View the appropriate adjustment.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the adjustment is to be deleted.

An adjustment posted to a customer's account can also be deleted using the same procedure if:

  • The adjustment is not included on (appears on) an invoice or statement.
  • The adjustment has not been posted to the Sales Journal.
  • Security privileges are available.

If deletion is not possible, the adjustment should be reversed and the adjustment entered against the correct account.
