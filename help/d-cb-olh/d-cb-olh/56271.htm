How Do I - System Integration Tasks - Interface Definitions

Select the option most appropriate to the required task.

View Interface Definitions

To view interface definition details:

  1. Select the Maintenance Server Interface Definition option. The Interface Definition Search form is displayed.
  2. Enter or select the relevant search criteria.
  3. Click the Search… button. A list of interface definitions matching the specified criteria is displayed.
  4. Select the relevant interface definition and click the Details... button. The Interface Definition Detail form is displayed in View Only mode.

Need more information on interface definitions?

Create Interface Definitions

To create an interface definition:

  1. Select the Maintenance Server Interface Definition option. The Interface Definition Search form is displayed.
  2. Select the Create a new interface check box.
  3. Select the interface type from the Interface Type drop-down list.
  4. Click the New… button. The Interface Definition Detail form is displayed in Insert mode.
  5. Select the interface template from the drop-down list in the Template Name field.
  6. Complete the relevant fields.
  7. Click the Save button.

Need more information on interface definitions?

Update Interface Definition Details

To update details of an interface definition:

  1. View the relevant interface definition.
  2. Click the Update button on the toolbar. The Interface Definition Detail form is displayed in Update mode.
  3. Update the relevant interface definition details.
  4. Click the Save button on the toolbar.

Need more information on interface definitions?

Delete Interface Definitions

To delete an interface definition:

  1. View the relevant interface definition.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the interface definition is to be deleted.
