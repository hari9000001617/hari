Schedule Types

Each schedule is assigned a schedule type. The schedule type defines the type of task to create, the program that the schedule task will run, and the parameters required.

Defines when an activity (or task) will be executed and the frequency at which that task will be executed.

Context: Convergent Billing

See also Pre-schedule Count and Task.

To define a schedule type, the following information is necessary:

  • Format of the command to be executed when a task based on this schedule type is run.
  • Details of any parameters the user may need to pass to the command at run time (for example, parameter type, order in which the parameters are expected, and which parameters are mandatory).
