Function Parameters

Function parameters can be passed in the following ways:

  • By value
  • As a constant value
  • By reference.

When a parameter is passed by value, the function receives a copy of the parameter; any change the function makes to this value does not affect the value of the parameter outside the function.

When a function parameter is preceded by the const keyword, it is treated as a constant and the function body is not allowed to modify its value.

Constants defined in a function can be made available to other functions or expressions using the import command.

When a parameter is passed by reference, the function has access to the original variable, and can change it.

A complete array is passed as a parameter (by value or by reference) by specifying the array name (including its base type), followed by square brackets ('[]'). Similarly, a complete hash is passed as a parameter by specifying the hash name (including its base type) followed by braces ('{}').

Note:

Only variables can be passed by reference.
