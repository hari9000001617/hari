Facility Group Definition Form

The Facility Group Definition form is used to view, define, and maintain a facility group.

Collection of one or more service options (known as facilities) defined in a product and associated with a specified service. A facility group must contain at least one facility, and can also contain details of compatible facility groups.

Context: Convergent Billing, Customer Management, Configuration Studio

See also Facility and Facility Group Compatibility.

The upper part of the form is used to specify the name of the facility group, its description, effective dates, and an icon by which the facility group can be identified.

The lower part of the form consists of the following tabs that for listing facilities to be included in the facility group and the facility groups compatible with the facility group being specified:

Facilities

The fields on the Facilities tab are used to:

  • Select one or more facilities to be included in the facility group.

    The list of available facilities (displayed on the drop-down list) includes only items that have been defined as reference items of the FACILITY reference type.

  • Remove a facility from the list of facilities specified for a facility group.

Compatibility

The fields on the Compatibility tab are used to specify one or more facility groups compatible with the current facility group. The term 'compatible' refers to whether a nominated facility group can be configured with the current facility group; some facility groups may be mutually exclusive.

The list from which to choose a compatible facility group is restricted to those facility groups currently defined.

The Compatibility tab is also used to remove a facility group from the list of compatible facility groups.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

Associated Actions

Viewing a Facility

To view a facility:

  1. Select the Maintenance Data Definition Reference Types option. The Reference Type Search form is displayed.
  2. Select Type Label in the Search field.
  3. Enter FACILITY in the Search Criteria field.
  4. Click the Search button. If the required reference type is not displayed, define it.
  5. Select the FACILITY reference type and click the Detail… button, or double-click the reference type. The Reference Type Definition form is displayed in View Only mode.

Defining a Facility

To define a facility:

  1. View the list of facilities on the Reference Type Definition form.
  2. Click the Update button on the toolbar. The Reference Type form is displayed in Update mode.
  3. Position the cursor in the last reference item listed and press the Down Arrow key on the keyboard. A new row is inserted at the bottom of the table of reference items.
  4. Enter details for the required facility.
  5. If a facility is defined in error, delete that facility's definition by selecting the table row on which the facility is defined and clicking the Delete button on the toolbar.
  6. Repeat step 4 for each facility required in the new facility group.
  7. Click the Save button on the toolbar.

Viewing a Facility Group

To view a facility group:

  1. Select the Maintenance Products Facility Group Definition option. The Facility Group Definition Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined facility groups matching the specified criteria is displayed.
  4. Select the appropriate facility group and click the Detail... button, or double-click the appropriate facility group. The Facility Group Definition form is displayed in View Only mode.

Need more information on facility groups?

Defining a Facility Group

To define a facility group:

  1. Ensure that all facilities to be included in the group have been defined.
  2. Select the Maintenance Products Facility Group Definition option. The Facility Group Definition Search form is displayed.
  3. Click the New... button. The Facility Group Definition form is displayed in Insert mode.
  4. Complete the appropriate data entry fields on the form.
  5. Click the Save button on the toolbar.

Need more information on facility groups?

Updating a Facility Group

Impact of Updating a Facility Group

To update a facility group:

  1. View the appropriate facility group.
  2. Click the Update button on the toolbar. The Facility Group Definition form is displayed in Update mode.
  3. Update the facility group details as required.
  4. Click the Save button on the toolbar.

Need more information on facility groups?

Deleting a Facility Group

Caution:

Do not delete a facility group if the facility group is referenced in one or more product instances because the product instances will be corrupted.

To delete a facility group:

  1. View the appropriate facility group.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the facility group is to be deleted.
