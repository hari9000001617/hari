How Do I - Operations - Events

Select the option most appropriate to the required task.

View a Normalised File

To view a normalised file:

  1. Select the Server Tasks Events View Normalised Files and Events option. The Normalised File Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the normalised files matching the specified criteria is displayed.
  4. Select the appropriate normalised file and click the Detail… button, or double-click the appropriate normalised file. The Normalised File Definition form is displayed in View Only mode.

Need more information on normalisation?

View a Normalised Event

To view a normalised event:

  1. View the appropriate normalised file.
  2. Select the Events tab to display any events returned for the file.
  3. To display details of the event, highlight the event in the grid and click the Details… button. The Normalised Event Detail form is displayed in View only mode.

Need more information on normalisation?

Rate a File

Events are usually rated using a bill run schedule.

Rate files on demand using a one-off task, as follows:

  1. Ensure that the file to be rated is in the appropriate input directory.
  2. Perform a one-off task of Rate Files schedule type.
  3. View the Task Details form to check the results.

Need more information on event rating?

Define a New Rerated File

To define a rerated file:

  1. View the appropriate normalised event error.
  2. Click the Update button on the toolbar.
  3. Click the Assign New button. The New Rerated File form is displayed.
  4. Enter the appropriate details in the File Name and Event Source fields and click the Create button. The New Rerated File form is closed, and the error event is assigned to the new file.
  5. Click the Save button on the toolbar.

View a Rerated File

To view a rerated file:

  1. Select the Server Tasks Events View Rerated Files and Events option. The Rerated File Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of rerated files matching the specified criteria is displayed.
  4. Select the appropriate rerated file and click the Detail… button, or double-click the appropriate rerated file. The Rerated File Detail form is displayed in View Only mode.

Need more information on normalisation?

Update a Rerated File

To update a rerated file:

  1. View the appropriate rerated file.
  2. Click the Update button on the toolbar. The Rerated File Detail form is displayed in Update mode.
  3. Update the rerated file details as required.
  4. Click the Save button on the toolbar.

Need more information on normalisation?

Delete a Rerated File

To delete a rerated file:

  1. View the appropriate rerated file.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the rerated file is to be deleted.

Need more information on normalisation?

Assign an Event to a Rerated File

To assign an error event to a rerated file:

  1. View the appropriate normalised event error.
  2. Click the Update button on the toolbar.
  3. Click the Assign/View Existing button to open the Assign/View Existing Rerated File Search form.
  4. Click the Search button to view the list of available files.
  5. Select the required file and then click the Assign button. Check that the file name chosen is displayed in the Rerated group. The sequence number of the event in this file is also visible.
  6. Click the Save button on the toolbar.
