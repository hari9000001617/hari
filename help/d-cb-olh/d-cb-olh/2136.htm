Call Usage

Convergent Billing enables searching for, viewing, and printing a customer’s usage details. The usage details display exactly what charges a customer has accrued, both billed (reported) and unbilled (outstanding in the account).

If a customer has more than one service, the exact charges (billed and unbilled) for each service number can be identified. The normalised event type associated with each event determines the type of additional usage details displayed.

Specifies the type of normalised event used to generate normalised events based on transactions occurring in Convergent Billing. An event type is also used to derive additional information about a normalised event from the information supplied by external systems.

Context: Convergent Billing

See also Normalised Event and Normalised Event File Type.

Category

A category is a grouping of entities within Process Engine (for example, classes, facts and objects) which facilitates easy transfer, filtering and general manipulation of the group.

Categories are assigned only to Convergent Billing users with access to Process Engine.
