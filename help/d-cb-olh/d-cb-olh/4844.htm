View a Direct Variable

Direct variables reference specific fields in a database record, so it is necessary to identify a specific database view to display the required variables.

To view the direct variables having a Service context:

  1. Select the Maintenance Tariffs Variables option. The Variable Definition Search form is displayed.
  2. Select Variable Name in the Search Field and the appropriate view in the Search Criteria field.

Define a Direct Variable

To define a direct variable:

  1. Select the Maintenance Tariffs Variables option. The Variable Definition Search form is displayed.
  2. Click the New... button. The Variable Definition form is displayed in Insert Mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Use a Direct Variable in an Expression

Like other variables, direct variables can be referenced in tariffs and subtotals simply by including the name of the variable in an expression.

Update a Direct Variable

To update a direct variable:

  1. View the appropriate direct variable.
  2. Click the Update button on the toolbar. The Variable Definition form is displayed in Update mode.
  3. Update the details of the direct variable as required.
  4. Click the Save button on the toolbar.

Need more information on direct variables?

Delete a Direct Variable

Note:

A direct variable that is used by another variable cannot be deleted.

To delete a direct variable:

  1. View the appropriate direct variable.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the direct variable is to be deleted.
