Cache Status Form

The Cache Status form is used to view and manage internal elements of the Convergent Billing client, which affect the performance of the Convergent Billing client. Saving the query cache to disk enhances performance. Because the cache is loaded on startup using the saved cache content, the content is available immediately it is required rather than waiting for the cache to be built.

Caches can be cleared using Clear Cache buttons and must be cleared when underlying elements are modified. Examples of underlying elements are reference types, attribute types, or entity validation components for the query cache, or icons for the icon cache.

Refreshing security settings (Ctrl+F5) implicitly clears the query and icon caches.

The form consists of the following tabs:

Query Cache

The Query Cache tab displays statistics on executed queries (Active Cache Statistics) or ready to execute (Prepared Cache Statistics). The size of each cache is displayed and can be updated.

The tab contains the Active Cache Contents and Prepared Cache Contents tabs, which list the queries in each cache.

Icon Cache

The Icon Cache tab displays the number of cached icons.

Server Message Queue

The Server Message Queue tab displays information about messages exchanged between the client and server processes, and contains the following tabs:

  • Event Queue, which displays messages received from the server which are yet to be processed.
  • Socket Response List, which displays subscribed events requiring a reply from the server but do not require an acknowledgment from the server.
  • Socket Acknowledgement List, which displays subscribed events requiring a reply from the server and require an acknowledgment from the server.

Registered Forms

The Registered Forms tab displays the forms that have been built into the client.

Automation

The Automation tab is used to specify configuration for running automation functions.

Log Files

The Log Files tab enables tracing of some types of calls, and specifies the locations of the files that contain the traces.
