trerwdb

The trerwdb Transactional Server is a general purpose server supporting global transactions using the Tuxedo two phase commit. The Transactional Server advertises services that read and write to the database and ensures the integrity of the transactions.

The server provides the following services:

  • biAccount

    Implements functions for querying and controlling account data.

  • biAccountEvent

    Processes events for the biAccount service.

  • biAdjBatch

    Processes functions that control and query adjustment batch data.

  • biAdjBatchEvent

    Processes events for the biAdjBatch service.

  • biAdjustEvent

    Processes events for the biAdjustment service.

  • biAdjustment

    Implements functions that control and query adjustment data.

  • biContract

    Performs processing for functions that query and control contact details.

  • biContractEvent

    Performs processing for events used to communicate with the biContract service.

  • biCustNdeEvent

    Processes events used to communicate with the biCustomerNode service.

  • biCustomerNode

    Implements functions for querying and controlling customer node data.

  • biDerivedAttEvt

    Processes events that communicate with the biDerivedAttr service.

  • biDerivedAttr

    Implements derived attribute table maintenance functions where the storage context is Global or Global-Shared.

  • biEquipment

    Implements functions for querying and controlling equipment data.

  • biEquipmentEvt

    Processes events for the biEquipment service.

  • biFacGroup

    Implements functions for querying and controlling facility group data.

  • biFacGroupEvent

    Processes events for the biFacGroup service.

  • biFacGroupInst

    Implements functions for querying and controlling facility group instance data.

  • biFacGrpInstEvt

    Processes events for the biFacGroupInst service.

  • biFnEvaluate

    Evaluates user defined EPM functions.

  • biInvoice

    Implements functions for querying and controlling invoice data.

  • biInvoiceEvent

    Processes events for the biInvoice service.

  • biPayBatchEvent

    Processes events for the biPaymentBatch service.

  • biPayment

    Implements functions for querying and controlling payment data.

  • biPaymentBatch

    Implements functions for querying and controlling payment batch data.

  • biPaymentEvent

    Processes events for the biPayment service.

  • biPerson

    Implements functions for querying and controlling person data.

  • biPersonEvent

    Processes events for the biPerson service.

  • biProdInstEvent

    Processes events for the biProductInst service.

  • biProductInst

    Implements functions for querying and controlling product instance data.

  • biProgress

    Implements functions that control and query customer query progress data.

  • biProgressEvent

    Processes events for the biProgress service.

  • biQuery

    Implements functions for querying and controlling customer query data.

  • biQueryEvent

    Processes events for the biQuery service.

  • biService

    Implements functions for querying and controlling service data.

  • biServiceEvent

    Processes events for the biService service.

  • biSQLExecEvent

    Processes events for the biSQLExecute service.

  • biSQLExecute

    Implements functions that perform SQL Data Manipulation Language (DML) queries against the Convergent Billing database. The service is provided in read-write servers that can participate in global transactions.

  • biTrcEpm

    Processes the event for setting the EPM debug level.
