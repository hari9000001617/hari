Audit Records

Audit records are used to track changes to selected tables in the database.

The Audit Search form is used to search the audit records of a specific table and restrict the records based on a specific user, an open or closed date and time range, or the additional criteria specified by a Where clause.

Reuse the last audit form rather than opening a new audit form every time a search is performed.

For information about enabling auditing of a table and viewing the audit records generated, refer to the System Administration Guide for Singleview.

Choosing a Table

The form provides a drop-down list of the tables that can be queried for audit records. The list contains only tables configured for auditing.

Choosing an User

The search for audit records can be restricted to those created by a specific operator. Database changes that have been made automatically by the system (for example, as part of the billing cycle), are assigned the user associated with the billing process.

Choosing an Open or Closed Date and Time Range

The search for audit records can be restricted to those created before or after a specific date and time or between. If the time field is left blank, the selected time defaults to one second past midnight on the chosen day. As an example, to restrict the date range to a single day, the start time could be left blank but an end time of 23:59:59 entered.

Using an SQL Where Clause

The search for audit records can be restricted to those matching additional criteria by entering an SQL Where clause.

Note:

The clause entered must be valid within the context of the table being examined and the syntax of SQL. The clause must not contain the Where keyword. It is necessary to be familiar with the database table structure and SQL to be able to use this feature of the search form.

Reuse Last Audit Form

More than one audit record can be open at the same time. Enter new selection criteria in the Audit Search form and click the Audit button. However, if the new selection should overwrite the previous selection, select the Reuse last audit form check box before clicking the Audit... button.
