Equipment Type Definition Form

The Equipment Type Definition form is used to enter, view, and update details of an equipment type.

Specifies any defaults or entity validation for equipment specified with that type. If required, an icon can be specified to identify each equipment type.

Context: Convergent Billing, Customer Management, Configuration Studio

See also Equipment, Service, Service Type, and Icon.

The upper part of the form is used to enter or view general information about the equipment type, such as its name and description.

The lower part of the form consists of the following tabs, for entering more specific details about the equipment type and the mechanism by which equipment numbers and service names are generated:

Settings

The fields on the Settings tab are used to enter general details about the equipment type being defined. For example, the following can be specified:

  • Equipment form to be displayed
  • Icon to be used
  • Reference types that specify possible states
  • Warranty periods.
  • Ranges of equipment and whether a
  • Network addresses to be used as the basis for the equipment type.
  • Quarantine period
  • Status for the equipment type.

Quarantine and status can be used to specify a period of time in which the equipment is unavailable for use (for example, after a service number is cancelled). The service number is then held in quarantine for the period specified until it is reactivated.

Encoding/Decoding

The fields on this tab are used to enter expressions to encode and decode equipment serial numbers and/or network addresses. The make, model, and sub-type codes of the equipment can be used in the expression.

The encode and decode fields for:

  • Serial numbers are enabled only if the Allow ranges of equipment? field on the Settings tab are selected.
  • The network address are enabled only if both check boxes on the Settings tab are selected.

Entity Validation

The fields on this tab are used to enter:

  • Entity validation definitions
  • Optional set of validation rules to be applied against any equipment type.

Equipment validation can be used to provide a default layout for customers on the Equipment Detail form. Similar validation definitions can be specified for the Equipment Search form, Customer Details form, Equipment Reallocation form, and Equipment Split form.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing an Equipment Type

To view an equipment type:

  1. Select the Maintenance Products Equipment Types option. The Equipment Type Search form is displayed.
  2. Specify the appropriate search criteria and search method.
  3. Click the Search button. A list of equipment types matching the specified criteria is displayed.
  4. Select the appropriate equipment type and click the Detail... button, or double-click the appropriate equipment type. The Equipment Type Definition form is displayed in View Only mode.

Need more information on equipment types?

Defining an Equipment Type

To define an equipment type:

  1. Select the Maintenance Products Equipment Types option. The Equipment Type Search form is displayed.
  2. Click the New... button. The Equipment Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on equipment types?

Updating an Equipment Type

To update an equipment type:

  1. View the appropriate equipment type.
  2. Click the Update button on the toolbar. The Equipment Type Definition form is displayed in Update mode.
  3. Update the equipment type details as required.
  4. Click the Save button on the toolbar.

Need more information on equipment types?

Deleting an Equipment Type

To delete an equipment type:

  1. View the appropriate equipment type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the equipment type is to be deleted.

Need more information on equipment types?

Configuring Equipment Allocation

To configure equipment allocation:

  1. Define an entity validation definition to determine equipment allocation.
  2. Select the Maintenance Products Product Definition option. The Product Definition Search form is displayed.
  3. Click the New… button. The Product Definition form is displayed in Insert mode.
  4. Follow the steps described in Prepare for Product Definition to populate the form.
  5. Click the Services tab. Place the cursor in the Allocation Form Entity Validation field and click the down arrow to insert the entity validation definition created in Step 1.
  6. Save the product definition.

    When a product instance is created, the configured equipment allocation is applied on the Equipment Allocation form when allocating equipment of the specified equipment type.

Note:

The entity validation definition is usually set up to restrict the number of items displayed when allocating an equipment to a service. For example, the definition may restrict the number of pools available from which equipment can be allocated (by Service Prefix, Region, or Pool Type).
