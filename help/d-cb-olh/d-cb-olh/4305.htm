How Do I - Product Configuration Tasks - Facility Groups

Select the option most appropriate to the required task.

View a Facility

To view a facility:

  1. Select the Maintenance Data Definition Reference Types option. The Reference Type Search form is displayed.
  2. Select Type Label in the Search field.
  3. Enter FACILITY in the Search Criteria field.
  4. Click the Search button. If the required reference type is not displayed, define it.
  5. Select the FACILITY reference type and click the Detail… button, or double-click the reference type. The Reference Type Definition form is displayed in View Only mode.

Need more information on facility groups?

Define a Facility

To define a facility:

  1. View the list of facilities on the Reference Type Definition form.
  2. Click the Update button on the toolbar. The Reference Type form is displayed in Update mode.
  3. Position the cursor in the last reference item listed and press the Down Arrow key on the keyboard. A new row is inserted at the bottom of the table of reference items.
  4. Enter details for the required facility.
  5. If a facility is defined in error, delete that facility's definition by selecting the table row on which the facility is defined and clicking the Delete button on the toolbar.
  6. Repeat step 4 for each facility required in the new facility group.
  7. Click the Save button on the toolbar.

Delete a Facility

Caution:

Deleting a facility that is referenced by one or more database entities may corrupt the Convergent Billing database. Unless a facility has been defined in error, do not delete a facility.

To delete a facility:

  1. View the list of facilities on the Reference Type Definition form.
  2. Select the table row containing the facility to be deleted.
  3. Click the Delete button on the toolbar.
  4. When prompted, click the Yes button to confirm that the facility is to be deleted.

View a Facility Group

To view a facility group:

  1. Select the Maintenance Products Facility Group Definition option. The Facility Group Definition Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined facility groups matching the specified criteria is displayed.
  4. Select the appropriate facility group and click the Detail... button, or double-click the appropriate facility group. The Facility Group Definition form is displayed in View Only mode.

Need more information on facility groups?

Define a Facility Group

To define a facility group:

  1. Ensure that all facilities to be included in the group have been defined.
  2. Select the Maintenance Products Facility Group Definition option. The Facility Group Definition Search form is displayed.
  3. Click the New... button. The Facility Group Definition form is displayed in Insert mode.
  4. Complete the appropriate data entry fields on the form.
  5. Click the Save button on the toolbar.

Need more information on facility groups?

List Facilities Associated with a Facility Group

To view a list of facilities associated with a facility group:

  1. View the appropriate facility group.
  2. Click the Facilities tab. A list of the facilities associated with the facility group is displayed.

Include a Facility in a Facility Group

To include a facility in a facility group:

  1. View the appropriate facility group.
  2. Click the Facilities tab.
  3. Position the cursor over the data entry field in the Facilities column and click the left mouse button.
  4. Click the arrow on the right-hand side of the field and select one of the listed facilities.
  5. To include another facility in the facility group, click the Insert button and then repeat the previous step.

    Note:

    The order in which the facilities are listed is irrelevant.

  6. Repeat the previous step for each facility to be included in the facility group.
  7. Click the Save button on the toolbar.

Need more information on facility groups?

Remove a Facility from a Facility Group

To remove a facility from the list of facilities specified for a facility group:

  1. View the appropriate facility group.
  2. Click the Facilities tab.
  3. Select the table row containing the facility to be removed from the facility group.
  4. Click the Delete button. The facility is removed from the list.
  5. Repeat steps three and four for each facility to be removed.
  6. Click the Save button on the toolbar.

Need more information about facility groups?

Include a Facility in List of Compatible Facility Groups

To include a facility group in the list of compatible facility groups:

  1. View the appropriate facility group.
  2. Click the Compatibility tab.
  3. Position the cursor over the data entry field in the Compatible Facility Group column and click the left mouse button.
  4. Click the arrow on the right-hand side of the field and select one of the listed facility groups.
  5. Enter a description of the facility group in the Description field.
  6. To specify another compatible facility group, click the Insert button and then repeat the previous two steps.
  7. Repeat the previous step for each facility group to be included.

    Note:

    The order in which the compatible products are listed is irrelevant.

  8. Click the Save button on the toolbar.

Remove a Facility from a List of Compatible Facility Groups

To remove a facility group from the list of compatible facility groups:

  1. View the appropriate facility group.
  2. Click the Compatibility tab.
  3. Select the table row containing the facility group to be removed from the list of compatible facility groups.
  4. Click the Delete button on the toolbar. The facility group is removed from the list.
  5. Repeat steps three and four for each facility group to be removed.
  6. Click the Save button on the toolbar.

Update/Modify a Facility Group

Impact of Updating a Facility Group

To update a facility group:

  1. View the appropriate facility group.
  2. Click the Update button on the toolbar. The Facility Group Definition form is displayed in Update mode.
  3. Update the facility group details as required.
  4. Click the Save button on the toolbar.

Delete a Facility Group

Caution:

Do not delete a facility group if the facility group is referenced in one or more product instances because the product instances will be corrupted.

To delete a facility group:

  1. View the appropriate facility group.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the facility group is to be deleted.
