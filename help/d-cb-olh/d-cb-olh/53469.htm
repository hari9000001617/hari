Profile Definition Form

The Profile Definition form is used to view, define, and update profile definitions.

Set of attributes to control a user's access to, and usage of, elements of Convergent Billing and Process Engine.

Context: Convergent Billing, Process Engine, Configuration Studio

See also Access Level, Permission, Security Group, User, Form Role, and Work Group.

The upper part of the form is used to enter general details about the profile.

The lower part of the form consists of the following tabs allowing more specific details to be displayed or entered:

Access Level

The Access Level tab contains a set of check boxes that define access rights to Convergent Billing for users assigned the profile.

Security Group

The Security Group tab contains a grid specifying the groups to which users with this profile belong. The primary security group is added to this grid automatically. The Access Level Settings button is used to select the access levels the user should have to data belonging to each security group.

Work Group

The Work Group tab contains a grid specifying the work groups to which users with this profile belong. The Access Level Settings button is used to select the access levels the user should have to queries for each work group.

Form Role

The Form Role tab contains a grid specifying the form roles assigned to users with this profile.

Function Role

The Function Role tab contains a grid specifying the user-defined functions that are accessible to users with this profile.

Oracle Profile

The Oracle Profile tab is visible only if the Maintain Oracle Profile? check box is selected. An Oracle profile is created with the same name as the Convergent Billing profile name. The name must conform to Oracle's profile naming requirements:

  • All characters uppercase
  • First character in the set [A-Z]
  • Subsequent characters in the set [A-Z, _, 0-9].

The Password Attributes group defines failed attempts to log in, and life time (days) and password reuse. The Reuse Max field is not visible until the Enable Password Reuse? field is selected.

General

The General tab is used to add new fields to the Profile Definition form using entity validation.

Used to customise fields, add fields to generic forms, validate fields and forms, and add field security. It can also be used to customise an entity's data model using derived non-stored attribute types, attribute display labels, attribute interface labels, and custom key field functions.

Entity validation is used in conjunction with reference types, attribute types, lookup functions and context-sensitive help.

Context: Convergent Billing

See also Attribute Type and Reference Type.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing a Profile

To view a profile definition:

  1. Select the Maintenance Security Profiles option. The Profile Definition Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of profile definitions matching the specified criteria is displayed.
  4. Select the appropriate profile definition and click the Detail… button, or double-click the appropriate profile definition. The Profile Definition form is displayed in View Only mode.

Need more information on security?

Defining a Profile

To define a profile definition:

  1. Select the Maintenance Security Profiles option. The Profile Search form is displayed.
  2. Click the New… button. The Profile Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on security?

Updating a Profile

To update a profile definition:

  1. View the appropriate profile definition.
  2. Click the Update button on the toolbar. The Profile Definition form is displayed in Update mode.
  3. Update the profile definition details as required.
  4. Click the Save button on the toolbar.

Need more information on security?

Deleting a Profile

To delete a profile definition:

  1. View the appropriate profile definition.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the profile is to be deleted.
