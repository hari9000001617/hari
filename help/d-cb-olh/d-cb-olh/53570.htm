How Do I - System Administration Tasks - Partition Allocation

Select the option most appropriate to the required task.

Configure Automatic Partition Allocation

To customise utilisation metrics used for automatic partition allocation:

  1. Select the Maintenance Tariffs Derived Attributes Table Maintenance option. The Derived Attribute Table Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of derived attribute table definitions matching the specified criteria is displayed.
  4. Select the PartitionUtilisation table name and click the Detail... button. The PartitionUtilisation form is displayed.
  5. Click the Update button.
  6. Edit the Utilisation Metric field as required.
  7. Edit the Target Utilisation field as required.
  8. Click the Save button.

View Automatic Partition Allocation

To view partition allocation configuration:

  1. Select the Maintenance Tariffs Derived Attributes Table Maintenance option. The Derived Attribute Table Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of derived attribute table definitions matching the specified criteria is displayed.
  4. Select the PartitionUtilisation table name and click the Detail... button. The PartitionUtilisation Definition form is displayed.

Need more information on table partitioning?

Collect Actual Partition Utilisation

To collect actual partition utilisation using a one-off task:

  1. Select the Server Tasks One-off Task option. The One-off Task Definition form is displayed.
  2. Select Calculate Partition Utilisation from the Schedule Type drop-down list. The lower part of the form displays the parameters associated with the task.
  3. Enter appropriate parameter values.
  4. Click the Save button on the toolbar. The one-off task is submitted and the PartitionUtilisation derived attribute table is updated.
