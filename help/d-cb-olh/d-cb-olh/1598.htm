Number Generation Mechanism

The fields on the Encoding/Decoding tab are used to enter expressions to encode and decode equipment serial numbers and/or network addresses. In each case, the make, model, and subtype codes of the equipment can be used in the expression. The expressions for encoding and decoding must have a decoded serial number or network address of the same length.

The encode and decode fields for serial numbers are enabled only if the Allow ranges of equipment? option on the Settings tab is selected.

  • The encode expression defines the method by which to generate the serial number for each piece of equipment based on a single unique number (defined as a local variable, @value$) as input. The expression returns the serial number as a string.
  • The decode expression defines the method by which to generate a unique number from the equipment's serial number. The serial number is defined as the local variable @value$.

The Encode and Decode fields for the Network Address are enabled only if the Equipment has network address? option and the Allow Ranges of Equipment? check box on the Settings tab is selected.

  • The encode expression defines the method by which to generate the network address of the device based on a single unique number (defined as a local variable @value$) as input. The expression returns the network address as a string.
  • The decode expression defines the method by which to generate a unique number from the equipment's network address. The network address is defined as the local variable @value$.

Number Expression Examples

Serial Number

Encode Expression

‘ABC-GSM-‘ + to_string(@value$,’000’)

Decode Expression

to_long(substr(@value$,8,3))

 

Network Address

Encode Expression

‘ABC-GSM-NA-‘ + to_string(@value$,’000’)

Decode Expression

to_long(substr(@value$,11,3))
