Reporting Entity Search Form

The Reporting Entity Search form is used to search for one or more reporting entities, using a specific field, criteria, method, and date.

The lower part of the form is used to list each of the reporting entities returned by the search that matches the specified criteria.

Multiple instances of the Reporting Entity Definition form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Viewing a Reporting Entity

To view a reporting entity:

  1. Select the Maintenance Accounts Reporting Entities option. The Reporting Entity Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of reporting entities matching the specified search criteria is displayed.
  4. Select the appropriate reporting entity and click the Detail… button, or double-click the appropriate reporting entity. The Reporting Entity Definition form is displayed in View Only mode.

Defining a Reporting Entity

To define a reporting entity:

  1. Select the Maintenance Accounts Reporting Entities option. The Reporting Entity Search form is displayed.
  2. Click the New… button. The Reporting Entity Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on reporting entities?

Updating a Reporting Entity

To update a reporting entity:

  1. View the appropriate reporting entity.
  2. Click the Update button on the toolbar. The Reporting Entity Definition form is displayed in Update mode.
  3. Update the reporting entity as required.
  4. Click the Save button on the toolbar.

Need more information on reporting entities?

Deleting a Reporting Entity

To delete a reporting entity:

  1. View the appropriate reporting entity.
  2. Click the Delete button on the toolbar.
  3. When prompted, specify the date from which the entity is deleted and click the Yes button to complete the delete operation.

Need more information on reference types?

Viewing an Historical Data Record

To view a historical record:

  1. Display the details of the current data record on the appropriate form. For example, to display the details of a customer, access and display the Customer Detail form with the customer's details.
  2. Click the History button on the toolbar. The appropriate history form is displayed listing each of the historical records associated with the current record.
  3. To view a specific historical record, double-click the appropriate entry in the list of historical records, or select one of the entries and click the Details... button. The relevant form is displayed with details of the historical record.
