How Do I - System Administration Tasks - Charge Output

Select the option most appropriate to the required task.

View a Charge Output Definition

To view a charge output definition:

  1. Select the Maintenance Server Charge Output option. The Charge Output Definition Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined charge output definitions matching the specified criteria is displayed.
  4. Select the appropriate charge output definition and click the Detail... button, or double-click the appropriate charge output definition. The Charge Output Definition form is displayed in View Only mode.

Need more information on charge output definitions?

Define a Charge Output Definition

To define a charge output definition:

  1. Select the Maintenance Server Charge Output option. The Charge Output Definition Search form is displayed.
  2. Click the New... button. The Charge Output Definition form is displayed in Insert Mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Charge Output Assignments… button to open the Expressions List form.
  5. Enter expressions to calculate the required values for each direct variable.
  6. Click the Close button on the toolbar to close the Expressions List form and make the Charge Output Definition form active.
  7. Click the Save button on the toolbar.

Need more information on charge output definitions?

Update a Charge Output Definition

To update a charge output definition:

  1. View the appropriate charge output definition.
  2. Click the Update button on the toolbar. The Charge Output Definition form is displayed in Update Mode.
  3. Update the charge output definition details as required.
  4. Click the Save button on the toolbar.

Need more information on charge output definitions?

Delete a Charge Output Definition

To delete a charge output definition:

  1. View the appropriate charge output definition.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the charge output definition is to be deleted.

View Charge Output Assignments

To view charge output assignments:

  1. View the appropriate charge output definition.
  2. Click the Charge Output Assignments button.
  3. The Expression List form is displayed in View Only mode.

Need more information on charge output definitions?

Delete Charge Output Assignments

To delete charge output assignments:

  1. View the appropriate charge output definition.
  2. Click the Charge Output Assignments button. Details of the selected charge output assignments are displayed on the Expression List form.
  3. Click the Delete button on the toolbar.
  4. When prompted, click the Yes button to confirm that the charge output assignment is to be deleted.

View Charge Output Definition History

To view charge output definition history:

  1. View the appropriate charge output definition.
  2. Click the History button on the toolbar.
  3. The History of Charge Output Definition form in displayed in View Only Mode.
