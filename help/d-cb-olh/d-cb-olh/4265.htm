How Do I - Customer Care Configuration Tasks - Icons

Select the option most appropriate to the required task.

View the Icon Definition Form

To display the Icon Definition form, select the Maintenance Data Definition Icons option. The Icon Definition form is displayed in View Only mode.

Need more information on icons?

Create an Icon

To create an icon:

  1. View the Icon Definition form.
  2. Click the Insert button. An empty row is inserted in the list of icons.
  3. Enter the name and description of the icon.
  4. Click on a colour from the Paint Colour group, then either click, or click and drag, in the Icon Preview group to create a picture with that colour.

    Note:

    To clear an image in the Icon Preview group, click the Clear Icon button.

  5. Click the Save button on the toolbar.

Need more information on icons?

Update an Icon

To update an icon:

  1. View the Icon Definition form.
  2. Select the appropriate icon and click the Update button on the toolbar. The Icon Definition form is displayed in Update mode.
  3. Update the icon as required.

    Note:

    To clear an image in the Icon Preview group, click the Clear Icon button.

  4. Click the Save button on the toolbar.

Need more information on icons?

Delete an Icon

To delete an icon:

  1. View the Icon Definition form.
  2. Select the appropriate icon and click the Update button on the toolbar. The Icon Definition form is displayed in Update mode.
  3. Click the Delete button on the toolbar.
  4. When prompted, click the Yes button to confirm that the icon is to be deleted.

Need more information on icons?

Import an Icon

To import an icon:

  1. View the Icon Definition form.
  2. Click the Update button on the toolbar. An empty row is inserted in the list of icons.
  3. Click the Load Icon from file button. The Open Icon form is displayed.
  4. Browse for the appropriate file or type the name of the file in the File Name field, then click the OK button.
  5. If the graphic was created with more than 16 colours, a confirmation message is displayed. Click Yes to continue.
  6. If the graphic was created with more than 14 x 14 pixels, a confirmation message is displayed. To resize the drawing to 14 x 14 pixels, click the Yes button. To import the drawing without resizing it, click the No button.
  7. The graphic or icon is displayed in the Icon Preview group.
  8. Click the Save button on the toolbar.

Need more information on importing icons?

Save an Icon as a BMP File

To save an icon as a BMP file:

  1. View the Icon Definition form.
  2. Select the appropriate icon and click the Update button on the toolbar. The Icon Definition form is displayed in Update mode.
  3. Click the Save Icon to File button. The Save Icon form is displayed.
  4. Select the appropriate directory at the top of the form.
  5. Enter a name in the File Name field.
  6. Click the Save button on the toolbar.

Need more information about icons?

Clear the Icon Cache

To clear the icon cache:

  1. From the menu select File Cache Status… option.
  2. Select the Icon Cache tab.
  3. Click the Clear Cache button.
  4. The icon cache can also be cleared by refreshing the security settings.
