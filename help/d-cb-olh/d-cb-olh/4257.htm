How Do I - Financial Configuration Tasks - Currencies

Select the option most appropriate to the required task.

View a Currency

To view a currency and its associated conversion rates:

  1. Select the Maintenance Accounts Currencies option. The Currency Search form is displayed.
  2. Specify the appropriate search criteria and search method.
  3. Click the Search button. A list of currencies matching the specified criteria is displayed.
  4. Select the appropriate currency and click the Detail... button, or double-click the appropriate currency. The Currency Definition form is displayed in View Only mode.

Define a Conversion Rate

To define a currency and associated conversion rate to an existing currency:

  1. Ensure that the currency for which a conversion rate needs to be added is defined. If not, define the currency before proceeding.
  2. Select the Maintenance Accounts Currencies option. The Currency Search form is displayed.
  3. Set the Search field to Currency Name.
  4. In the Search Criteria field, enter the name of the currency to which a conversion rate is being added, then click the Search button.
  5. Select the currency record to add a conversion rate to, and click the Detail… button. The Currency Definition form is displayed in Insert mode.
  6. Position the cursor in the table in the lower part of the form and click the Update button on the toolbar.
  7. Click the Insert button on the toolbar. An empty row is inserted in the table.
  8. Select the required currency from the drop-down list of currencies and then enter the conversion rate between the selected currency and the local currency.

    Note:

    The drop-down list displays only the names of currencies that have been defined in the system.

  9. Click the Save button on the toolbar.

Define a Currency

To define a currency:

  1. Select the Maintenance Accounts Currencies option. The Currency Search form is displayed.
  2. Click the New... button. The Currency Definition form is displayed in Insert mode.
  3. Complete the data entry fields on the upper part of the form and insert a row in the table in the lower part of the form for each currency requiring conversion. To allow the billing engine to perform automatic currency conversion (using the conversion rates specified in the lower part of the form), select the Convertible? check box.
  4. Click the Save button on the toolbar.

Need more information on currencies?

Remove a Conversion Rate

To remove a conversion rate:

  1. View the appropriate currency.
  2. Position the cursor in the table in the lower part of the form and click the Update button on the toolbar.
  3. Select the row containing the currency (and its associated conversion rate) to be removed.
  4. Click the Delete button on the toolbar.
  5. Click the Save button on the toolbar.

Need more information about currencies?

Update a Currency

To update a currency:

  1. View the appropriate currency.
  2. Click the Update button on the toolbar. The Currency Definition form is displayed in Update mode.
  3. Update the currency details as required.
  4. Click the Save button on the toolbar.

Need more information on currencies?

Delete a Currency

To delete a currency:

  1. View the appropriate currency.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the currency is to be deleted.
