Adjustment Type Definition Form

The Adjustment Type Definition form is used to view, define, update, or delete an adjustment type.

The upper part of the form displays, or allows entry of, basic details about the adjustment type, such as its name and a description.

The lower part of the form consists of the following tabs for specific details to be entered and viewed:

Settings

The Settings tab is used to enter a default adjustment description, expressions to generate adjustment numbers and adjustment batch numbers, and to select an entity validation definition to customise the Individual Adjustment Detail form.

Account Details

The Account Details tab is used to:

  • Select the default currency associated with the adjustment type.

    Note:

    If a default currency is selected, the currency must be the same as the currencies associated with the debit and credit accounts.

  • Define account types and default GL codes associated with the adjustment type when adjustments are applied or deducted.
  • Specify the function used to determine how adjustments are allocated to invoices.

Currency Conversion

The Currency Conversion tab is used to select the function used to determine discrepancy amounts caused by the currency conversion.

The form can be displayed in one of three form modes, the current mode being indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing an Adjustment Type

To view an adjustment type:

  1. Select the Maintenance Customer Care Adjustment Types option. The Adjustment Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of adjustment types matching the specified criteria is displayed.
  4. Select the appropriate adjustment type and click the Detail... button, or double-click the appropriate adjustment type. The Adjustment Type Definition form is displayed in View Only mode.

Need more information on adjustment types?

Defining an Adjustment Type

To define an adjustment type:

  1. Select the Maintenance Customer Care Adjustment Types option. The Adjustment Type Search form is displayed.
  2. Click the New... button. The Adjustment Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on adjustment types?

Updating an Adjustment Type

To update an adjustment type:

  1. View the appropriate adjustment type.
  2. Click the Update button on the toolbar. The Adjustment Type Definition form is displayed in Update mode.
  3. Update the adjustment type details as required.
  4. Click the Save button on the toolbar.

Need more information on adjustment types?

Deleting an Adjustment Type

To delete an adjustment type:

  1. View the appropriate adjustment type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the adjustment type is to be deleted.
