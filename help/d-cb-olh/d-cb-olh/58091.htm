Security Group Definition Form

The Security Group Definition form is used to view, define, and update security group definitions. The form displays each security group and a description of each security group.

The form is displayed in one of two form modes, View Only and Update, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Form Mode

Description

View Only mode

Allows information held in Convergent Billing to be viewed, but not updated (modified).

Insert mode

Allows new information to be recorded (entered) in Convergent Billing.

Update mode

Allows information held in Convergent Billing to be viewed and updated (modified).

Associated Actions

Viewing a Security Group

To view a security group:
  1. Select the Maintenance Security Security Groups option. The Security Group Definition form is displayed in View Only mode, with security groups and their descriptions.

Defining a Security Group

To define a security group:
  1. Select the Maintenance Security Security Groups option. The Security Group Definition form is displayed in View Only mode.
  2. Select the line above where a new security group is to be inserted, and click the Insert button on the toolbar.
  3. Enter the name and description of the new security group.
  4. Click the Save button on the toolbar.

Need more information on security?

Updating a Security Group

To update a security group:
  1. View the appropriate security group.
  2. Click the Update button on the toolbar. The Security Group Definition form is displayed in Update mode.
  3. Update the security group definition as required.
  4. Click the Save button on the toolbar.

Need more information on security?

Deleting a Security Group

To delete a security group:
  1. View the appropriate security group.
  2. Click the Delete button on the toolbar. A Confirm window is displayed.
  3. When prompted, click the Yes button to confirm deletion.
