Receivable Type Search Form

The Receivable Type Search form is used to search for one or more receivable types, using a specific field, criteria, method, and date.

Accounting entities used to:

  • Separate the amount owing on an invoice into different components. When payments and adjustments are applied to an invoice, the payment or adjustment amount is allocated to the receivable type components using an allocation algorithm (for example, based on the priority of the receivable type).
  • Generate invoice receivable type history records that represent the accounts receivable sub-ledger. The invoice receivable type history can be reconciled against the balances of the accounts receivable amounts uploaded to an external GL system.

Context: Convergent Billing, Configuration Studio

See also GL Code and Invoice Receivable Type History Records.

The lower part of the form is used to list receivable types returned by the search that match the specified criteria.

Multiple instances of the form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Viewing a Receivable Type

To view a receivable type:

  1. Select the Maintenance Accounts Receivable Types option. The Receivable Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of receivable types matching the specified search criteria is displayed.
  4. Select the appropriate receivable type and click the Detail… button, or double-click the appropriate receivable type. The Receivable Type Definition form is displayed in View Only mode.

Defining a Receivable Type

To define a receivable type:

  1. Select the Maintenance Accounts Receivable Types option. The Receivable Type Search form is displayed.
  2. Click the New… button. The Receivable Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on receivable types?

Deleting a Receivable Type

To delete a receivable type:

  1. View the appropriate receivable type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the receivable type is to be deleted.

Need more information on receivable types?

Updating a Receivable Type

To update a receivable type:

  1. View the appropriate receivable type.
  2. Click the Update button on the toolbar. The Receivable Type Definition form is displayed in Update mode.
  3. Update the receivable type as required.
  4. Click the Save button on the toolbar.
