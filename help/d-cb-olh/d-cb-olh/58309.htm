Entitlement Detail Form

The Entitlement Detail form is used to view entitlement details.

The upper part of the form displays basic details about the selected entitlement.

The lower part of the form consists of the following tabs that allow more specific information to be displayed:

Details

The Details tab displays details about the entitlement being created. The fields enabled on this tab are determined by the type of entitlement being created.

Recurring

The Recurring tab is displayed only when the Recurring option is specified in the Entitlement Class group on the upper part of the Entitlement Detail form.

The tab displays details of an allocation that recurs on a periodic basis for the entitlement. The tab also provides the option of suspending the recurring allocation and setting all allocations to the entitlement to expire.

Date Range

The Date Range tab displays details about the:

  • Entities that are associated to the entitlement
  • Duration of the entitlement being associated with the entity (date-time the entitlement becomes valid, and date-time the entitlement expires).

General

The General tab is used to display additional information about the entitlement.

Note:

The fields on this tab depend on the entity validation of the entitlement type.

Allocation

The Allocation tab displays details of all allocations for the entitlement. Each allocation is displayed on a separate row in the grid, and sorted in descending order of expiry and start date.

Associated Actions

View an Entitlement

Entitlements can be viewed from the following forms:

  • Customer Detail
  • Product Instance Detail
  • Service Detail.

To view an entitlement using the Customer Detail form:

  1. View the customer’s details.
  2. Click the Entitlements button.

To view an entitlement using the Product Instance Detail form:

  1. Display the Customer Hierarchy form for the appropriate customer.
  2. Select the Products or Companion Products option from the Display for customer group. Summary information about each product instance and its associated services for the specified customer is displayed in the right-hand panel.
  3. Double-click the appropriate product record, or select the record and click the Details… button. The Product Instance Detail form is displayed.
  4. Select the Companion Products tab on the Product Instance Detail form.
  5. Click the Entitlements button .

To view an entitlement using the Service Detail form:

  1. Display the Customer Hierarchy form for the appropriate customer.
  2. Select Products and Services options from the Display for customer group. Summary information about each product instance and its associated services for the specified customer is displayed in the right-hand panel.
  3. Double-click the appropriate service record, or select the record and click the Details… button. The Service Detail form is displayed.
  4. Click the Entitlements button.
