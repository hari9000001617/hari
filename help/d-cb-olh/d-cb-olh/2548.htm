Customer Query Progress History Form

The Customer Query Progress History form is used to list progressions associated with the current query.

Enquiry made by a customer or contact that is recorded. The resolution of that enquiry generally results in information being passed back to that customer or contact. A query can range from an enquiry about available products by a prospective customer, to a complaint from a customer about charges on an invoice. If the query cannot be resolved immediately, it is progressed to a work group for resolution.

The query facility is also used to record internal enquiries. For example, the query facility can be used to enquire about a customer's credit rating by raising a query and forwarding it to an organisation's finance department.

Context: Convergent Billing, Customer Management

See also Progress, Progression, and Work Group.

To view details of a specific progression listed on the form, double-click the appropriate progression, or highlight the progression and click the Details… button. The Customer Query Progress form is displayed.

Associated Actions

Viewing a Progression

View a progression by:

  • Searching for the progressions associated with a query and then viewing the relevant progression.
  • Clicking the Query Progressions button on the Customer Query form and then viewing the relevant progression.

To search for and view a progression:

  1. Select the Customer Care Queries Manage Queries option. The Customer Query Search form is displayed.
  2. Enter the query number.
  3. Select Progress-All in the Search For field.
  4. Click the Search button. Any progressions associated with the specified query are retrieved and listed on the Customer Query Progress List form.
  5. To view a specific progression, double-click the progression or highlight the progression and click the Details... button. The Customer Query Progress Detail form is displayed with details of the selected progression.

To access and view a progression using the Customer Query form:

  1. Search for and display the query with the relevant progressions. The selected query is displayed on the Customer Query form.
  2. Click the Query Progressions… button. Any progressions associated with the specified query are listed on the Customer Query Progress List form.
  3. To view a specific progression, double-click the progression or highlight the progression, and click the Details... button. The Customer Query Progress Detail form is displayed with details of the selected progression.

Work on a Progression

To work on a progression:

  1. View the appropriate customer query.
  2. Click the Query Progressions… button. The Customer Query Progress History form is displayed with each of the progressions associated with the specified query.
  3. Select the last progression and click the Details... button. The Customer Query Progress Detail form is displayed.
  4. Click the Update button. The Customer Query Progression Detail form is displayed in Update mode.
  5. In the Action tab, record any actions taken to resolve the query.

    Note:

    Record the actions taken even if the query is not fully resolved

  6. In the Resolution tab, record the results of the actions taken to resolve the query. The record of actions taken is used as the basis for any further progressions or as the resolution text of the original query.
  7. Click the Save button on the toolbar.
  8. If the query has been resolved, resolve the progression.
  9. If the query cannot be resolved, forward the progression.

Forwarding a Progression

To forward a progression to another work force:
  1. Ensure that the appropriate progression is displayed on the Customer Query Progress form and that the details have been saved.
  2. Click the Progress button.
  3. When prompted, select a new work force and click Yes. As a result of this action, a new progression is created and the resolution text from the current progression is copied to the Progress tab of the new progression.

Need more information on progressions?

Resolving a Progression

Ensure that work on the progression is completed.

To resolve a progression (and therefore resolve the associated query):

  1. Ensure that the details on the Customer Query Progress Detail form have been saved.
  2. Click the Resolve button. A prompt is displayed for confirming that the query is to be closed.
  3. Click Yes to close the query. As a result of this action, the resolution text of the progression is copied to the Resolution tab of the associated query and the query status is set to Closed.
