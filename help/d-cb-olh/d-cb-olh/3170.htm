Invoice Format History Form

The Invoice Format History form is used to view a:

  • List of the physical records held in the Convergent Billing database for the specified invoice format.

    Specifies the format of generated invoices. The format specified determines which invoice template is used during the Invoice Generation Process (IGP). Can be specified on a per customer node basis.

    Context: Convergent Billing, Configuration Studio

    See also Invoice Generation Process (IGP) and Invoice Template.

    The list of records is displayed in chronological order and includes the current record, one or more historical records (if available), and one or more future records (if defined).

    Record having a start date in the future.

    Context: Convergent Billing, Customer Management

    See also Current Record and Historical Record.

    Record having an end date before today.

    Context: Convergent Billing

    See also Current Record and Future Record.

    Record having a start date before or equal to today, and an end date equal to or after today.

    Context: Convergent Billing, Customer Management, Process Engine

    See also Future Record and Historical Record.

  • Specific record and, if appropriate privileges are available, to update a record.

The form can be displayed in one of two form modes (View Only mode or Update mode), with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Form Mode

Allows new information to be recorded (entered) in Convergent Billing.

Allows information held in Convergent Billing to be viewed and updated (modified).

Associated Actions

Viewing an Invoice

Tip:

The method to use to search for and view the details of an invoice depends on whether the assigned invoice number is known.

If the invoice number is known:

  1. Select the Accounts Invoices option. The Invoice Search form is displayed.
  2. Ensure that the Search field is set to Invoice Number and enter the invoice number in the Search Criteria field.
  3. Click the Search button. If a valid invoice number is entered, the invoice is displayed.
  4. Click the Detail… button. The Invoice Detail form is displayed with details of the invoice.

If the invoice number is not known:

  1. View the customer’s account history.
  2. Double-click the appropriate invoice transaction in the list of transactions. The Invoice Detail form is displayed with details of the specified invoice.

Tip:

To locate the correct invoice transaction within the list of transactions, ask the enquirer for the date of the invoice; the date is usually printed prominently on the invoice in a predefined location.

Viewing an Invoice Image

Image Viewer Requirements

An image can be generated in a number of different formats; the appropriate image viewer must therefore be available. Consult the System Administrator about the relevant viewer used.

To view the image:

  1. View the appropriate invoice.
  2. Select the Images tab.
  3. If the image has been generated, click the Invoice Image... button. If not, click the Generate Image button.
  4. Depending on whether the invoice image is generated, Convergent Billing may ask whether the user wants to:
    • View the image

      Click the Yes button to display the image using the appropriate viewer, or the No button to regenerate the image.

    • Schedule a task to generate the image

      Click the OK button to schedule the appropriate task. A message indicating that the schedule was submitted to and reached the server is displayed. When the scheduled task is complete, the appropriate viewer is run automatically to display the generated image.

      To cancel the request to schedule a task to generate the image, click the Cancel button.

Viewing an Invoice Format

To view an invoice format:

  1. Select the Maintenance Printing Invoice Formats option. The Invoice Format Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of invoice formats matching the specified criteria is displayed.
  4. Select the appropriate invoice format and click the Detail… button, or double-click the appropriate invoice format. The Invoice Format Definition form is displayed in View Only mode.

Need more information on invoice formats?

Updating an Invoice Format

To update an invoice format:

  1. View the appropriate invoice format.
  2. Click the Update button on the toolbar. The Invoice Format Definition form is displayed in Update mode.
  3. Update the invoice format details as required.
  4. Click the Save button on the toolbar.

Need more information on invoice formats?

Defining an Invoice Format

To define an invoice format:

  1. Select the Maintenance Printing Invoice Formats option. The Invoice Format Search form is displayed.
  2. Click the New… button. The Invoice Format Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on invoice formats?

Deleting an Invoice Format

To delete an invoice format:

  1. View the appropriate invoice format.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the invoice format is to be deleted.
