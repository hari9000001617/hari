Online Help for Singleview Convergent Billing 9.1

User Preferences Form

The User Preferences form is used to:

  • View, define, and maintain user preferences.
  • Select options to help navigate through Convergent Billing, and to display the Summary of Changes column on history forms.

The form is divided into the following tabs:

Mandatory fields

The Mandatory Fields tab is used to:

  • Specify whether or not mandatory fields are emphasised, and provides examples of the mandatory field settings defined in this form.
  • Define settings for mandatory fields and their labels.

    For example, the colour and characteristics of the font used for mandatory field labels, colour of the text displayed in mandatory fields, and background colour of mandatory fields can be specified.

Form Navigation

The Form Navigation tab is used to:

  • Select options to view the Quick Window List and the location on the desktop for it to be displayed.

    Selecting the Show Quick Window List? option makes navigating through forms easier by being able to view a list of all opened forms grouped by their entity, order in which the forms have been opened, or by bookmark.

  • Display the option to enable bookmarking, which makes navigating through forms easier by being able to use shortcut keys to make the required form active.

    To bookmark a form, press Ctrl+Shift+Number, where Number is the number to select to identify the form. When the form is bookmarked, pressing Ctrl+Number makes the form active.

General

The General tab is used to select the option to display the Summary of Changes column on history forms.

The form can be displayed in View or Update mode, with the current mode indicated in the form’s title.
