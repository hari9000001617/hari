General Ledger Code Definition Form

The General Ledger Code Definition form is used to view, define, and update a General Ledger (GL) code.

ID that each account in a general ledger is identified by, and sometimes known as an account code. GL codes are used to map transactions in Singleview to the appropriate account in the external general ledger system.

Context: Convergent Billing

See also Adjustment Type and Payment Type.

The upper part of the form is used to specify name of the code, its description, and period (date range) for which it is valid.

Mechanism for recording period-specific entity data. A date range represents a period of time bounded by an effective start date and an effective end date. A date ranged entity is an entity with data that can change over time, and the data is recorded for the date range that the data is valid. The time span of the entity is defined by contiguous date ranges that specify historical, current, and future versions of the data.

Context: Convergent Billing, Customer Management, Process Engine

See also Effective Date.

The lower part lists the associated reporting entity, currency, parent GL code, hierarchy level, account class, and account category.

Category (or subclass) of one of the available account classes a Convergent Billing account belongs to. Although account categories are extensible, Convergent Billing does not validate account categories against account classes.

Associated Actions

Viewing a General Ledger Code

To view a General Ledger code (GL code) and its associated account class:

  1. Select the Maintenance Accounts GL Codes option. The General Ledger Code Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined general ledger codes matching the specified criteria is displayed.
  4. Select the appropriate general ledger code and click the Detail... button, or double-click the appropriate general ledger code. The General Ledger Code Definition form is displayed in View Only mode.

Need more information on GL Codes?

Defining a General Ledger Code

To define a GL code and associate it with an account class:

  1. Select the Maintenance Accounts GL Codes option. The General Ledger Code Search form is displayed.
  2. Click the New... button. The General Ledger Code Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.

    Note:

    Only account classes defined as reference items in the GL_CODE_ACCOUNT_CLASS reference type can be selected.

  4. Click the Save button on the toolbar.

Need more information on GL Codes?

Updating a General Ledger Code

To update a General Ledger code (GL code):

  1. View the appropriate GL code.
  2. Click the Update button on the toolbar. The General Ledger Code Definition form is displayed in Update mode.
  3. Update the GL code as required.
  4. Click the Save button on the toolbar.

Need more information on GL Codes?

Deleting a General Ledger Code

To delete a GL code:

  1. View the appropriate GL code.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the GL code is to be deleted.
