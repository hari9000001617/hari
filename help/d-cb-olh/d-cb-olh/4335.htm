How Do I - Product Configuration Tasks - Subtotals

Select the option most appropriate to the required task.

View a Subtotal

To view a subtotal definition:

  1. Select the Maintenance Tariffs Subtotals option. The Subtotal Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of subtotal definitions matching the specified criteria is displayed.
  4. Select the appropriate subtotal definition and click the Detail… button, or double-click the appropriate subtotal definition. The Subtotal Definition form is displayed in View Only mode.

Define a Subtotal

To define a subtotal:

  1. Select the Maintenance Tariffs Subtotals option. The Subtotal Search form is displayed.
  2. Click the New... button. The Subtotal Definition form is displayed in Insert Mode.
  3. Complete the appropriate data entry fields.
  4. Click the Save button on the toolbar.

Update a Subtotal

To update a subtotal:

  1. View the appropriate subtotal.
  2. Click the Update button on the toolbar. The Subtotal Definition form is displayed in Update mode.
  3. Update the subtotal details as required.
  4. Click the Save button on the toolbar.

Delete a Subtotal

Caution:

Before deleting a subtotal, ensure that the subtotal is not referenced in another subtotal or tariff.

To delete a subtotal:

  1. View the appropriate subtotal.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the subtotal is to be deleted.

Create a Progressive Subtotal

To create a progressive subtotal:

  1. Create the subtotal.
  2. On the Settings tab, select the Service context, and select the Use Subtotal before final value? check box.
  3. Select the Terms tab. Create a subtotal term expression that returns a constant value of 1. Select the Normalised Event evaluation context.
  4. Save the subtotal.
  5. Define a tariff configuration. On the Settings tab, select the Normalised Event context. Click the Eligibility… button on the Settings tab to enter an eligibility expression. The eligibility expression should return TRUE if the progressive subtotal is less than or equal to 3.
  6. Select the Charges tab of the Tariff Configuration form. In the Charge Expression field, enter an expression specifying that if the subtotal of normalised events is less than 3, no charges are applied to the subtotal. In the Default Charge Expression field, enter the default charge -Charge# (to cancel out the rating charge).
  7. Save the tariff configuration.

The value of the service subtotal can be accessed by the normalised event variables after each normalised event is processed.
