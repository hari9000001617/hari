Normalised Event Type Definition Form

The Normalised Event Type Definition form is used to view, update, and maintain one or more normalised event types.

The upper part of the form contains the normalised event type name and description.

The lower part of the form is used to select entity validation definitions to associate with the normalised event administration forms (Normalised Event Detail, Normalised Event Error, Normalised File Detail, and Rerate File forms), Usage Detail Search form, and Call Usage Display form.

The entity validation definitions associated with these forms allow definition of settings for the normalised event being configured. Entity validation definitions can also be selected to determine how events are displayed to users.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Form Mode

Associated Actions

Viewing a Normalised Event Type

To view a normalised event type:

  1. Select the Maintenance Server Normalised Event Types option. The Normalised Event Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the normalised event types matching the specified criteria is displayed.
  4. Select the appropriate normalised event type and click the Detail… button, or double-click the appropriate normalised event type. The Normalised Event Type form is displayed in View Only mode.

Creating a Normalised Event Type

To define a normalised event type:

  1. Select the Maintenance Server Normalised Event Types option. The Normalised Event Type Search form is displayed.
  2. Click the New... button. The Normalised Event Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on normalisation?

Updating a Normalised Event Type

To update a normalised event type:

  1. View the appropriate normalised event type.
  2. Click the Update button on the toolbar. Normalised Event Type form is displayed in Update mode.
  3. Update the normalised event type details as required.
  4. Click the Save button on the toolbar.

Need more information on normalisation?

Deleting a Normalised Event Type

To delete a normalised event type:

  1. View the appropriate normalised event type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the normalised event type is to be deleted.
