Function Role Search Form

The Function Role Search form is used to search for one or more function roles, using a specific field, criteria, and method.

One of the security features in Convergent Billing that, in combination with profiles, determines the access level a user has to functions. The function role determines which functions, or set of functions, a user can access. Function roles are assigned using a user profile.

Context: Convergent Billing, Process Engine, Configuration Studio

See also Access Level, Function, and User Profile.

The lower part of the form is used to list each of the function roles returned by the search that matches the specified criteria.

Multiple instances of the Function Role Definition form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Defining a Function Role

To define a function role:

  1. Select the Maintenance Security Function Role option. The Function Role Search form is displayed.
  2. Click the New… button. The Function Role Definition form is displayed in Insert mode.
  3. Complete the Name and Description fields in the top part of the form.
  4. Add function roles to the table in the lower part of the form as required. Each function role must be entered on a separate row of the table.
  5. The Invoked Function column displays the name of the remote TRE function to be invoked when a TRE client calls the function in the Called Function column. The invoked function must be defined as 'remote' and return the same data type as the function in the Called Function column.
  6. To add a row to the table, press the Down Arrow key.
  7. To insert a blank row before the current row, click the Insert button on the toolbar.
  8. Click the Save button on the toolbar.

    Note:

    If required, multiple functions can be added to the function role.

Updating a Function Role

To update a function role:

  1. View the appropriate function role.
  2. Click the Update button on the toolbar. The Function Role Definition form is displayed in Update mode.
  3. Add function roles to the table in the lower part of the form as required. Each function role must be entered on a separate row of the table.
  4. To add a row to the table, press the Down Arrow key.
  5. To insert a blank row before the current row, click the Insert button on the toolbar.
  6. Click the Save button on the toolbar.

    Note:

    If required, multiple functions can be added to the function role.

Deleting a Function Role

To delete a function role:

  1. View the appropriate function role.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the function role is to be deleted.

Viewing a Function Role

To view a function role:

  1. Select the Maintenance Security Function Role option. The Function Role Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined function roles matching the specified criteria is displayed.
  4. Select the appropriate function role and click the Detail… button, or double-click the appropriate function role. The Function Role Definition form is displayed in View Only mode.
