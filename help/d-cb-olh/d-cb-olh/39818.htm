This guide provides information about Singleview's security and auditing capabilities. The guide is separated into multiple parts describing concepts and configuring the elements.

Who Should Read this Guide?

This guide is aimed at personnel who need to understand the Singleview security and auditing model and configure Singleview security and auditing elements.

Is Prerequisite Knowledge Assumed?

This guide assumes the reader has:

  • Basic knowledge of security and auditing concepts, principles, and methods.

    Note:

    Security and auditing concepts and methods are included in this guide to only provide context and supporting information to explain the Singleview model and configuration. This guide is not a general purpose guide on security and auditing.

  • Familiarity with the content of the Architecture Overview for Singleview.
  • General Singleview knowledge.
  • Some experience in using UNIX.
  • Thorough knowledge of:
    • Apache HTTP server
  • Apache Tomcat.
