Invoice Type Definition Form

The Invoice Type Definition form is used to:

  • View, define, and maintain an invoice type, by entering general information about the invoice type including the name, validity dates, and a description.
  • Select entity validation that defines fields displayed on the General tab on the Invoice Detail form, and how the form is displayed to users and administrators.
  • Specify the function used to determine allocation of payments and adjustments to invoices of this type.

The Invoice Expressions… button is used to create and assign invoice assignment expressions.

The Consolidation Expression tab is used to specify if a consolidated invoice is to be generated for the current bill run.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing an Invoice Type

To view an invoice type:

  1. Select the Maintenance Printing Invoice Types option. The Invoice Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of invoice types matching the specified criteria is displayed.
  4. Select the appropriate invoice type and click the Detail… button, or double-click the appropriate invoice type. The Invoice Type Definition form is displayed in View Only mode.

Defining an Invoice Type

To define an invoice type:

  1. Select the Maintenance Printing Invoice Types option. The Invoice Type Search form is displayed.
  2. Click the New… button. The Invoice Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on invoice types?

Updating an Invoice Type

To update an invoice type:

  1. View the appropriate invoice type.
  2. Click the Update button on the toolbar. The Invoice Type Definition form is displayed in Update mode.
  3. Update the invoice type details as required.
  4. Click the Save button on the toolbar.

Need more information on invoice types?

Deleting an Invoice Type

To delete an invoice type:

  1. View the appropriate invoice type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the invoice type is to be deleted.

Note:

An invoice type referenced by an account type must exist from the current date-time until the end of time.
