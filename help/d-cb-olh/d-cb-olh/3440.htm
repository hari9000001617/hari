Payment Type Search Form

The Payment Type Search form is used to search for one or more payment types, using a specific field, criteria, and method.

Amount paid by a customer. The payment is recorded against a specific customer's account and optionally against one or more specified invoices.

Context: Convergent Billing, Customer Management

See also Payment Batch and Payment Type.

The lower part of the form is used to list payment types returned by the search that match the specified criteria.

Multiple instances of the Payment Type Definition form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Viewing a Payment Type

To view a payment type:

  1. Select the Maintenance Customer Care Payment Type option. The Payment Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of payment types matching the specified criteria is displayed.
  4. Select the appropriate payment type and click the Detail... button, or double-click the appropriate payment type. The Payment Type Definition form is displayed in View Only mode.

Defining a Payment Type

To define a payment type:

  1. Select the Maintenance Customer Care Payment Type option. The Payment Type Search form is displayed.
  2. Click the New... button. The Payment Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on payment types?

Updating a Payment Type

To update a payment type:

  1. View the appropriate payment type.
  2. Click the Update button on the toolbar. The Payment Type Definition form is displayed in Update mode.
  3. Update the payment type details as required.
  4. Click the Save button on the toolbar.

Need more information on payment types?

Deleting a Payment Type

To delete a payment type:

  1. View the appropriate payment type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the payment type is to be deleted.
