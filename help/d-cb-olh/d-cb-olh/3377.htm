Output Method Type Definition Form

The Output Method Type Definition form is used to view, define, and maintain an output method type.

The upper part of the form is used to specify the output method type name, description, and name of the view used in the invoice output query.

The lower part of the form consists of the following tabs for viewing more specific details:

Output Image

The Output Image tab is used to specify the format of the output image, and the location of the document image for this output method type.

Note:

The Type field cannot be updated for an output method type that is in use.

Update

The Update tab is used to specify the name of the database table to be updated, and the clauses for updating and identifying the records to update in the table.

The output method type:

  • Determines the source of the data that is to be printed.
  • Can Update a database table (to set a flag indicating an invoice was printed).

During processing, the General Output Process (GOP) performs placeholder substitution on some output method type fields, based on the query view for the output method.

The form can be displayed in one of three form modes, with the current mode being indicated in the form’s title.

Associated Actions

Viewing an Output Method Type

To view an output method type:

  1. Select the Maintenance Printing Output Method Types option. The Output Method Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of output method types matching the specified criteria is displayed.
  4. Select the appropriate output method type and click the Detail... button, or double-click the appropriate output method type. The Output Method Type Definition form is displayed in View Only mode.

Need more information on output method types?

Defining an Output Method Type

To define an output method type:

  1. Select the Maintenance Printing Output Method Types option. The Output Method Type Search form is displayed.
  2. Click the New button. The Output Method Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on output method types?

Updating an Output Method Type

To update an output method type:

  1. View the appropriate output method type.
  2. Click the Update button on the toolbar. The Output Method Definition form is displayed in Update mode.
  3. Update the output method details as required.

    Note:

    The Type field in the Output Image tab, cannot be updated for an output method type that is in use.

  4. Click the Save button on the toolbar.

Need more information on output method types?

Deleting an Output Method Type

To delete an output method type:

  1. View the appropriate output method type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the output method type is to be deleted.

Placeholder Substitution - Output Method Type Definition Form

During processing, the General Output Process (GOP) performs substitutions on the following output method type fields, based on the query view for the output method:

  • Where Clause field on the Output Image tab.
  • Set Clause field on the Update tab.
  • Where Clause field on the Update tab.

Placeholders must be in the form ':<COLUMN_NAME>' and are replaced with the value of the <COLUMN_NAME> column from the current query view record.

For example, the Where Clause field on the Output Image tab could contain 'INVOICE_ID = :INVOICE_ID and SEQNR = :INVOICE_SEQNR', where INVOICE_ID and INVOICE_SEQNR are fields from the INV_DUNNING_V view.

Note:

Only string and numeric values can be substituted; the behaviour is undefined for raw or date values (except for the placeholder :EFFECTIVE_DATE).

Additionally, if the GOP is running in:

  • Billing output mode, the placeholders :BILL_RUN_ID, :BILL_RUN_OPERATION_ID, and :EFFECTIVE_DATE are replaced with the relevant values. EFFECTIVE_DATE has the Oracle date format, 'DD-MM-YYYY HH24:MI:SS'.
  • Reporting output mode, the placeholder :TASK_QUEUE_ID is replaced with the relevant value.

Note:

Substituted strings are single-quoted, and nested single quotes are not escaped and cause problems.
