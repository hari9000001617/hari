Licence Enforcement

The Authentication server checks the Convergent Billing licence at three levels:

  • When starting the rating engine or the billing engine.
  • When users log in.
  • When users attempt to save new entities subject to licensing.

If the licensed levels are inadequate for the volume of work, a new licence must be negotiated with CSG International.

Functions Defined on the biLicence Service

The Authentication server and the client workstations enforce the licence conditions by making calls on functions defined in the biLicence Tuxedo service. Functions defined on the biLicence service are:

  • biLicenceReload&

    Reloads the licence restrictions from the database.

  • biLicenceExpiry~

    Returns the expiry date of the licence.

  • biLicenceStatus&

    Returns the current status of the entity.

Example 1 describes how to determine the expiry date of the licence.
