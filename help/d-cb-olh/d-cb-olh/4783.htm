Type Options Form

The Type Options form is used to customise the contents displayed for the selected configuration object, and to filter the information displayed on the Convergent Billing Configuration Explorer form.

Associated Actions

Viewing a Diagram of the Configuration

To display a diagram of the configuration for the object selected in the left-hand panel:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer Diagram View option.
  3. Select an object. The diagram is displayed in the right-hand panel of the form.

Tip:

To view the configuration object's:

  • Properties, select the Explorer Properties View option.
  • Contents, select the Explorer Grid View option.

Need more information on Configuration Explorer?

Viewing the Properties of a Configuration Object

To view the properties of a configuration object:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer Properties View option.
  3. Click the relevant object.
  4. Click in the second level of the hierarchy, then select the relevant object from the third level to display that object's properties in the right-hand panel.

The properties can also be viewed by right-clicking on the relevant object and selecting Properties.

Tip:

To view a configuration object's:

  • Diagram of the properties, select the Explorer Diagram View option.
  • Contents, select the Explorer Grid View option.

Need more information on Configuration Explorer?

Viewing the Items Referenced by a Configuration Object

To display the items referenced by a configuration object:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer Grid View option.
  3. Right-click on the configuration object and select References. The References form with the items referenced by the selected object is displayed in the right-hand panel of the form in grid view.

Tip:

To view the configuration object's references in diagram view, select the Explorer Diagram View option.

Need more information on Configuration Explorer?

Viewing the Contents of a Configuration Object

To view the contents of a configuration object:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select Explorer Grid View option. The contents of the object are displayed in the right-hand panel.

When viewing a grid, objects can be expanded or collapsed by double-clicking the object in the right-hand panel.

Tip:

To view the configuration object's:

  • Diagram, select the Explorer Diagram View option. References to the object are displayed to the left of the object.
  • Properties in a grid format, select the Explorer Properties View option.

Need more information on Configuration Explorer?

Modifying the Presentation of Configuration Information

To change how configuration information is displayed:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer option, then the appropriate method of displaying the configuration information (Grid View, Diagram View, or Properties View).

Note:

Selecting Properties View displays the properties of the selected configuration object in a grid format. When viewing a grid, each object can be expanded or collapsed by double-clicking the object in the right-hand panel.

Need more information on Configuration Explorer?

Modifying the Configuration Details

To modify configuration details:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer option, then the appropriate method of displaying the configuration information (Grid View, Diagram View, or Properties View).
  3. Navigate to the relevant entity by expanding or collapsing the view.
  4. Highlight the entity, click the right mouse button, and select the Detail… option.
  5. Use the Insert, Update, or Delete buttons on the toolbar to modify the configuration.

Note:

The buttons are only enabled if appropriate security settings are available.

Need more information on Configuration Explorer?

Modifying the Explorer Options

To modify the Configuration Explorer options:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer Options... option. The Explorer Options form is displayed.
  3. Modify the relevant options and click OK.

Need more information on Configuration Explorer?

Viewing the Explorer Legend

To view the explorer legend:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select the Explorer Legend... option. The Explorer Legend form is displayed.
  3. To view a specific entity, folder type, or icon, right-click on the relevant object and select the appropriate option.

Need more information on Configuration Explorer?

Printing Configuration Information

To print configuration information:

  1. Select the Maintenance Configuration Explorer option. The Convergent Billing Configuration form is displayed, and the Explorer option is added to the menu bar.
  2. Select Explorer Diagram View option.
  3. Select the relevant diagram.
  4. Right-click on the diagram and select Print Diagram. The Print Selection form is displayed.
  5. Enter the required printer selections and click OK.
