Template Search Form

The Template Search form is used to search for one or more template definitions, using a specific field, criteria, method, and date.

Entity containing document processing instructions that are used by the IGP when creating a document image.

The Invoice Design Tool is used to manage template design.

Context: Convergent Billing, Singleview

See also Document, IGP, and Invoice Design Tool.

The lower part of the form is used to list each of the template definitions returned by the search that matches the specified criteria.

Multiple instances of the Template Definition form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Viewing a Template

To view a template:

  1. Select the Maintenance Printing Templates option. The Template Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of template definitions matching the specified criteria is displayed.
  4. Select the appropriate template name and click the Detail… button, or double-click the appropriate template name. The Template Definition form is displayed in View Only mode.

Need more information on templates?

Defining a Template

To define a template:

  1. Select the Maintenance Printing Templates option. The Template Search form is displayed.
  2. Click the New… button. The Template Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on templates?

Updating a Template

To update a template:

  1. View the appropriate template.
  2. Click the Update button on the toolbar. The Template Definition form is displayed in Update mode.
  3. Update the template details as required
  4. Click the Save button on the toolbar.

Need more information on templates?

Deleting a Template

To delete a template:

  1. View the appropriate template.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the template is to be deleted.
