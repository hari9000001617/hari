Equipment Search Form

The Equipment Search form is used to search for one or more equipment records.

Physical components required to provide a service; for example, modem, leased line, or mobile phone. Equipment information is stored for information purposes only, but can also be used to allocate service numbers.

Context: Convergent Billing, Customer Management

See also Service.

Search criteria can be specified on the Details, Address, or Transit tabs.

The Transit tab displays details of the equipment while in transit. The tab is only displayed if the equipment type is configured to enable this tab.

The Address tab is used to enter the equipment's address (either in full or in part) so that it can be used as one of the search criteria.

The fields in which to enter the equipment's address are displayed only when an address format is selected (in the Format field). The address format determines the layout (number and type of fields) of the address.

The Details tab is used to refine a search by date, serial number, network address, equipment type, and equipment status. The search can also be refined by specifying the minimum number of items to return by the search, and a customer name, for which the equipment is reserved, can be nominated as one of the search criteria.

By default, all records active on the date the search is performed are included in the search. The date field can be used to search for historical records active at a specific point in time.

Selecting the:

  • Search for equipment option enables the Search… button on the form, for searching for equipment matching the specified criteria
  • Create new equipment option enables the New… button, for creating equipment items.

The display format for the retrieved records can be determined by selecting or clearing the Always Display List? and the Reuse Detail Form? check boxes.

If multiple instances of the detail form are required for display (for example, to access details for several payments) the Reuse detail form? option should not be selected.

Caution:

If selected, the Reuse detail form? option overrides the display of information in a new form; the details are displayed in the existing form.

If a single equipment record is returned from a search, the details of that record are displayed on the Equipment Detail form. If more than one equipment record is returned, the records are listed on the Equipment List form. The required record can then be selected and its details displayed on the Equipment Detail form.

If selected, the Always Display List? option overrides the automatic display of the Equipment Detail form when only a single record is retrieved; in such cases the Equipment List form is displayed.

Associated Actions

Viewing Equipment Details

To view equipment details:

  1. Select the Customer Care Equipment Manage Equipment option. The Equipment Search form is displayed.

    Note:

    The Transit tab is only displayed on the Equipment Search form if the equipment type is configured to enable the tab.

  2. Enter or select the appropriate search criteria.
  3. Click the Search… button. A list of equipment records matching the specified criteria is displayed.
  4. Select the appropriate equipment record and click the Details... button. The Equipment Detail form is displayed in View Only mode.

Recording Equipment Details

Convergent Billing provides two methods in which equipment details can be recorded in the equipment inventory:

  • Individually
  • In bulk.

To record (enter) equipment details for individual items of equipment:

  1. Select either of the following options:
    • Customer Care Equipment New Equipment Items option.
    • Customer Care Equipment Manage Equipment option.

    The Equipment Search form is displayed.

  2. Select the Create new equipment option.
  3. Select the equipment type and enter the equipment serial number.
  4. Click the New… button to insert a new equipment record. The Equipment Detail form is displayed in Insert mode.
  5. Complete the appropriate data entry fields.
  6. Click the Save button.

To record (enter) equipment items in bulk:

  1. Create the appropriate equipment record.
  2. Click the Bulk Insert… button. The Equipment Bulk Insert form is displayed in Insert mode.
  3. Enter the serial number and/or network address for a specified item of equipment.
  4. To add a new item, press the Down Arrow key to create an empty line and repeat step 3.
  5. Repeat step 4 for each item of equipment to be inserted.
  6. Click the Save button on the toolbar.
  7. Close the Equipment Bulk Insert form to return control to the Equipment Detail form.

Updating Equipment Details

Convergent Billing provides facilities to update:

  • Individual equipment records
  • A range of two or more equipment records of the same equipment type.

To update the details of an individual equipment record:

  1. View the appropriate equipment record.
  2. Click the Update button on the toolbar. The Equipment Detail form is displayed in Update mode.
  3. Update the equipment details as required.
  4. Click the Save button on the toolbar.

To update a range of equipment records of the same equipment type:

  1. Search the appropriate equipment records and list them in the Equipment List form.
  2. Select each equipment record to be included in the range to be updated.
  3. For a contiguous number of equipment records, select the first record in the required range and do either of the following:
    • Hold down the Shift key, and select the last record in the required range.
    • Specify the number of records to be updated in the Select how many items? field, and click the Select button.
  4. To select a number of individual equipment records, hold down the Ctrl key and select each record.

    Note:

    When more than one record is selected, the Detail… button on the Equipment List form is replaced with the Bulk Update button.

  5. Click the Bulk Update button. The Equipment Detail form is displayed in Update mode with details of the first equipment record in the selected range.
  6. Make any required changes to the displayed record. Any changes made to this record are also made to each of the other records in the selected range.
  7. Click the Save button on the toolbar.

Deleting Equipment Details

To delete equipment details:

  1. View the appropriate equipment record.
  2. Click the Delete button on the toolbar.
  3. When prompted, specify the date from which to delete the equipment. Depending on security settings, the date consists of a single component (date), or two components (date and time).
  4. When prompted, click the Yes button to confirm that the equipment details are to be deleted.
