Entity Saving Licence Checks

When a user attempts to save an entity subject to licensing (that is, service instances, customer nodes, or customers), Convergent Billing checks the number of active entities against the licence. The options are similar to those for user login. That is, when the entity count is:

  • Below the warning level, the entity is saved.
  • Between the warning level and the licensed number, a warning message is displayed and the entity is saved.
  • Between the licensed number and the maximum level, a more strongly worded error message is displayed and the entity is saved.
  • Greater than the maximum level, the entity cannot be saved.

Note:

All saved entities are still available to this or any other user, even if new entities cannot be saved because the maximum level is reached. The saving of entities is permitted again when the entity count falls below the licensed maximum. Reducing the entity count might be achieved by archiving and purging redundant entities or by negotiating variations in the licence with CSG International.
