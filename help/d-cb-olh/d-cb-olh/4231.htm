How Do I - Customer Care Configuration Tasks - Context-sensitive Help

Select the option most appropriate to the required task.

Define a Help Entry Definition

To define a help entry definition:

  1. Select the Maintenance Data Definition Help Entries option. The Help Entry Search form is displayed.
  2. Click the New… button. The Help Entry Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on context-sensitive help?

Delete a Help Entry Definition

Caution:

Deleting a help entry definition deletes the context-sensitive help for a Convergent Billing component.

To delete a help entry definition:

  1. View the appropriate help entry definition.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the help entry is to be deleted.

Dump Form Components

To dump the components of a form:

  1. Display the form containing the components to be dumped.
  2. Select the Maintenance Data Definition Form Field Help Dump.
  3. The Form Fields form is displayed in View Only mode.
  4. Click the Update button on the toolbar.
  5. Click the Copy button. The form components are copied to the Form Definition form.
  6. Click the Save button on the toolbar.
  7. Select the Maintenance Data Definition Form Field Help option. The Form Field Definition Search form is displayed.
  8. Enter or select the appropriate search criteria and search method.
  9. Click the Search button. A list of the defined forms matching the specified criteria is displayed.
  10. Select the appropriate form and click the Detail… button, or double-click the appropriate form. The Form Definition form is displayed and lists all its components in the lower part of the form.

View a Form Definition Form

To view a form help definition:

  1. Select the Maintenance Data Definition Form Field Help option. The Form Field Definition Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined forms matching the specified criteria is displayed.
  4. Select the appropriate form and click the Detail… button, or double-click the appropriate form. The Form Definition form is displayed in View Only mode.

  3. Click the Search button. A list of the defined help entries matching the specified criteria is displayed.

Assign a Help Entry

To assign a help entry:

  1. Do either of the following:
    • View the Help Entry Definition form for the help entry to be assigned
    • View the Help Entry Definition Search form and select the help entry definition to be assigned.
  2. View the Form Definition form. Click the Update button on the toolbar.
  3. Select the appropriate component's Help Entry field and click the Assign button. The help entry definition selected in step 1 is assigned to the component and displayed in the Help Entry field.

Duplicate a Help Entry Definition

To duplicate a help entry definition:

  1. View the Help Entry Search form.
  2. Select the help entry to be duplicated.
  3. Display the Help Entry Definition form in Insert mode by clicking the New… button.
  4. Click the Duplicate button. A message is displayed prompting confirmation that the specified help entry is to be duplicated.
  5. Click the Yes button. Details of the specified help entry are copied to create a new help entry in the Help Entry Definition form.
  6. Complete the Help Entry Name and Title fields, ensuring that the same naming conventions as the existing help entries are used.
  7. Click the Save button on the toolbar.

Need more information on context-sensitive help?

Modify a Help Entry

To modify a help entry:

  1. View the appropriate form definition form.
  2. Select the appropriate component and click the Details… button. The Help Entry Definition form is displayed in View Only mode.
  3. Click the Update button on the toolbar. The Help Entry Definition form is displayed in Update mode.
  4. Update the form definition details as required.
  5. If changed the name of the help entry is changed, ensure that the appropriate component is still highlighted on the Form Definition form and click the Assign button (this action assigns the help entry to the component).
  6. Click the Save button on the toolbar.
