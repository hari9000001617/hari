Online Help for Singleview Convergent Billing 9.1

How Do I - Customer Care Tasks - Invoices and Statements

Select the option most appropriate to the required task.

Select an Invoice Cycle

To select an invoice cycle or change an invoice format selection:

  1. View the appropriate Customer Detail form.
  2. Select the Invoicing tab. Click the Update button on toolbar.
  3. Select the appropriate invoice cycle in the Invoice Cycle field.
  4. Click the Save button on the toolbar.

Need more information on invoices?

Select an Invoice Report Level

To set a customer's invoice report level:

  1. View the appropriate Customer Detail form.
  2. Select the Invoicing tab. Click the Update button on the toolbar.
  3. Select the appropriate value in the Report Level field.

    Note:

    If the customer is a root customer, Report Level is automatically set to Invoice but can be changed to Transferred Statement.

  4. Select a value in the Transferred Account Number field, if required.

    Note:

    The Transferred Account Number field is:

    • Available only if the customer is a root customer.
    • Mandatory if Report Level is set to Transferred Statement.
  5. Click the Save button on the toolbar.
  6. Ensure that the appropriate invoice cycle is set.

Select an Invoice Format

To select an invoice format or change an invoice format selection:

  1. View the appropriate Customer Detail form.
  2. Select the Invoicing tab.
  3. Click the Invoice Formats… button. The Invoice Format form is displayed.
  4. Click the Update button on the toolbar. Select the required invoice format from the Invoice Format drop-down list.
  5. Click the Save button on the toolbar.

Note:

After an invoice is generated for a customer in the selected invoice format, it is not possible to change the format or the order of the invoice formats for that customer.

Search for an Invoice

Tip:

The method to use to search for and view the details of an invoice depends on whether the assigned invoice number is known.

If the invoice number is known:

  1. Select the Accounts Invoices option. The Invoice Search form is displayed.
  2. Ensure that the Search field is set to Invoice Number and enter the invoice number in the Search Criteria field.
  3. Click the Search button. If a valid invoice number is entered, the invoice is displayed.
  4. Click the Detail… button. The Invoice Detail form is displayed with details of the invoice.

If the invoice number is not known:

  1. View the customer’s account history.
  2. Double-click the appropriate invoice transaction in the list of transactions. The Invoice Detail form is displayed with details of the specified invoice.

Tip:

To locate the correct invoice transaction within the list of transactions, ask the enquirer for the date of the invoice; the date is usually printed prominently on the invoice in a predefined location.

Generate an Invoice

Invoices are raised against the customer's primary account and generated if the report level of the customer node is set to 'Invoice'. The customer's invoice format must be set and the appropriate invoice cycle selected.

To generate an invoice, perform a bill run, ensuring that the 'To Operation' of the bill run includes invoice operations.

Generate an Interim Invoice

Interim invoices include all rental events whose tariffs have been marked as 'interim'. To generate an interim invoice, perform the bill run schedule type Interim Bill Run.

View Invoice Details

To view the details of a generated invoice:

  1. View the appropriate invoice.
  2. Click the Invoice Details... button. The Invoice Item Details form is displayed.
  3. Enter the appropriate search criteria in the Invoice Hierarchy Search Criteria group and click the Search button. A list of entities matching the specified criteria is displayed in the Invoice Hierarchy section.
  4. Select the appropriate entity. The invoice charges for the entity are displayed in the Item Details grid.

View an Invoice Image

Image Viewer Requirements

An image can be generated in a number of different formats; the appropriate image viewer must therefore be available. Consult the System Administrator about the relevant viewer used.

To view the image:

  1. View the appropriate invoice.
  2. Select the Images tab.
  3. If the image has been generated, click the Invoice Image... button. If not, click the Generate Image button.
  4. Depending on whether the invoice image is generated, Convergent Billing may ask whether the user wants to:
    • View the image

      Click the Yes button to display the image using the appropriate viewer, or the No button to regenerate the image.

    • Schedule a task to generate the image

      Click the OK button to schedule the appropriate task. A message indicating that the schedule was submitted to and reached the server is displayed. When the scheduled task is complete, the appropriate viewer is run automatically to display the generated image.

      To cancel the request to schedule a task to generate the image, click the Cancel button.

Reprint an invoice

To reprint an invoice:

  1. View the appropriate invoice.
  2. Select the Images tab.
  3. Click the Re-print Image button. The Reprinting Invoice Image form is displayed.
  4. To print the invoice immediately, select the Direct option in the Print Type group and select the appropriate printer from the Printer field. Click OK.
  5. To mark the invoice to print at predetermined scheduled intervals, select the Scheduled option in the Print Type group and click OK.

Search for a Statement

Two ways to search for a statement are to:

  • Perform a general statement search.
  • View the statement using the customer's account history.

To perform a general statement search:

  1. Select the Accounts Statements option. The Statement Search form is displayed.
  2. Use any one of the search criteria to restrict the number of records retrieved by specifying a wildcard search.
  3. Click the Search button. A list of the records that match the selection criteria is displayed on the lower part of the form. The list includes records for statement nodes and non-reporting nodes.

    The list of records retrieved when a non-specific search is performed includes a record for each statement node (a customer node that has a report level of Statement) and each non-reporting node (a customer node that has a report level of None) that match the selection criteria.

    For statement nodes, the amount displayed in the Statement Amount column is the 'rolled-up' amount displayed on the customer's statement. The 'rolled-up' amount includes the charges incurred at the statement node and the charges for any non-reporting nodes that are children of the specified statement node. For non-reporting nodes, the amount displayed in the Statement Amount column indicates only the charges incurred by that node.

  4. Double-click the required entry in the list or select the entry and click the Detail... button. The Statement Detail form is displayed.

To view a statement using the customer's account history:

  1. View the customer’s account history.
  2. Double-click the appropriate statement transaction in the list of transactions. The Statement Detail form is displayed with details of the specified statement.

Tip:

To locate the correct statement within the list of transactions, ask the enquirer for the date of the statement; the date is usually printed prominently on the statement in a predefined location.
