Test an Expression

To test an expression:

  1. Select the Maintenance Data Definition Expression Test option.
  2. Maximise the form to expand the Results field to its maximum size.
  3. Enter the expression to be tested in the Expression field.

    Tip:

    Skeleton code containing a function call to a defined function with appropriate variables can be inserted in the upper part of the form. To insert the skeleton code, enter a function name in the Function Name field and click the Generate Function Call button. The skeleton code can be modified, as required, before the expression is evaluated.

  4. Click the Evaluate button. The results of the expression are displayed in the Results field.

Note:

Expression blocks can be evaluated by entering them as described. The braces are required around the expression block, and each expression in the block must end with a semicolon.

Functions can be called within a test expression. Direct variables can be used in test expressions. Other variables used in expressions are empty unless declared as local variables and assigned a value in the expression block.
