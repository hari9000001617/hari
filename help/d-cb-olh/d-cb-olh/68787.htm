How Do I - Security Tasks - Tenants

Select the option most appropriate to the required task.

View a Tenant

To view a tenant:

  1. Select the Maintenance Security Tenants option. The Tenant Definition Search form is displayed.
  2. Enter or select the relevant search criteria and search method.
  3. Click the Search button. A list of tenants matching the specified criteria is displayed.
  4. Select the relevant tenant and click the Detail… button, or double-click the relevant tenant name. The Tenant Definition form is displayed in View Only mode.

Define a Tenant

To define a tenant:

  1. Select the Maintenance Security Tenants option. The Tenant Definition Search form is displayed.
  2. Click the New… button. The Tenant Definition form is displayed in Insert mode.
  3. Complete the relevant fields on the form.
  4. Click the Save button on the toolbar.

Update a Tenant

To update a tenant:

  1. View the relevant Tenant Definition form.
  2. Click the Update button on the toolbar. The form is displayed in Update mode.
  3. Update tenant details as required.
  4. Click the Save button on the toolbar.

Delete a Tenant

To delete a tenant:

  1. View the relevant Tenant Definition form.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the tenant is to be deleted.
