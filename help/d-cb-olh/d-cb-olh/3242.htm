New Rerated File Detail Form

The New Rerate File Detail form is used to:

  • Create and assign a rerated file to this normalised event error (using the Create button).

    Type of event output when either of the following conditions occurs:

    • If an Event Normalisation (ENM) process successfully normalises an alpha event, but does not evaluate all normalised event file type expressions for that event, the event is output as a normalised error event. Normalised error events are sent only to an Event Rating Output (ERO) process for output; they are not sent to any Event Rating (ERT) process, and so are not rated.
    • If a normalised event is sent to one or more ERT processes, but no charges are generated for the event, the ERO process outputs the event as a normalised error event.

    Context: Convergent Billing

    See also Alpha Event, Event Normalisation (ENM) Process, Event Rating Output (ERO) Process, and Normalisation.

  • Specify the file name and the event source of the rerated file.

    Note:

    The file name for the rerated file cannot be the same as an existing normalised event file.

  • Cancel the rerated file (using the Cancel button).

Associated Actions

Viewing a Normalised File

To view a normalised file:

  1. Select the Server Tasks Events View Normalised Files and Events option. The Normalised File Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the normalised files matching the specified criteria is displayed.
  4. Select the appropriate normalised file and click the Detail… button, or double-click the appropriate normalised file. The Normalised File Definition form is displayed in View Only mode.

Viewing a Rerated file

To view a rerated file:

  1. Select the Server Tasks Events View Rerated Files and Events option. The Rerated File Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of rerated files matching the specified criteria is displayed.
  4. Select the appropriate rerated file and click the Detail… button, or double-click the appropriate rerated file. The Rerated File Detail form is displayed in View Only mode.

Need more information on normalisation?

Viewing a Normalised Event

To view a normalised event:

  1. View the appropriate normalised file.
  2. Select the Events tab to display any events returned for the file.
  3. To display details of the event, highlight the event in the grid and click the Details… button. The Normalised Event Detail form is displayed in View only mode.

Need more information on normalisation?

Viewing a Normalised Event Error

To view a normalised event error:

  1. Select the Server Tasks Events View/Assign Error Events option. The Normalised Event Error Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the normalised error events matching the specified criteria is displayed.
  4. Select the appropriate normalised error event and click the Detail… button, or double-click the appropriate normalised error event.
  5. The Normalised Event Error Detail form is displayed in View Only mode.

Need more information on normalisation?

Viewing a Normalised Event Type

To view a normalised event type:

  1. Select the Maintenance Server Normalised Event Types option. The Normalised Event Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the normalised event types matching the specified criteria is displayed.
  4. Select the appropriate normalised event type and click the Detail… button, or double-click the appropriate normalised event type. The Normalised Event Type form is displayed in View Only mode.

Need more information on normalisation?

Creating a Normalised Event Type

To define a normalised event type:

  1. Select the Maintenance Server Normalised Event Types option. The Normalised Event Type Search form is displayed.
  2. Click the New... button. The Normalised Event Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on normalisation?

Updating a Normalised Event Type

To update a normalised event type:

  1. View the appropriate normalised event type.
  2. Click the Update button on the toolbar. Normalised Event Type form is displayed in Update mode.
  3. Update the normalised event type details as required.
  4. Click the Save button on the toolbar.

Need more information on normalisation?

Deleting a Normalised Event Type

To delete a normalised event type:

  1. View the appropriate normalised event type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the normalised event type is to be deleted.
