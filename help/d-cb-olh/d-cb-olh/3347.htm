Output Device Definition Form

The Output Device Definition form is used to view, define, or update an output device.

The upper part of the form displays or allows entry of the name and the description of the output device.

The remainder of the form is used to specify the type of output image used, and how print jobs are executed.

During processing, the GOP performs placeholder substitution on the Run Command field, based on the query view for the output method.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing an Output Device

To view an output device definition:

  1. Select the Maintenance Printing Output Devices option. The Output Device Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of output devices matching the specified criteria is displayed.
  4. Select the appropriate output device and click the Detail… button, or double-click the appropriate output device. The Output Device Definition form is displayed in View Only mode.

Need more information on output devices?

Defining an Output Device

To define an output device definition:

  1. Select the Maintenance Printing Output Devices option. The Output Device Search form is displayed.
  2. Click the New… button. The Output Device Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields.
  4. Click the Save button on the toolbar.

Need more information on output devices?

Updating an Output Device

To update an output device definition:

  1. View the appropriate output device definition.
  2. Click the Update button on the toolbar. The Output Device Definition form is displayed in Update mode.
  3. Update the output device details as required.
  4. Click the Save button on the toolbar.

Need more information on output devices?

Deleting an Output Device

To delete an output device definition:

  1. View the appropriate output device definition.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the output device is to be deleted.

Placeholder Substitution - Output Device Definition Form

During processing, the General Output Process (GOP) performs substitutions on the Run Command field from the OUTPUT_DEVICE table when the Concatenated Images? check box is not selected. Substitutions are not made on the Start Command or Stop Command fields. Placeholders must be in the form ':<COLUMN_NAME>' and are replaced with the value of the <COLUMN_NAME> column from the OUTPUT_DEVICE table record.

Note:

Only string and numeric values can be substituted; the behaviour is undefined for raw or date values (except for the placeholder :EFFECTIVE_DATE).

Additional substitutions are made on the Run Command, Start Command, and Stop Command fields. If the GOP is running in:

  • Billing output mode, the placeholders :BILL_RUN_ID, :BILL_RUN_OPERATION_ID, and :EFFECTIVE_DATE are replaced with the relevant values. EFFECTIVE_DATE has the Oracle date format, 'DD-MM-YYYY HH24:MI:SS'.
  • Reporting output mode, the placeholder :TASK_QUEUE_ID is replaced with the relevant value.

Note:

Substituted strings are single-quoted, and nested single quotes are not escaped, and cause problems.
