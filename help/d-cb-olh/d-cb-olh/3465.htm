Person Type Definition Form

The Person Type Definition form is used to view, define, update, and delete a person type.

Person, position, or organisation recorded in Singleview that may need to be contacted. A contact can have multiple accounts, addresses, or other details.

Context: Convergent Billing, Customer Management, Process Engine

See also Customer, Customer Contact, Primary Contact, and Secondary Contact.

The upper part of the form displays or allows entry of general information about the person type (for example, name and description).

The remainder of the form is used to:

  • Specify the form displayed when entering, viewing, or updating person detail.
  • Specify the default address format.
  • Select any entity validation definitions to customise the Person Detail and Person Search forms.
  • Nominate the icon used to represent the person type.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Form Mode

Description

View Only mode

Allows information held in Convergent Billing to be viewed, but not updated (modified).

Insert mode

Allows new information to be recorded (entered) in Convergent Billing.

Update mode

Allows information held in Convergent Billing to be viewed and updated (modified).

Associated Actions

Viewing a Person Type

To view a person type:

  1. Select the Maintenance Customer Care Person Types option. The Person Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the person types matching the specified criteria is displayed.
  4. Select the appropriate person type and click the Detail... button, or double-click the appropriate person type. The Person Type Definition form is displayed in View Only mode.

Defining a Person Type

To define a person type:

  1. Select the Maintenance Customer Care Person Types option. The Person Search form is displayed.
  2. Click the New... button. The Person Type Definition form is displayed in Insert mode.
  3. Complete the appropriate data entry fields on the form.
  4. Click the Save button on the toolbar.

Need more information on person types?

Updating a Person Type

To update a person type:

  1. View the appropriate person type.
  2. Click the Update button on the toolbar. The Person Type Definition form is displayed in Update mode.
  3. Update the person type details as required.
  4. Click the Save button on the toolbar.

Need more information on person types?

Deleting a Person Type

To delete a person type:

  1. View the appropriate person type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the person type is to be deleted.
