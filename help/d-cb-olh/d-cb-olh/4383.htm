How Do I - Operations - Reports

Select the option most appropriate to the required task.

Schedule a Periodic Report

To schedule a periodic report:

  1. Select the Server Tasks Schedules option. The Schedule Search form is displayed.
  2. Select the Create a new schedule option and click the New… button. The Schedule Detail form is displayed in Insert mode.
  3. Select the report name from the drop-down list in the Schedule Type field. All report schedules are prefixed with 'Report'.
  4. Complete the remaining fields as required. If there are any problems in completing a field, position the cursor in the field and press F1 to obtain help on that field.
  5. Complete the fields in the Task Generation, Parameters, and Prerequisites tabs.
  6. Click the Save button on the toolbar. The Submit Schedule form is displayed.
  7. Click the Submit button to submit the schedule; no options on the Submit Schedule form need to be selected.

The schedule creates one or more tasks run for generating the report at the dates and times specified. If an email address is specified, an email message is received when each task is completed.

Receiving an email does not indicate whether the report was created successfully. The task status must be examined to check this. For more information, refer to Examine the Status of a Task.

Schedule an Immediate Report

To schedule an immediate, one-off report:

  1. Select the Server Tasks One-Off Task option.
  2. Choose the report name from the drop-down list in the Schedule Type field. When this field loses input focus (when another field is selected), the lower part of the form is populated with fields for any parameters the report requires.
  3. Enter a value in each parameter field.
  4. Click the Save button on the toolbar.

View and Print a Report

The default format for reports is Microsoft Excel XML; however, the format can be altered by using alternate stylesheets. The client workstation used to view reports must have the appropriate software to view the report in its output format.

To view a generated report:

  1. Display the Task Details form.
  2. Click the Report button to start the report viewing software (if it is not running), and display the report.
  3. View, edit, or print the report as required.
