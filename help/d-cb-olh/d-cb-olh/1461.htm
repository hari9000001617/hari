Online Help for Singleview Convergent Billing 9.1

Using the Help

Access help using one of the following methods.

 

Method

Description

Move the mouse pointer over a field.

Hint text. Field's purpose (data entry, display only, button, check box, or option).

Tip:

To enable or disable hint text, select or deselect the Options Show Hints option.

Click inside or tab to a field, then press the F1 key.

Field help. Provides information about the field.

If field help is not defined, form help is displayed.

Make the form active, then click the Help button.

Form help. Provides information about the form.

Select Help How Do I...?.

Task-based help. Provides step-by-step instructions to perform a task.

Select Help Reference Information.

Reference information. Provides information about elements of Convergent Billing to assist new users.

For example, reference information describes queries, which can be used to record customer enquiries.

Select Help Contents.

Online help. Consists of the following:

  • Form help
  • Task-based help
  • Reference information
  • Glossary of terms.

Select Help Contents, then click on the Search tab.

Searches online help using words, exact phrases, exclusions, and logical users to define match criteria.

Note:

The search does not include hint text, field help, query type help, confirm window, and message help.

Select Help Contents, then select Glossary of Terms in the Contents tab.

Glossary of terms. Provides definitions of terms used in Singleview.

Click the Help button in the window.

Confirm window and message help. Provides information about window options, or cause of the message and possible corrective action.

Click the Query Type Help... button on the Customer Query form.

Query type help. Provides customised information about the type of query (for example, what information to collect).
