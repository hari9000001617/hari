How Do I - System Integration Tasks - Interface Types

Select the option most appropriate to the required task.

View an Interface Type

To view an interface type:

  1. Select the Maintenance Server Interface Type option. The Interface Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of defined interface types matching the specified criteria are displayed.
  4. Select the appropriate interface type and click the Detail... button, or double-click the appropriate interface type. The Interface Type Definition form is displayed in View Only mode.

Need more information on interface types?

Define an Interface Type

To define an interface type:

  1. Select the Maintenance Server Interface Type option. The Interface Type Search form is displayed.
  2. Click the New... button. The Interface Type Definition form is displayed in Insert mode.
  3. Complete the appropriate fields on the form.
  4. Click the Save button on the toolbar.

Need more information on interface types?

Update an Interface Type

To update an interface type:

  1. View the appropriate interface type.
  2. Click the Update button on the toolbar. The Interface Type Definition form is displayed in Update mode.
  3. Update the interface type details as required.
  4. Click the Save button on the toolbar.

Need more information on interface types?

Delete an Interface Type

To delete an interface type:

  1. View the appropriate interface type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm deletion.
