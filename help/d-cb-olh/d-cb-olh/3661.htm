Reference Type Search Form

The Reference Type Search form is used to search for one or more reference types, using a specific field, criteria, and method.

Defined list of items that can be referenced throughout Convergent Billing. Typically used for a list of predefined options for users. Each item in a reference type is identified by a reference code. A reference type should be used whenever there is a requirement to access a set number of values where the values are constants. Reference types should not be used for variable value lists.

A shared class in Process Engine can map to a Convergent Billing reference type.

Context: Convergent Billing, Process Engine, Customer Management

See also Child Reference Type, Derived Attribute, Reference Code, and Lists.

The lower part of the form is used to list reference types returned by the search that match the specified criteria.

Multiple instances of the form can be displayed by clearing the Reuse form? check box.

Caution:

If multiple instances of the form are required for display (for example, to access details for several account types), do not select the Reuse form? check box.

If selected, the Reuse form? check box overrides display of information in a new form, and details are displayed in the existing form.

Associated Actions

Viewing a Reference Type

To view a reference type:

  1. Select the Maintenance Data Definition Reference Types option. The Reference Type Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of reference types matching the specified criteria is displayed.
  4. Select the appropriate reference type and click the Detail… button, or double-click the appropriate reference type. The Reference Type Definition form is displayed in View Only mode.

Updating a Reference Type

Caution:

Modifying a reference type may result in corruption of data and/or forms that use the reference type.

To update a reference type:

  1. View the appropriate reference type.
  2. Click the Update button on the toolbar. The Reference Type Definition form is displayed in Update mode.
  3. Update the reference type details as required.
  4. Click the Save button on the toolbar.

Need more information on reference types?

Defining a Reference Type

To define a reference type:

  1. Select the Maintenance Data Definition Reference Types option. The Reference Type Search form is displayed.
  2. Click the New… button. The Reference Type Definition form is displayed in Insert Mode.
  3. Complete the appropriate data entry fields on the form.
  4. Add reference items to the table in the lower part of the form as required. Each reference item must be entered on a separate row of the table.

    To add a row to the table, press the Down Arrow key.

    To insert a blank row before the current row, click the Insert button on the toolbar.

  5. If necessary, change the order of the reference items using the Up and Down buttons.
  6. Click the Save button on the toolbar.

Need more information on reference types?

Deleting a Reference Type

Caution:

Deleting a reference type may result in corruption of data and/or forms that use the reference type.

To delete a reference type:

  1. View the appropriate reference type.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the reference type is to be deleted.
