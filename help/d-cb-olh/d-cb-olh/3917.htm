Tariff Table Form

The Tariff Table form is used to enter, view, and update a tariff table.

Table of conditions and associated charges. The table is used to determine the charge values to be substituted into the charge expression to calculate the value of the specified tariff. The charge values used for the substitution are those associated with the conditions in the tariff table that match the conditional expressions used as keys into the tariff table.

The structure of the tariff table is created during tariff configuration and the appropriate conditions and charges entered into the table during tariff definition.

Context: Convergent Billing

See also Tariff, Tariff Configuration, and Tariff Definition.

The upper part of the form contains a number of read-only fields that display the values entered in the Settings tab of the associated Tariff Definition form.

The lower part of the form displays a tariff table, with table columns reflecting the conditions and charges specified in the associated tariff definition (one column per condition or charge specified). The table also contains an Invoice Text column in which text to be included on an invoice for a given condition or charge can be specified. The tariff table can be populated by adding one or more table rows, each row containing a value in each condition and charge column, and optionally, some invoice text.

The Definition... button allows the tariff definition to be viewed.

The form can be displayed in one of three form modes, with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Associated Actions

Viewing a Tariff Table

Before a tariff table can be viewed, the appropriate tariff definition must be defined.

To view a tariff definition:

  1. Select the Maintenance Tariffs Tariffs Table Maintenance option. The Tariff Table Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of tariff tables matching the specified criteria is displayed.
  4. Select the appropriate tariff table and click the Detail... button, or double-click the appropriate tariff table. The Tariff Table Definition form is displayed in View Only mode.

Defining a Tariff Table

To define a tariff table:

  1. For a given set of:
    • Conditions, determine the appropriate charge values to be used in the charge expression to calculate the value for the tariff.
    • Charge values, determine the conditions that must be satisfied for the appropriate charges to apply.
  2. View the appropriate tariff table.
  3. Click the Insert button and enter one set of conditions and charges (identified in step 1) as a row in the tariff table.
  4. If specific text is required on the invoice for a given set of conditions, enter that text in the Invoice Text field associated with the current row.
  5. Repeat steps 3 and 4 for each set of conditions identified in step 1.
  6. If necessary, rearrange the order of the rows using the Up and Down buttons.

    Note:

    The order in which the rows are listed determines the order in which the conditions are evaluated by the rating engine. Only the values associated with the first match are used to calculate the charge for the event.

  7. Click the Save button on the toolbar.

Updating a Tariff Table

To update a tariff table:

  1. View the appropriate tariff table.
  2. Click the Update button on the toolbar. The Tariff Table Definition form is displayed in Update mode.
  3. Update the form as required.
  4. Click the Save button on the toolbar.
