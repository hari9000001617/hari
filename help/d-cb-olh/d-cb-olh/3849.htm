Subtotal Definition History Form

The Subtotal Definition History form is used to view a:

  • List of records for the specified subtotal.

    Variable containing calculated values (such as aggregated cost totals) stored for subsequent use in the rating and billing process.

    A subtotal defines one or more subtotal terms aggregated using the method specified by the associated aggregation function.

    Context: Convergent Billing

    See also Aggregation Function, Progressive Subtotal, Keyed Subtotal, and Subtotal Term.

    The list of records is displayed in chronological order and includes the current record, one or more historical records (if available), and one or more future records (if defined).

    Record having a start date in the future.

    Context: Convergent Billing, Customer Management

    See also Current Record and Historical Record.

    Record having an end date before today.

    Context: Convergent Billing

    See also Current Record and Future Record.

    Record having a start date before or equal to today, and an end date equal to or after today.

    Context: Convergent Billing, Customer Management, Process Engine

    See also Future Record and Historical Record.

  • Specific record and, if appropriate privileges are available, to update records.

The form can be displayed in one of two form modes (View Only or Update), with the current mode indicated in the form’s title.

A form can be displayed in one of the following modes:

Form Mode

Description

View Only mode

Allows information held in Convergent Billing to be viewed, but not updated (modified).

Insert mode

Allows new information to be recorded (entered) in Convergent Billing.

Update mode

Allows information held in Convergent Billing to be viewed and updated (modified).

Associated Actions

Viewing a Subtotal

To view a subtotal definition:

  1. Select the Maintenance Tariffs Subtotals option. The Subtotal Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of subtotal definitions matching the specified criteria is displayed.
  4. Select the appropriate subtotal definition and click the Detail… button, or double-click the appropriate subtotal definition. The Subtotal Definition form is displayed in View Only mode.

Need more information on subtotals?

Defining a Subtotal

To define a subtotal:

  1. Select the Maintenance Tariffs Subtotals option. The Subtotal Search form is displayed.
  2. Click the New... button. The Subtotal Definition form is displayed in Insert Mode.
  3. Complete the appropriate data entry fields.
  4. Click the Save button on the toolbar.

Need more information on subtotals?

Updating a Subtotal

To update a subtotal:

  1. View the appropriate subtotal.
  2. Click the Update button on the toolbar. The Subtotal Definition form is displayed in Update mode.
  3. Update the subtotal details as required.
  4. Click the Save button on the toolbar.

Need more information on subtotals?

Deleting a Subtotal

Caution:

Before deleting a subtotal, ensure that the subtotal is not referenced in another subtotal or tariff.

To delete a subtotal:

  1. View the appropriate subtotal.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the subtotal is to be deleted.

Need more information on subtotals?

Viewing an Historical Data Record

To view a historical record:

  1. Display the details of the current data record on the appropriate form. For example, to display the details of a customer, access and display the Customer Detail form with the customer's details.
  2. Click the History button on the toolbar. The appropriate history form is displayed listing each of the historical records associated with the current record.
  3. To view a specific historical record, double-click the appropriate entry in the list of historical records, or select one of the entries and click the Details... button. The relevant form is displayed with details of the historical record.
