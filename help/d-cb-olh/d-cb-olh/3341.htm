Optional Product Components Form

The Optional Product Components form is used to:

  • List service and equipment types available to the product instance for allocation.

    Specifies any defaults or entity validation for equipment specified with that type. If required, an icon can be specified to identify each equipment type.

    Context: Convergent Billing, Customer Management, Configuration Studio

    See also Equipment, Service, Service Type, and Icon.

    Specifies any defaults or entity validation for services specified with that type. If required, an icon can be specified to identify each service type.

    Context: Convergent Billing, Configuration Studio

    See also Icon and Service.

  • Display details of the service and equipment types, number of additional components allocated, and maximum number of components available for allocation.

The options available for selection are dependent on the configuration of the product.

Associated Actions

Viewing a Service

To view summary information about each service:

  1. Display the Customer Hierarchy form for the appropriate customer.
  2. Select the Products and Services options in the Display for customer group. Summary information about each product instance and its associated services for the specified customer is displayed in the right-hand panel.

Creating a Service

To create a service:

  1. View the relevant product instance.
  2. Highlight one of the service types listed.
  3. Click the Create… button. A message is displayed prompting confirmation that the start and end dates for the product instance are correct.
  4. Click the Yes button to continue creating the service with the specified dates. The Service Detail form is displayed.
  5. If the dates are not correct, click the No or Cancel button to return to the Product Instance Detail form to change the dates.
  6. Complete the form for the selected service.
  7. Click the Save button on the toolbar. Control is returned to the Product Instance Detail form.
  8. To allocate an optional service to the product instance, click the Add Service… button. The Optional Product Component form is displayed.
  9. Select the optional component and click the Add button. The additional component is added to the Product Instance Detail form, the Optional Product Component form is closed, and control is returned to the Product Instance Detail form.
  10. Complete steps 2 to 6 for each additional component to be allocated to the product instance.
  11. Click the Save button on the toolbar.

Viewing Equipment Details

To view equipment details:

  1. Select the Customer Care Equipment Manage Equipment option. The Equipment Search form is displayed.

    Note:

    The Transit tab is only displayed on the Equipment Search form if the equipment type is configured to enable the tab.

  2. Enter or select the appropriate search criteria.
  3. Click the Search… button. A list of equipment records matching the specified criteria is displayed.
  4. Select the appropriate equipment record and click the Details... button. The Equipment Detail form is displayed in View Only mode.

Allocating Equipment

To allocate equipment:

  1. Verify that the equipment to be allocated is recorded in the equipment inventory.
  2. View the appropriate product instance.
  3. Select one of the equipment entries and click the Allocate… button. If the item of equipment is not allocated, a message is displayed prompting confirmation that the start and end dates for the product instance are correct. Click the appropriate button. The Equipment Allocation form is displayed.
  4. Select either of the following:
    • An item in each of the available equipment pools and click the Select button. A list of valid equipment items is displayed. If no equipment pools are displayed in the Equipment Pool Select group, all equipment items held in the equipment inventory are listed.
    • The Customer's equipment only option and click the Select button. A list of equipment allocated to the specified customer is displayed.
  5. Choose one of the selection options in the Allocate equipment group and where necessary, enter an appropriate value in the associated field.

    The options in the Allocate equipment group on the Equipment Allocation form are used to select:

    • The first item in the range of items displayed

      Select the First item in range option and click the Allocate... button.

    • Other item in the range of items displayed

      The Other item in range option is available only when there is more than one item in the range. Select the Other item in range option, enter the required item's serial number, and click the Allocate... button.

    • An item having a specific serial number

      Select the Specific serial number option, enter the required item's serial number, and click the Allocate... button.

    In each case, the specified item of equipment is allocated.

    Note:

    If equipment allocated to the customer is being reused, a warning message is displayed when the Allocate... button is clicked, prompting confirmation that it should be allocated.

  6. Click the Allocate button. The selected item of equipment is allocated.
  7. To allocate an optional equipment item to the product instance, click the Add Equipment... button. The Optional Product Component form is displayed.
  8. Select the optional component and click the Add button. The additional component is added to the Product Instance Detail form, and control is returned to the Product Instance Detail form.
  9. Complete steps 3 to 6 for each additional equipment item to be allocated to the product instance.
