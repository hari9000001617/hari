Adjustment Batch Search Form

The Adjustment Batch Search form is used to search for one or more adjustment batches, based on the specified search criteria (for example, batch date and type, adjustment type, batch number, and status, currency, and description).

Selecting the Search for batches option enables the Search… button, for searching for batches matching the specified criteria. Selecting the Create a new batch option enables the New… button to create a new batch.

The display format for the retrieved records can be determined by selecting or clearing the Always Display List? and Reuse Detail Form? check boxes.

If multiple instances of the detail form are required for display (for example, to access details for several payments) the Reuse detail form? option should not be selected.

Caution:

If selected, the Reuse detail form? option overrides the display of information in a new form; the details are displayed in the existing form.

If a single adjustment batch record is returned from a search, the details of that record are displayed on the Individual Adjustment Batch form. If more than one adjustment batch record is returned, the records are listed on the Adjustment Batch List form. The required record can then be selected and its details displayed on the Adjustment Batch form.

If selected, the Always Display List? option overrides the automatic display of the Adjustment Batch form when only a single record is retrieved; in such cases the Adjustment Batch List form is displayed.

Note:

Only the Adjustment type, Batch Date, and Description fields are enabled when the Create a new batch option is selected. To retrieve adjustment batch records created before today’s date, delete the date in the Batch Date field.

Associated Actions

Viewing an Adjustment Batch

Tip:

To view an adjustment not yet applied, view the adjustment batch, and details of all applied and unapplied adjustments for that adjustment batch are listed in the adjustment Detail section.

To search for an adjustment batch and list adjustments in the batch:

  1. Select the Accounts Adjustments Manage Batch Adjustments option. The Adjustment Batch Search form is displayed.
  2. Enter or select the appropriate search criteria. If the batch date is unknown, remove the date from the Batch Date field.
  3. Click the Search… button.

    If the search returns:

    • A single record, the Adjustment Batch form is displayed in View Only mode (if the Always Display List? option was not selected).
    • Multiple records, or the Always Display List? option was selected, the Adjustment Batch List form is displayed.
  4. To access the Adjustment Batch form from the Adjustment Batch List form, double-click the appropriate batch in the list, or select the appropriate batch and click the Details… button. The Adjustment Batch form is displayed listing each of the adjustments included in that batch.

    The Details… button is enabled when the batch adjustments have been applied.

Creating an Adjustment Batch

To create an adjustment batch:

  1. Collate adjustments into batches (groups) of the same adjustment type.
  2. Select the Accounts Adjustments New Batch Adjustment option. The Adjustment Batch Detail form is displayed in Insert mode.
  3. Enter a description of this batch in the Description field. The description entry is used to identify the batch at a later date.
  4. Select the appropriate adjustment type and batch type.
  5. Enter the account number or invoice number in the appropriate columns associated with the adjustment and press the Tab key. An adjustment number is assigned to the adjustment and displayed in the first column of the current row.

    The account number or invoice number is verified against the account details held in the system and, if valid, corresponding account and invoice details are displayed in the relevant columns.

    If invalid, a warning message is displayed on the Status bar. To rectify this, click the Delete button and enter a valid account or invoice number.

    Tip:

    Ensure that the customer name is correct. If not, a valid, but incorrect account or invoice number has probably been entered. To rectify this, click the Delete button to remove the row, click the Insert button to insert a new row, and re-enter the account or invoice number.

  6. If there is a selected invoice for the account, the invoice number appears in the Invoice column. Enter the amount of the adjustment in the Amount column, and press the Tab key. The Invoices for Adjustment form is displayed in Update mode.
  7. If there are a number of invoices against the account, the word 'Split' appears in the Invoice column. Enter the amount of the adjustment in the Amount column, and press the Tab key. The Invoices for Adjustment form is displayed in Update mode.
  8. Allocate an amount to one or more of the listed invoices as required.

    When an adjustment is being recorded, it (or part thereof) can be allocated to one or more of the outstanding invoices or simply recorded as a credit (or debit if it is a negative adjustment). Recording an adjustment as a credit or debit (rather than against a specific invoice) may be necessary if, for example, it is not clear which invoice should be allocated the adjustment.

    When an adjustment is made, Singleview automatically makes an initial allocation of the funds against any outstanding invoices (if there are any). The initial allocations can be modified before the adjustment is applied. Adjustments are allocated differently depending on the allocation function specified for the adjustment type.

    After the initial allocation is made, view and update the allocations by selecting the Invoices tab (for an individual adjustment) or by selecting an adjustment and clicking the Invoices button (for a batch adjustment).

    The amount allocated to each invoice and receivable type is displayed in This Adjustment field (in the Individual Adjustment Detail form for individual adjustments or in the Invoices for Adjustment form for batch adjustments). Any credit amount not allocated to an invoice is displayed in the Credit field. The Credit field is updated as funds are allocated or de-allocated to an invoice or receivable type.

    To adjust the initial allocations made to the adjustment (whichever form is displayed):

    • Enter zero in the This Adjustment field of the invoices not being allocated any funds.
    • If the whole amount is being allocated against a single invoice, the full amount is entered in the This Adjustment field of the appropriate invoice.
    • If the amount is being allocated against one or more receivable types, enter the amount in the This Adjustment column for the appropriate receivable type.
    • If the adjustment is being split between two or more invoices, the required amount for each invoice is entered in the This Adjustment field. Each time an amount is entered, an equivalent amount is deducted from the amount in the Credit field. If the adjustment is more than the combined total of the outstanding invoice balances, the funds remaining are treated as a credit against the specified account.

    Note:

    The amount in the Credit field is displayed in brackets if amounts are entered with a total of more than the adjustment amount.

    View and modify how an adjustment is allocated at any stage.

  9. Press the Tab key or close the Invoices for Adjustment form. Control is returned to the Adjustment Batch Detail form.
  10. If the adjustment has been entered against more than one invoice, the word 'Split' is displayed in the Invoice column.
  11. If the adjustment was not made against an invoice, the word 'Credit' (or 'Debit' if it was a negative adjustment) is displayed in the Invoice column.
  12. If required, overwrite the default text in the Adjustment Text field and enter a description of (or reason for) the adjustment in the Description field.

    Note:

    Any text written in the Adjustment Text field is printed on the customer's next invoice or statement; text written in the Description field is saved with the adjustment details, but not printed.

  13. Complete any other fields defined for this adjustment. The fields displayed depend on the adjustment type selected for the batch. As each adjustment is entered, the total value of the adjustments in the batch is displayed in the lower right corner of the form.
  14. Press the Down Arrow key to insert the next adjustment. A new row is inserted below the current row with a new receipt number.
  15. Repeat steps 5 to 11 until all adjustments have been entered, or until the batch needs to be saved.
  16. Click the Save button on the toolbar. The Adjustment Batch Detail form is displayed in View Only mode.
  17. To post the adjustments to the respective accounts, apply the batch.
  18. To close the adjustment batch without posting the adjustments, click the Close button on the toolbar.

Applying an Adjustment Batch

To apply (or reapply) the adjustments in an adjustment batch to their respective accounts:

  1. Ensure that the adjustment batch is saved.
  2. Click the Apply button. A message indicating that the adjustments in the adjustment batch have been applied (posted) is displayed on the Status bar.

Updating an Adjustment Batch

Notes on Updating Adjustment Batches

To update an adjustment batch:

  1. View the appropriate adjustment batch.
  2. If necessary, un-apply the batch.
  3. Modify individual adjustments within the batch as required.
  4. Click the Save button on the toolbar to save the modifications and re-apply the batch.

Need more information on adjustment batches?

Deleting an Adjustment Batch

Caution:

An adjustment batch can only be deleted if it has not been applied or applied and is subsequently un-applied.

To delete an adjustment batch:

  1. View the appropriate adjustment batch.
  2. If required, un-apply the adjustment batch.
  3. Click the Delete button on the toolbar.
  4. When prompted, click the Yes button to confirm that the adjustment batch is to be deleted.
