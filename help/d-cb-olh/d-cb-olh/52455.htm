External Definition

External definitions are a key component of the interface between Singleview and external systems. External definitions define the details of the data structure and the fields that must be supplied either to or from an external system, and how they map to Singleview entities and their fields.

External definitions do not include specific details on transport or encoding mechanisms. These details are defined in a separate layer specific to the type of interface that is using the external definitions.

External definitions can map to a Convergent Billing entity or a Process Engine class; these are concrete external definitions. An external definition not mapped to a Convergent Billing entity or Process Engine class is an abstract definition. External definitions generated from an abstract definition are automatically mapped to the Convergent Billing entity or Process Engine class of a parent definition.

External definitions can also be linked to create complex structures combining details from multiple entities. Entities can be different, or belong to the same type (for example, customers and child customers). These are referred to as nested external definitions.

XML Schema terminology has been adopted for representing components of an external definition. Terminology includes:

  • Namespace
  • Attributes
  • Elements
  • Content models (Simple Content, Sequence, Choice, and All).

External definitions are created and maintained in the External Definition form.

External definitions support combinations of the following basic operations on their underlying Convergent Billing or Process Engine entities:

  • Insert
  • Update
  • Delete
  • Search
  • Fetch.

To simplify the creation of external definitions, they can be generated:

  • For a Convergent Billing entity (using an entity validation)
  • For a Process Engine class
  • From XML schema files.

The generated external definition can be customised to suit requirements of the external interface that is to use it, and it can also be nested.

For more information about external definitions, refer to the System Configuration Guide for Singleview Convergent Billing.
