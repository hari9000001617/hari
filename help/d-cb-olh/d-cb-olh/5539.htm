This guide contains reference lists necessary for the configuration of Convergent Billing. The guide is contained in two volumes.

Who Should Read this Guide?

Users and administrators who require reference information for configuring Convergent Billing.

Is Prerequisite Knowledge Assumed?

This guide assumes:

  • Some experience using Microsoft Windows
  • A good understanding of the data entry forms used in Convergent Billing
  • A thorough knowledge of the normal conventions of writing expressions and functions in a programming language.

This guide is to be used in conjunction with other guides in the documentation suite. Examples are:

  • Configuration Fundamentals Guide for Singleview Convergent Billing
  • Customer Care and Product Configuration Guide for Singleview Convergent Billing
  • Finance Guide for Singleview.
