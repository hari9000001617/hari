Interface Template Search Form

The Interface Template Search form is used to search for one or more interface templates.

Search criteria can be specified in the Interface Type and Name fields.

Selecting the:

  • Search for interfaces option enables the Search… button on the form, for searching for interface templates matching the specified criteria.
  • Create a new interface option enables the New… button, for creating interface templates.

The display format for the retrieved records can be determined by selecting or clearing the Always Display List? and the Reuse Detail Form? check boxes.

Associated Actions

View Interface Templates

To view interface template details:

  1. Select the Maintenance Server Interface Template option. The Interface Template Search form is displayed.
  2. Enter or select the appropriate search criteria.
  3. Click the Search… button. A list of interface templates matching the specified criteria is displayed.
  4. Select the appropriate interface template record and click the Details... button. The Interface Template Detail form is displayed in View Only mode.

Create Interface Templates

To create an interface template:

  1. Select the Maintenance Server Interface Template option. The Interface Template Search form is displayed.
  2. Select the Create a new interface check box.
  3. Select the interface type from the Interface Type drop-down list.
  4. Select the Interface Template from the drop-down list, if required.

    If a value is specified, the interface definition is created based on the interface template specified. If not specified, the first available interface template of the selected interface type is used.

  5. Click the New… button. The Interface Template Detail form is displayed in Insert mode.
  6. Complete the relevant fields.
  7. Click the Save button.

Need more information on interface templates?

Update Interface Template Details

To update details of an interface template:

  1. View the relevant interface template.
  2. Click the Update button on the toolbar. The Interface Template Detail form is displayed in Update mode.
  3. Update the relevant interface template details.
  4. Click the Save button on the toolbar.

Need more information on interface templates?

Delete Interface Templates

To delete an interface template:

  1. View the relevant interface template.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the interface template is to be deleted.

Interface templates cannot be deleted if interface definitions based on the template exist.
