How Do I - System Integration Tasks - File Processing

Select the option most appropriate to the task.

Search for a File Processing Record

To search for a file processing record in the File Processing Register:

  1. Select the Server Tasks File Processing option. The File Processing Search form is displayed.
  2. Enter or select the appropriate search criteria.

    Note:

    The file name and file path are not case-sensitive and can contain the '*' wildcard. The Error Search group of fields is only enabled if the Processing Status field is blank or set to Error.

  3. Click the Search button. The File Processing List form, or the File Processing Details form is displayed.

View File Processing Details

To view processing details of a file:

  1. Select the Server Tasks File Processing option. The File Processing Search form is displayed.
  2. Enter or select the appropriate search criteria.

    Note:

    The file name and file path are not case-sensitive and can contain the '*' wildcard. The Error Search group of fields is only enabled if the Processing Status field is blank or set to Error.

  3. Deselect the Always Display List? option.
  4. Select the Reuse Detail Form? option, if required.
  5. Click the Search button.

    If one search result is returned, the File Processing Details form is displayed. If multiple search results are returned, the File Processing List form is displayed.

  6. If the File Processing List form is displayed, double-click the file for which the processing details are to be displayed, or highlight the file and click the Details… button. The File Processing Details form is displayed with detailed processing information for the selected file.
