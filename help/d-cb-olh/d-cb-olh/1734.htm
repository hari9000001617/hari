Access Level Settings Form (Form Roles)

The Access Level Settings form is used to specify security privileges (access levels) to be granted to a user to access specific forms and menu items.

Security feature that determines what a user can do and what data they are permitted to access. Security is based on form roles and groups, which in combination, determine a user's privileges. Each user is assigned a specific profile that defines the level of access for each role and group.

Context: Convergent Billing, Process Engine

See also Group, User, User Profile, and Form Role.

The access levels specified using this form can also control the privileges required to update the data on the specified form.

A selected check box indicates that the associated access level is granted.

Associated Actions

Viewing a Form Role

To view a form role:

  1. Select the Maintenance Security Form Roles option. The Form Role Search form is displayed.
  2. Enter or select the appropriate search criteria and search method.
  3. Click the Search button. A list of the defined roles matching the specified criteria is displayed.
  4. Select the appropriate role and click the Detail… button, or double-click the appropriate role. The Form Role form is displayed in View Only mode.

Defining a Form Role

To define a form role:

  1. Select the Maintenance Security Form Roles option. The Form Role Search form is displayed.
  2. Click the New… button. The Form Role Definition form is displayed in Insert mode.
  3. Complete the Name and Description fields.
  4. Complete a line in the grid for each menu item or form to which the form role is to provide access.
  5. Click the Save button on the toolbar.

Need more information on security?

Updating a Form Role

To update a form role:

  1. View the appropriate form role.
  2. Click the Update button on the toolbar. The Form Role form is displayed in Update mode.
  3. Update the form role details as required:
    • Click on the Insert button to insert a new line in the grid for a menu item or form to which the form role is to provide access.
    • Highlight a form and click on the Access Level Settings button to display the access levels for that form. Select the appropriate access levels for the form.
  4. Click the Save button on the toolbar.

Need more information on security?

Deleting a Form Role

To delete a form role:

  1. View the appropriate form role.
  2. Click the Delete button on the toolbar.
  3. When prompted, click the Yes button to confirm that the form role is to be deleted.
