Properties Form

The following properties of a data entry form can be viewed (and if required, modified):

  • Security group to which the data in the form belongs
  • Name of the user who last made changes to the data in the form
  • Date on which the changes were made
  • Entity ID that displays the database ID for this entity type.

Some configuration forms also have the following properties:

  • Version when latest modifications were made
  • Categories assigned to the entity.

All users are permitted to view such details; only users with Update security privileges for the form are permitted to modify the details.

Users permitted to modify this information can change the security group to which the information belongs. Modifying the details allows data to be viewed by users who were not in the security group associated with the user who entered the data.

Associated Actions

Viewing a Form’s Properties

To view the properties of a data entry form:

  1. Make the form active.
  2. Select the Edit Properties option. The Properties form is displayed.

    The form displays the security group to which the data in the form belongs, name of the user who last made changes to the form, and date on which those changes were made.

  3. Click the OK button to close the form.

Modifying a Form’s Properties

To modify properties of a data entry form:

  1. Make the form active.
  2. Click the Update button on the toolbar.
  3. Select the Edit Properties option. The Properties form is displayed.
  4. Modify the group by selecting an alternative group from the drop-down list.
  5. Click the OK button to close the form.
  6. Click the Save button on the toolbar.
